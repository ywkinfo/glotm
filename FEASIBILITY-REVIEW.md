## 1. 프로젝트 개요

GloTm은 해외 진출 기업의 브랜드 관리자 및 인하우스 IP 담당자를 위한 글로벌 상표 운영 지식베이스다.

| 항목 | 내용 |
|------|------|
| 핵심 포지셔닝 | 인터넷 검색(신뢰 불가) ↔ 로펌 유료 상담(비용·속도 부담) 사이의 정보 공백 |
| 콘텐츠 저자 | 김앤장 법률사무소 상표 변리사 (경력 20년+) |
| 현재 포트폴리오 | LatTm(중남미), MexTm(멕시코), UsaTm(미국), JapTm(일본), ChaTm(중국), EuTm(유럽) — 6개 live guide |
| 타겟 사용자 | 해외 진출 한국·아시아 소비재 기업의 브랜드 관리자, 인하우스 IP 담당자 |
| 현재 운영 방식 | 무료 서비스, 재직 중 비상업적 운영 |
| 장기 목표 | 은퇴 후 B2B 라이선스 모델로 전환 |

## 2. 시장 기회 분석

### 2-1. 수요 기반

| 항목 | 평가 |
|------|------|
| 해외 진출 기업 수 | 한국 기업의 해외법인 보유 기업 4,600개+, SME 해외 진출 지속 증가 |
| 타겟 직무 규모 | 중견~대기업은 인하우스 IP팀 존재, SME는 브랜드 관리자가 1인 처리 |
| 라틴아메리카·멕시코 파일럿 | K-뷰티·식품·패션 수출 급증 → 상표 분쟁 사례 증가 |
| 정보 격차 현실 | 로펌 블로그는 단편적·마케팅 목적, AI는 신뢰성 검증 불가 → 구조화된 정보 수요 실재 |

### 2-2. 핵심 차별화 요소

저자가 20년 실무 경력의 현직 변리사라는 점이 콘텐츠 신뢰성과 깊이의 본질적 차별화 요소다. "법 조문 설명"이 아닌 출원이 아니라 운영 관점 — 어떤 체크포인트를 먼저 보고, 어떤 운영 시스템을 깔아야 하며, 어떤 순서로 대응해야 하는지를 실무 흐름에 맞춰 정리한 구조는 시중에 거의 없다.

### 2-3. 경쟁 환경

| 유형 | 대표 예시 | GloTm 대비 |
|------|-----------|-----------|
| 로펌 블로그 | 국내 주요 로펌 IP 블로그 | 산발적·마케팅 목적, 운영 프레임 없음 |
| 전문가 협회 가이드 | INTA 핸드북, Chambers | 변호사용, 고가, 제도 중심 |
| AI 서비스 | ChatGPT, Claude | 검증 불가, 개별 질의 한계 |
| 상표 관리 SaaS | Dennemeyer, Anaqua | 도구 중심, 지식/판단 가이드 아님 |

결론: 직접 경쟁자가 부재한 수준이다. 단, 서비스가 존재하지 않기 때문에 수요가 아직 잠재적이며, 인지도 확보가 핵심 과제다.

## 3. 수익 모델 타당성

### 3-1. 현재 제약

- 김앤장 재직 중 → 직접 수익화·적극 프로모션 불가
- 은퇴 전까지 무료 서비스 + 이메일 구독 기반 운영 계획

### 3-2. 장기 수익 모델 평가

| 모델 | 내용 | 타당성 |
|------|------|--------|
| B2B 라이선스 | 기업 연간 $3,000~$30,000 | ★★★★★ — 구독 기반, 예측 가능한 반복 수익 |
| 화이트라벨 | 로펌·컨설팅 OEM 라이선스 | ★★★★☆ — 저자 네트워크 활용 가능 |
| 강의·웨비나 | 기업 IP 담당자 대상 | ★★★☆☆ — 단발성 수익, 확장 한계 |
| 개인 구독 | 월/연 SaaS 구독 | ★★★☆☆ — 개인 니즈 대비 가격 저항 가능 |

핵심 관찰: 은퇴 후 타겟 고객(브랜드 관리자·IP 팀)은 기업 예산으로 구독 결정 가능하다. $3,000~$5,000/year 진입 티어는 로펌 단 1회 자문 비용보다 저렴하여 ROI 납득이 용이한 가격 구조다.

### 3-3. 수익화 경로 (Funnel)

```
유기 검색 유입 (Google SEO)
    ↓
무료 1챕터 공개 → 이메일 게이트 (Phase 3)
    ↓
구독자 리스트 축적 (파일럿 목표: 500명)
    ↓
은퇴 후 B2B 전환 연락 → 팀/기업 라이선스
```

SEO 기반 오가닉 유입이 CAC(고객 획득 비용)를 제로에 가깝게 유지한다는 점이 1인 운영 구조에 최적화되어 있다.

## 4. 기술 완성도 평가

### 4-1. 현재 구현 상태

| 항목 | 상태 | 비고 |
|------|------|------|
| 핵심 읽기 경험 | ✅ 완성 | 6개 가이드 모두 live shell 연결 |
| 전문 검색 | ✅ 작동 | MiniSearch 클라이언트 검색, LatTm 780개 엔트리 |
| 모바일 반응형 | ✅ 구현 | 모바일 드로어 내비게이션 |
| 진행률·북마크 | ✅ 구현 | localStorage 기반 continue reading |
| 법적 고지 | ✅ 구현 | 전 페이지 하단 고지 |
| SEO prerender | ❌ 미구현 | Phase 2 핵심 과제 |
| 이메일 게이트 | ❌ 미구현 | Phase 3 과제 |
| MexTm 빌드 파이프라인 | ⚠️ 불완전 | manifest/build-master/qa 미구축 |
| ChaTm 콘텐츠 | ⚠️ 최소 | 6챕터만 존재 |
| 배포 | ❌ 미배포 | GitHub Pages 파이프라인 준비 완료 |

### 4-2. 기술 완성도 요약

Phase 1 기준 완성도 85%. 배포를 막는 기술적 장벽은 없다. SEO prerender(Phase 2)가 오가닉 유입에 직결되는 가장 중요한 다음 단계다.

### 4-3. 기술 스택 적합성

| 항목 | 평가 |
|------|------|
| React + Vite SPA | 경량·빠른 빌드, 단일 개발자에 최적화 |
| MiniSearch (클라이언트 검색) | 서버 없이 전문 검색 가능 — 운영 비용 최소화 |
| Markdown → JSON 빌드 파이프라인 | 콘텐츠 업데이트가 비개발자도 가능한 구조 |
| GitHub Pages 배포 | 무료 호스팅, CI/CD 자동화 완비 |

## 5. 리스크 요소

| 리스크 | 심각도 | 현재 대응 상태 |
|--------|--------|--------------|
| 재직 중 이해충돌 | 높음 | 무료 서비스·비공개 운영으로 관리 중 |
| 1인 운영 한계 | 중간 | 콘텐츠 구조화·빌드 자동화로 유지보수 최소화 |
| SEO 경쟁 (영어권) | 중간 | 한국어·스페인어 특화로 경쟁 회피 가능 |
| 법적 책임 리스크 | 낮음 | 전 페이지 법적 고지 명시, "자문 아님" 포지셔닝 |
| 콘텐츠 노후화 | 중간 | 법령 변경 시 업데이트 필요 — 정기 점검 프로세스 미정비 |
| 은퇴 전 모멘텀 상실 | 높음 | 파일럿 배포 타이밍이 구독자 축적에 결정적 |

가장 큰 리스크: 파일럿 배포를 은퇴 전 충분한 시간을 확보하지 않고 미루면 구독자 축적 기회를 잃을 수 있다. Phase 2(SEO prerender) 완료 후 즉시 배포하는 것이 중요하다.

## 6. 종합 사업가능성 평가

| 평가 항목 | 점수 | 근거 |
|----------|------|------|
| 시장 기회 | ★★★★★ | 명확한 정보 공백, 직접 경쟁 부재 |
| 콘텐츠 차별성 | ★★★★★ | 20년 실무 경력 저자, 운영 관점의 구조화 |
| 수익 모델 | ★★★★☆ | B2B 라이선스 가격 구조 타당, 실행은 은퇴 후 |
| 기술 준비도 | ★★★★☆ | 핵심 기능 완성, SEO·이메일 게이트만 남음 |
| 실행 리스크 | ★★★☆☆ | 1인 운영·재직 제약이 속도를 제한 |
| 종합 | ★★★★☆ | 사업가능성 높음 — 타이밍 관리가 핵심 |

결론: GloTm은 명확한 시장 공백을 겨냥하고, 신뢰성 높은 콘텐츠로 차별화된 지식베이스다. 기술 구현도 핵심 기능 면에서 완성 단계에 있어 사업 가능성은 충분히 높다. 다만 은퇴 전 구독자 기반을 충분히 쌓기 위해 배포 타이밍 관리가 성패를 가를 핵심 변수다.
