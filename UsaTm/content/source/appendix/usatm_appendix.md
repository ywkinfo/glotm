# UsaTm 부록

## 출원 전 질문표

- 실제 미국 시장에서 먼저 쓸 표장은 무엇인가
- owner는 누구이며 실제 user는 누구인가
- 현재 판매 중인 goods/services와 향후 확장 범위는 무엇인가
- 사용할 specimen 후보를 지금 당장 확보할 수 있는가
- 미국 변호사, 브랜드팀, 영업팀 사이의 승인 라인은 정리되어 있는가

## specimen 수집 체크리스트

- 제품 포장 또는 라벨 이미지
- 온라인 상품 상세 페이지 캡처
- 서비스 제공 화면 또는 제안서 캡처
- 사용 시점이 드러나는 자료
- owner와 user 관계를 설명할 수 있는 내부 메모 또는 계약

## 분쟁 triage 메모

- 문제 유형: 유사상표 / 위조 / 무단 셀러 / domain / 플랫폼
- 목표: 등록 저지 / 판매 중단 / 협상 / 증거 수집
- 채널: TTAB / marketplace / CBP / cease-and-desist / 법원 검토
- 선확인 항목: 미국 내 판매·유통·수입 사실 / owner-user 구조 / license chain / 원고적격
- 즉시 필요한 자료: 등록 정보, 사용 증거, 혼동 사례, 거래 캡처, 수입·유통 흔적
