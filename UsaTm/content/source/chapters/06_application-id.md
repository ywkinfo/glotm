# 출원서 작성 실무와 ID 설계

출원서의 품질은 입력값의 품질에서 갈린다. 특히 goods/services 식별과 owner 정보, 표장 버전, specimen 계획의 정합성이 중요하다.

## ID 작성 원칙

ID는 실제 사업과 증거에 맞아야 한다. 너무 넓게 쓰면 유지관리와 audit이 힘들어지고, 너무 좁으면 사업 확장 여지가 줄어든다.

## 증거와 출원문구 정합성

출원문구는 나중에 제출할 specimen 후보를 염두에 두고 설계해야 한다. 실제 포장, 상품 페이지, 서비스 화면에서 설명 가능한 문구인지 미리 확인한다.

## 제출 전 리뷰

owner 명의, 주소, filing basis, class 구성, goods/services, 표장 버전, specimen 계획을 한 번에 점검한다. 글로벌 기업일수록 내부 승인 라인을 단순하게 만들어 두는 편이 좋다.
