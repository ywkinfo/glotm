# UKTm Source Register

## 목적

이 문서는 `UKTm` 공개본에 반영될 핵심 사실의 공식 출처를 추적하기 위한 레지스터다.
본문에 확정 서술을 넣기 전에 먼저 이 레지스터에 출처와 검증 상태를 기록한다.

## 핵심 출처 묶음

| 주제 | 1차 출처 | 상태 | 메모 |
| --- | --- | --- | --- |
| 영국 상표 등록 절차 | UKIPO / GOV.UK | 진행중 | 신청 절차, 심사 흐름, 갱신 구조 정리 필요 |
| opposition / challenge | UKIPO | 진행중 | 공고 후 challenge 흐름 정리 필요 |
| 사용 / non-use cancellation | UKIPO / legislation | 진행중 | 실무 문장 검증 필요 |
| 라이선스 / assignment | UKIPO / GOV.UK | 진행중 | 운영 문장용 근거 수집 필요 |
| 세관 / 국경조치 | GOV.UK / HMRC 관련 자료 | 진행중 | 상표와 국경 대응 연계 확인 필요 |
| 플랫폼 / 도메인 | 플랫폼 정책 / ICANN 관련 문서 | 진행중 | 채널별 요구자료 템플릿 필요 |

## 운영 규칙

- 본문에 수수료, 기한, 기관명, 공식 시스템명, 구제수단 범위를 넣기 전 이 레지스터에 출처를 추가한다.
- 출처가 없는 문장은 `uk_tm_fact_verification_log.md`에서 `추가검증 필요`로 표시한다.
- 요약 블로그나 2차 자료는 참고용으로만 두고, 공개본 확정 근거로는 1차 출처를 우선한다.
