# UKTm Fact Verification Log

## 목적

이 문서는 `UKTm` 공개본에 들어가는 변동성 높은 사실을 검증 상태별로 추적한다.
status는 `확정`, `조건부`, `추가검증 필요` 세 가지를 사용한다.

## 현재 확인 대상

| 항목 | 상태 | 출처 | 메모 |
| --- | --- | --- | --- |
| UKIPO 공식 절차명 | 진행중 | source register 참조 | 본문 표기 통일 필요 |
| opposition 관련 기간 문구 | 진행중 | source register 참조 | 숫자 단정 서술 전 재검증 필요 |
| 갱신 관련 기간 문구 | 진행중 | source register 참조 | 최종 공개 전 확인 필요 |
| non-use cancellation 설명 | 진행중 | source register 참조 | 실무 표현 정밀화 필요 |
| 세관 / 병행수입 설명 | 진행중 | source register 참조 | 정책과 실무 구분 필요 |

## 편집 규칙

- 숫자, 기간, 수수료, 제도 명칭은 이 로그에서 `확정` 되기 전 본문에 단정적으로 쓰지 않는다.
- 본문 수정이 들어가면 해당 항목의 상태를 같이 갱신한다.
- 공개 직전에는 `추가검증 필요` 항목을 0으로 만드는 것을 목표로 한다.
