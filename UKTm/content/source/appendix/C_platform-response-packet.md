## 채널별 추가 요구사항

### Amazon Brand Registry / IP Infringement Report

- 브랜드 레지스트리 등록 여부 확인
- ASIN 또는 판매자 ID 기재
- 침해 유형 선택: 위조 / 저작권 / 기타 IP

### eBay VeRO (Verified Rights Owner Program)

- VeRO 회원 등록 여부 확인
- Item number 기재
- 신고 양식: VeRO Report Infringement 폼 사용

### Google Ads 상표 불만 신고

- 상표 등록 증명 첨부
- 광고주 도메인 또는 광고 URL 기재
- 광고 텍스트 캡처 첨부

### Nominet DRS (.uk 도메인)

- 분쟁 도메인명 기재
- Nominet DRS 신청서 작성 필요
- 상표권 증명 및 침해 근거 서면 첨부

### ICANN UDRP (기타 gTLD 도메인)

- WIPO 또는 승인 분쟁 해결 기관 선택
- Complaint 서면 작성 필요 (변호사 권고)
