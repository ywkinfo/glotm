# Appendix D. RACI Template

이 템플릿은 영국 상표 운영에서 최소 역할 구조를 빠르게 잠그기 위한 기본형이다. 실제 조직도보다 `누가 입력하고, 누가 승인하고, 누가 외부와 말하는지`를 먼저 고정하는 데 목적이 있다.

| 업무 | 브랜드팀 | 법무/IP | 사업팀 | 외부 대리인 | 경영진 |
| --- | --- | --- | --- | --- | --- |
| 후보명 우선순위 결정 | R | C | C | I | A |
| 검색 브리프 작성 | R | A | C | C | I |
| goods/services 확정 | C | A | R | C | I |
| 출원 제출 | I | A | I | R | I |
| 심사 대응 방향 결정 | C | A | C | R | I |
| opposition 합의 여부 결정 | C | A | C | C | R |
| evidence vault 점검 | C | A | R | I | I |
| 갱신 유지/포기 결정 | C | A | C | I | R |
| 플랫폼 / 도메인 escalation | C | A | R | C | I |
| 소송 포럼 선택 | I | A | C | C | R |

## 운영 메모

- `R`은 실제 자료를 만든다.
- `A`는 최종 판단을 내린다.
- `C`는 반드시 의견을 제공한다.
- `I`는 결과를 통지받는다.

## 도입 체크리스트

- 각 업무에 `A`가 정확히 1명인지 확인
- 외부 대리인에게 `A`가 몰려 있지 않은지 확인
- 월간 보고 owner와 분기 리뷰 owner를 분리할지 결정
- 담당자 교체 시 handover owner를 미리 지정
