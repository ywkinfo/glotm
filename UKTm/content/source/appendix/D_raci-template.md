# Appendix D: RACI 운영 템플릿

Ch13의 RACI 표를 조직 구조에 맞게 커스터마이징하기 위한 기본 템플릿이다.

## 역할 정의 (조직에 맞게 수정)

| 역할 코드 | 역할명 | 실제 담당자/부서 | 응답 기한 기준 |
|---------|-------|--------------|-------------|
| BO | 브랜드 Owner | | 24시간 이내 |
| LO | 법무 Owner | | 48시간 이내 |
| BO2 | 사업 Owner | | 48시간 이내 |
| EA | 외부 대리인 (영국) | | 계약상 SLA 기준 |
| FA | 최종 승인자 | | 72시간 이내 |

## RACI 기호 정의

| 기호 | 의미 |
|------|------|
| R | Responsible: 실제 수행 담당 |
| A | Accountable: 최종 결과 책임, 승인권자 |
| C | Consulted: 의견 제공, 검토 참여 |
| I | Informed: 결과 공유 대상 |

## 업무 추가 가이드

조직 특성에 따라 아래 업무를 추가로 정의하는 편이 좋다.

- 서브브랜드 출원 결정
- Madrid Protocol 국제출원 지정 국가 결정
- 라이선스 계약 체결 승인
- 세관 국경조치 신청
- 외부 대리인 교체 결정

## 에스컬레이션 트리거

아래 상황에서는 최종 승인자(FA)에게 즉시 에스컬레이션한다.

| 트리거 | 에스컬레이션 기준 |
|--------|---------------|
| 전략적 양보 발생 | 핵심 표장 범위 조정, 상표 수정 필요 시 |
| 소송 결정 | IPEC 또는 High Court 제소 전 |
| 대형 침해 발견 | 핵심 유통 채널 또는 매출 상위 제품 관련 |
| 포트폴리오 대규모 변경 | 10건 이상 갱신 포기 또는 신규 대규모 출원 |
| 외부 대리인 교체 | 전체 포트폴리오 이관 수반 시 |
