# 사용증거와 Non-Use Cancellation 운영

영국 실무에서 사용 문제는 등록 후에 갑자기 생기는 것이 아니라, 출원 전략 단계부터 준비돼 있어야 한다. 실제 사용 구조를 정리하지 않으면 non-use cancellation 방어가 어려워지고, 플랫폼이나 협상에서도 권리 설명력이 떨어진다.

이 장은 사용증거를 "분쟁용 파일"이 아니라 평소 운영 산출물로 만드는 방법을 다룬다. evidence hygiene가 좋을수록 분쟁 비용이 줄어든다.

## 5년 사용 요건: 핵심 법적 기준

영국 상표법의 non-use cancellation 제도에서 가장 중요한 기준은 **등록일로부터 5년**이다.

| 기준 | 내용 |
|------|------|
| 취소 신청 가능 시점 | 등록일로부터 5년 경과 후 |
| 방어 요건 | 해당 5년 기간 동안 영국 내 진정한 사용(genuine use) 존재 |
| 사용의 범위 | 등록된 지정 상품·서비스에 대한 실제 상업적 사용 |
| 면제 사유 | 사용하지 않은 정당한 이유(proper reasons for non-use) 존재 시 |
| 일부 취소 | 사용이 입증된 상품·서비스만 유지, 나머지 취소 가능 |

등록 후 5년이 지나면 누구든지 UKIPO에 non-use cancellation을 신청할 수 있다. 경쟁사가 출원을 방해하거나 분쟁을 유리하게 이끌기 위해 이를 활용하는 경우도 있다. 따라서 5년 도달 전부터 사용증거를 정기적으로 정리해 두는 것이 중요하다.

## 어떤 증거를 모을지 정하기

사용증거는 화면 캡처만이 아니라 판매 페이지, 패키지, 거래 문서, 유통 증빙, 마케팅 자료처럼 복수의 층으로 쌓이는 편이 좋다. 한 종류 자료만 쌓아 두면 실제 분쟁에서 설득력이 약해질 수 있다.

| 증거 유형 | 예시 | 수집 주기 |
|-----------|------|----------|
| 판매 증빙 | 인보이스, 주문서, 영수증 | 분기별 |
| 온라인 사용 | 판매 페이지 스크린샷 (날짜 포함) | 분기별 |
| 마케팅 자료 | 광고, 카탈로그, 소셜미디어 게시물 | 반기별 |
| 패키지·라벨 | 실제 제품 포장 이미지 | 변경 시마다 |
| 유통 계약 | 리셀러·파트너 계약서 (표장 표시 확인) | 계약 갱신 시 |
| 전시·행사 | 영국 내 참가 이벤트 자료 | 행사 직후 |

브랜드팀과 사업팀이 자연스럽게 만들고 있는 자료를 법무가 재사용할 수 있게 연결해 두는 것이 핵심이다.

## evidence vault 운영

증거 보관함은 단순 폴더가 아니라 검색 가능한 기록이어야 한다. 표장, 기간, 상품군, 채널 기준으로 빠르게 찾을 수 있어야 cancellation 위협이나 협상 요청에 바로 대응할 수 있다.

실무적으로는 분기별 정리 리듬을 두고, 주요 표장별로 최신 증거 묶음을 유지하는 편이 좋다. 중요한 것은 완벽한 자료보다 `바로 꺼낼 수 있는 자료`다.

## non-use 리스크를 미리 줄이기

등록 후 실제 사용이 약할 것으로 보이는 권리는 초기부터 명세 범위를 무리하지 않는 편이 좋다. 지나치게 넓은 범위는 나중에 방어 부담으로 돌아오기 쉽다.

포트폴리오 리뷰 때마다 아래 세 항목을 같이 보면 cancellation 취약 권리를 미리 찾을 수 있다.

| 점검 항목 | 위험 신호 |
|----------|----------|
| 실사용 강도 | 최근 2년간 사용 증거 없음 |
| 확장 예정 | 사업팀에서 해당 상품군 계획 없음 |
| 방어 필요성 | 경쟁사 유사 출원 또는 시장 침해 없음 |

방어는 사건이 터진 뒤보다 평소 정비가 훨씬 싸다.
