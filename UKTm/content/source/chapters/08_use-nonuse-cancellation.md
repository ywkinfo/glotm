# 사용증거와 Non-Use Cancellation 운영

영국 실무에서 사용 문제는 등록 이후 갑자기 생기는 것이 아니라 출원 단계부터 누적된다. 상표가 등록됐더라도 사용증거를 정리하지 않으면 non-use revocation 대응이 약해지고, 플랫폼 신고나 협상에서도 권리 설명력이 떨어진다.

핵심 기준은 단순하다. 등록 후 5년이 지난 권리는 연속 5년 불사용이 문제될 수 있으므로, `언제부터 어떤 채널에서 어떤 표장으로 썼는지`를 꾸준히 설명할 수 있어야 한다.

## 5년 사용 요건을 운영 기준으로 바꾼다

| 항목 | 운영상 의미 | 실무 메모 |
| --- | --- | --- |
| 등록 후 5년 | non-use revocation이 현실화되는 첫 경계 | 4년 차부터 증거 상태 점검 시작 |
| 연속 5년 불사용 | 가장 위험한 신호 | 실제 판매가 약하면 명세 유지 전략 재검토 |
| 일부 goods/services만 사용 | 전부가 아니라 일부 범위만 흔들릴 수 있음 | 클래스 전체가 아니라 실제 사용 범위 기준으로 방어 준비 |

## 어떤 증거를 모을지 미리 정한다

사용증거는 한 종류만 모으면 약하다. 판매 화면, 패키지, 거래문서, 광고소재, 유통 기록이 함께 있어야 영국 시장에서의 genuine use story가 자연스럽게 이어진다.

| 증거 유형 | 가장 좋은 용도 | 놓치기 쉬운 점 |
| --- | --- | --- |
| 판매 페이지 캡처 | 온라인 판매 존재 설명 | 날짜와 URL이 같이 남아야 함 |
| 패키지·라벨 | 표장 사용 형태 설명 | 등록 형태와 차이가 큰지 확인 |
| 인보이스·PO | 거래 실재성 설명 | 영국 관련 거래인지 태그 필요 |
| 광고·SNS 소재 | 시장 노출 설명 | 실제 판매와 연결고리도 같이 확보 |
| 유통·출하 자료 | 채널과 지역 설명 | EU 자료와 UK 자료를 구분 |

## 최소 패킷 단위를 먼저 정한다

영국에서 genuine use를 설명할 때는 자료가 많다는 사실보다 자료가 서로 이어진다는 점이 중요하다. 최소 패킷을 아래처럼 고정해 두면 weak-use mark를 점검할 때도 훨씬 빠르다.

| 항목 | 최소 예시 | 빠지면 생기는 문제 |
| --- | --- | --- |
| 판매 페이지 | 영국 판매 페이지, UK 배송 안내, marketplace listing | 실제 영국 노출 문맥이 약해짐 |
| 패키지·라벨 | 제품 사진, 포장, 라벨 | 등록형과 실제 사용형 연결이 흔들림 |
| invoice / PO | 영국 거래 관련 문서 | 거래 실재성과 기간 설명이 약해짐 |
| 광고 | UK 대상 SNS, promo asset, campaign log | 시장 노출 설명이 약해짐 |
| 유통 자료 | seller, distributor, shipment memo | 채널과 지역 설명이 느려짐 |
| owner-user linkage | licence, approval line, actual user note | 누구의 사용인지 설명이 비어 버림 |

## evidence cadence를 분기 루틴으로 만든다

증거 관리가 가장 잘 되는 팀은 사건이 생겼을 때 모으지 않고 분기 루틴으로 정리한다. `표장`, `상품군`, `채널`, `기간`, `파일 위치`만 표준 카드에 넣어도 cancellation 대응 준비도가 크게 올라간다.

실제 운영에는 `Appendix B` evidence vault card를 그대로 복사해 쓰면 분기 업데이트 기준을 맞추기 쉽다.

### evidence vault card를 운영 카드로 올린다

카드는 `무엇이 있나`를 적는 수준에서 끝나면 부족하다. 실제 운영에서는 아래 필드까지 들어가야 분기 점검과 취소 방어가 연결된다.

| 필드 | 적어 둘 내용 |
| --- | --- |
| mark version | word / logo / composite |
| channel | D2C, retail, distributor, marketplace |
| proof | 포장, 판매 화면, invoice, promo asset |
| last checked | 최근 검토일 |
| owner / actual user | 권리자와 실제 사용 주체 |
| source URL or doc id | listing URL, invoice id, shipment ref |
| territory tag | UK / EU / Global |
| file status | 원본 / 제출용 / working memo |

| 주기 | 해야 할 일 | 담당 |
| --- | --- | --- |
| 월간 | 신규 launch와 seller 자료 수집 | 사업팀 / 브랜드팀 |
| 분기 | evidence vault card 업데이트 | 법무/IP |
| 반기 | weak-use mark 선별 | 법무 / 사업팀 |
| 갱신 전 | 사용 강도와 유지 필요성 재평가 | 법무 / 경영진 |

## non-use 리스크를 줄이는 체크리스트

- 등록 후 4년 차 진입 mark를 자동 알림으로 잡아두는가
- 증거를 `UK`, `EU`, `Global`로 태깅하는가
- 실제 사용이 약한 클래스·상품군을 포트폴리오 리뷰 때 분리해 보는가
- cancellation 위협이 오기 전에 evidence vault card가 최신 상태인가
- comparable UK right의 경우 2021년 이후 UK 사용 설명을 별도 준비하는가

## 실무 예시

예를 들어 영국에서는 아직 소량 판매만 하지만 EU 판매는 활발한 브랜드라면, EU 자료만으로 영국 사용을 당연히 설명할 수 있다고 가정하면 위험하다. 영국 판매 페이지, 영국 배송 기록, 영국 대상 마케팅 자료를 분리해 모아 두면 같은 상표라도 방어 난이도가 크게 달라진다.
