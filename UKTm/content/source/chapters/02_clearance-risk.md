# 출원 전 검색과 충돌 분석

영국 상표 검색은 단순 조회가 아니라 `출시 일정과 법적 리스크를 같은 표에서 보는 단계`다. 검색 결과가 깨끗해 보여도 명세나 사용 맥락이 다르면 후속 리스크가 남을 수 있고, 반대로 일부 충돌이 있어도 우선순위와 범위 조정으로 충분히 추진 가능한 경우도 있다.

영국에서는 최소한 UKIPO 검색, TMview, Madrid Monitor, Trade Marks Journal 흐름을 묶어서 보는 편이 실무적이다. 영국 단일 출원만 준비하더라도 비교군을 영국 register 밖으로 넓혀야 launch 이후 비용을 줄일 수 있다.

## 어떤 검색 채널을 조합할지 먼저 정한다

검색 채널마다 답하는 질문이 다르다. `Search for a trade mark`는 영국 register 중심의 실무 검색이고, TMview는 EU 및 다국가 포트폴리오와의 충돌 탐색에 좋다. Madrid Monitor는 국제등록을 보는 용도라 UK designation을 넣을지 판단할 때 의미가 커진다.

| 채널 | 공식 명칭 | 가장 좋은 용도 | 놓치기 쉬운 점 |
| --- | --- | --- | --- |
| UK register 검색 | Search for a trade mark | 동일·유사 UK right 확인 | UK 밖 포트폴리오 동향은 별도 확인 필요 |
| 클래스 탐색 | Search UK trade mark classes | 명세 초안과 클래스 정리 | 사업 언어를 그대로 넣으면 과도하게 넓어질 수 있음 |
| 다국가 검색 | TMview | EU 및 타국 register 비교 | 표시 언어와 transliteration 변형을 놓치지 않도록 보완 검색 필요 |
| 국제등록 검색 | Madrid Monitor | IR 지정 현황과 UK designation 여부 확인 | EU 지정만으로 영국을 커버하는지 착각하기 쉬움 |
| 공고 확인 | Trade Marks Journal | 공고 직후 opposition 리스크 감시 | 출원 직후가 아니라 공고 시점을 기준으로 알림 설계 필요 |

## 검색 범위를 사업 구조에 맞춘다

표장 후보가 여러 개일 때는 브랜드팀이 먼저 1순위, 2순위, 완전 대체안을 구분해야 한다. 검색 담당자가 후보군을 스스로 줄이게 두면 결과가 길어지고, 내부 의사결정은 오히려 늦어진다.

상품·서비스 범위는 "현재 판매"만이 아니라 12개월 내 실제 사용 계획까지 포함해 잡는 편이 낫다. 너무 좁으면 재출원이 늘고, 너무 넓으면 search noise가 과도하게 커진다.

## 검색 결과는 RAG로 정리한다

실무자는 유사도 분석을 장문 메모로만 받기보다 red / amber / green 구조로 받아야 빠르게 움직일 수 있다. 경영진이나 출시팀은 모든 판례 논리를 읽지 않으므로, `어떤 선택지를 남길지`가 먼저 보여야 한다.

| 등급 | 전형적 상황 | 권장 액션 |
| --- | --- | --- |
| Red | 동일·고유사 표장, 근접 클래스, launch 일정 촉박 | 표장 변경 또는 명세·시장 순서 재설계 검토 |
| Amber | 일부 유사, 사용 맥락 차이 있음, 협상 가능성 존재 | 조건부 출원, fallback 준비, coexistence 가능성 검토 |
| Green | 충돌도 낮고 범위도 합리적 | 출원 브리프 확정 후 바로 진행 |

## 검색 브리프에서 빠지면 안 되는 항목

검색은 자료가 좋아야 결과도 좋아진다. 외부 대리인에게 검색을 맡길 때는 표장 후보, 실제 사용 설명, 클래스 가설, 출시일, 포기 가능한 범위를 한 문서에 넣어야 한다.

대표적인 실패 패턴은 "검색은 했는데 무엇을 결정해야 하는지 모르는 상태"다. 검색 결과를 받은 날 바로 출원 경로와 fallback을 정하려면 처음 브리프에서 질문을 명시해야 한다.

검색 브리프를 바로 복사해 쓰려면 `Appendix A` 템플릿을 기본 포맷으로 삼는 편이 가장 빠르다.

## 검색 브리프 체크리스트

- 우선 표장과 대체 표장을 구분했는가
- UK only인지 UK+EU인지, 또는 Madrid 검토인지 명시했는가
- 12개월 내 실제 사용 계획을 적었는가
- `red / amber / green` 평가를 요청했는가
- 결과 보고서에서 `즉시 출원`, `조건부 출원`, `보류/변경` 중 어느 선택지를 추천할지 요구했는가
