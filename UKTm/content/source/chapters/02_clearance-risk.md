# 출원 전 검색과 충돌 분석

영국 상표 실무에서 검색은 단순 조회가 아니라 launch 전에 `어떤 리스크를 받아들일지` 정하는 단계다. 검색 결과가 깨끗해 보인다고 바로 안전한 것은 아니고, 반대로 유사한 선행표장이 있다고 해서 항상 출원을 포기해야 하는 것도 아니다.

이 장은 검색을 "출원 yes/no 판단"보다 "충돌 시나리오 정리"로 보는 관점을 다룬다. 실무자는 검색 단계에서 채널, 카테고리, 사용 예정 표장 형태를 함께 봐야 한다.

## 검색 채널 선택하기

영국 상표 검색은 단일 채널로 완결되지 않는다. 공식 데이터베이스와 시장 상황을 함께 봐야 충돌 후보를 제대로 잡을 수 있다.

| 검색 채널 | 목적 | 특징 |
|-----------|------|------|
| UKIPO Trade Marks Journal 및 공식 검색 | 영국 등록·출원 상표 조회 | 무료, 정본 데이터 |
| TMview | 영국 포함 다국 등록 상태 통합 조회 | EUIPO 운영, 영국 데이터 포함 |
| Madrid Monitor | Madrid System 국제출원 현황 | WIPO 운영, 영국 지정 확인 가능 |
| Companies House | 상호명 충돌 가능성 | 상표권과 별개이나 혼동 이슈 존재 |
| 시장 검색 (Google, Amazon 등) | 미등록 선사용 표장 탐지 | 미등록이어도 이의신청 가능한 경우 있음 |

TMview는 EUIPO가 운영하지만 영국 등록 데이터도 포함하고 있어, 다국 비교가 필요한 경우 시작점으로 활용하기 좋다. 단, 정본은 항상 UKIPO 데이터베이스에서 다시 확인해야 한다.

## 검색 범위를 먼저 설계하기

검색은 단어 표장, 철자 변형, 발음 변형, 로고 결합 가능성까지 최소한의 범위를 정한 뒤 시작하는 편이 좋다. 내부 명칭이 아직 흔들리는 브랜드라면 먼저 우선 표장 후보를 줄이고 검색을 들어가야 결과를 해석하기 쉽다.

상품/서비스 범위를 너무 좁게 보면 launch 이후 확장 리스크를 놓치고, 너무 넓게 보면 의미 없는 충돌 후보가 쌓인다. 검색 범위는 사업팀이 예정하는 12개월 내 사용 계획과 맞춰 잡는 편이 현실적이다.

## 충돌 후보 해석 기준 세우기

검색 결과는 "같다/다르다"보다 "어떤 대응 옵션이 가능한가"로 읽는 편이 좋다. 표장 유사성, 지정 범위, 실제 사용 맥락, 시장 근접성, 협상 가능성, 런치 일정 압박을 함께 보면 우선순위가 달라진다.

실무자는 충돌 후보를 아래 세 단계로 분류해 정리하면 이후 출원 전략 장과 자연스럽게 연결된다.

| 등급 | 기준 | 기본 대응 방향 |
|------|------|--------------|
| Red (즉시 차단 가능성) | 동일·유사 표장, 동일·유사 지정 범위, 실제 시장 근접 | 표장 수정 또는 범위 조정 필요, 출원 전 협의 검토 |
| Amber (조건부 진행 가능성) | 유사성 있으나 시장 구분 가능, 협상 여지 존재 | 추가 의견 구비 후 진행, 출원 후 모니터링 강화 |
| Green (모니터링 대상) | 유사성 낮고 시장 거리 있음, 미등록 또는 비사용 | 출원 진행, 주기적 감시 |

검색 결과 메모는 변호사 브리프의 초안으로 이어질 수 있게 남기는 편이 좋다.

## 검색 결과를 의사결정으로 연결하기

검색 단계의 목적은 완벽한 확신이 아니라, launch와 출원 시퀀스를 합리적으로 정하는 것이다. 위험도가 높은 시장이면 표장 수정이나 카테고리 조정이 더 빠를 수 있고, 일정이 촉박하면 우선 출원 후 감시 전략을 강화하는 조합이 나올 수 있다.

검색 리포트는 결과 요약 한 줄, 핵심 충돌 3개, 권장 액션 3개 정도로 축약돼야 내부 결정이 빨라진다. 장문의 검색 메모보다 경영진이 바로 이해할 수 있는 red / amber / green 구조가 실무에 더 유용하다.
