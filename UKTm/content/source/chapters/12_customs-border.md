# 세관, 국경조치, 병행수입 이슈

국경 단계의 상표 이슈는 평소에는 멀게 느껴지지만, 실제로는 온라인 침해와 유통 관리가 누적된 결과로 드러나는 경우가 많다. 영국으로 들어오는 위조품, 승인되지 않은 수입, 라벨 변경 상품은 플랫폼 문제와 별개가 아니라 연결된 공급망 문제로 보는 편이 정확하다.

영국에서는 HMRC에 Application for Action을 제출해 border intervention을 준비할 수 있다. 신청 자체는 무료지만, 실제 detention 이후에는 보관·폐기·후속 대응 비용이 따를 수 있으므로 `무조건 넣는다`가 아니라 `어떤 브랜드에 넣을지`를 먼저 정해야 한다.

## HMRC 대응이 필요한 브랜드를 선별한다

| 위험도 | 전형적 상황 | 권장 조치 |
| --- | --- | --- |
| High | counterfeiting 흔함, 계절성 launch, safety risk 있음 | HMRC AFA + 플랫폼 + 유통 점검 동시 운영 |
| Medium | 병행수입, 반복 reseller, 패키지 변형 우려 | 제품 식별 자료와 연락 체계 준비 |
| Low | UK 유통이 제한적이고 위조 위험 낮음 | 상시 모니터링 중심 |

## HMRC 사전 준비물 표

HMRC 대응은 사건 뒤에 설명하는 것보다 미리 패킷을 만들어 두는 편이 훨씬 빠르다. genuine product 식별 자료와 contact map이 핵심이다.

| 준비물 | 왜 필요한가 |
| --- | --- |
| 등록번호 / 권리 증빙 | 세관이 권리 기초를 빠르게 확인 |
| genuine product 이미지 | 현장 식별 지원 |
| 위조 의심 포인트 | 라벨, 박스, 시리얼, 패키지 차이 설명 |
| 공급망 설명 | 승인된 수입 루트인지 판단 |
| 긴급 연락처 | detention 시 즉시 판단 |

## Brexit 이후 border 운영 변화

Brexit 이후에는 EU 권리와 UK border action을 같은 이벤트로 보면 안 된다. 영국으로 들어오는 물류는 HMRC 기준으로 별도 관리해야 하므로, EU AFA 경험이 있더라도 UK용 패킷을 다시 설계해야 한다.

| 항목 | 운영 포인트 |
| --- | --- |
| AFA 제출 타이밍 | 최소 30 working days 전 제출 권장 |
| 권리 번호 | UK national right인지 comparable UK right인지 구분 |
| 연락체계 | UK 물류·세관 대응 담당자 분리 |
| 제품 자료 | 영국에서 실제 유통되는 genuine version 기준으로 준비 |

## 병행수입 판단 프레임

병행수입은 위조와 달리 genuine goods일 수 있어 자동으로 같은 대응을 쓰면 안 된다. 따라서 공급망 승인 구조, 최초 판매 지역, 재포장 또는 안전 이슈, 영국 소비자 혼동 가능성을 한 번에 보는 질문표가 필요하다.

| 질문 | Yes면 더 볼 것 | No면 더 볼 것 |
| --- | --- | --- |
| genuine goods인가 | exhaustion / approval / relabelling | counterfeit route 검토 |
| 영국 유통 승인 구조와 맞는가 | distributor agreement | unauthorized import triage |
| 재포장 또는 안전 리스크가 있는가 | 즉시 escalation | 일반 commercial issue 가능성 |

## 세관 대응 체크리스트

- HMRC에 넣을 우선 브랜드를 정했는가
- genuine vs suspicious product 비교 자료를 최신화했는가
- 병행수입과 위조를 같은 사건으로 섞지 않는가
- 플랫폼·유통·세관 대응이 같은 사건 보드에서 연결되는가
