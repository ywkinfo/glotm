# EuTm 초기 구조 메모

## 목표

- `MexTm`처럼 읽기 가능한 단일 마스터 원고 구조로 먼저 시작한다.
- 루트 `GloTm` 셸 연결은 보류하고, 권역 범위와 리서치 축을 먼저 고정한다.
- 후속 단계에서 필요하면 `LatTm`식 manifest + QA 파이프라인으로 승격한다.

## 현재 가정

- `EuTm`은 유럽 권역 가이드다.
- 초기 본문은 완성본이 아니라 구조 초안이다.
- 유럽은 권역형 성격이 강하므로, 장기적으로는 `MexTm`보다 `LatTm`에 가까운 파이프라인이 필요할 가능성이 높다.

## 후속 작업 후보

1. 챕터 구조 고정
2. 1차 출처 레지스터 보강
3. fact verification log 항목 채우기
4. `EuTm` 전용 build smoke 유지
5. 루트 셸 연결 여부 재판단
