# 제07장. Post-registration: 등록 후 유지관리 시스템

> **이 장의 목적**: 상표 등록 이후 권리를 유지하고 소멸을 방지하는 운영 체계를 구축한다.
> 등록은 시작이다. 갱신·사용선언·사용증거 관리를 놓치면 등록된 권리가 소멸된다.
> 06장(공고·이의·거절 대응) 완료 후 이 장을 진행하고, 08장(상표 사용 및 증거 관리)과 병행 운영한다.

---

## 1. 핵심 원칙

> **"등록 이후가 진짜 시작이다. 관리하지 않는 상표는 소멸한다."**

중남미에서 상표 권리 소멸의 가장 흔한 원인은 침해가 아니라 **내부 관리 실패**다.
이 단계의 실패는 세 가지 형태로 나타난다.

첫째, 갱신 기한을 놓쳐 등록이 만료된다. 국가마다 갱신 주기와 유예 기간이 다르며, 유예 기간 내 갱신 시 추가 비용이 발생하고 유예 기간도 도과하면 권리가 완전히 소멸한다.

둘째, 멕시코 등 일부 국가의 사용선언 의무를 이행하지 않아 등록이 취소된다. 사용선언은 등록 후 정해진 기간 내에 반드시 제출해야 하는 법정 의무이며, 기한을 놓치면 등록이 소멸한다.

셋째, 불사용 취소 청구에 대비한 사용증거를 관리하지 않아 경쟁사의 취소 청구에 무방비로 노출된다.

등록 후 유지관리는 "캘린더 기반 기한 관리 + 사용증거 축적 + 포트폴리오 현행화"의 세 축으로 운영한다.

---

## 2. 국가 범위 및 적용 조건

이 장의 절차는 등록이 완료된 모든 국가에 적용된다.
국가마다 갱신 주기, 사용선언 의무, 불사용 취소 요건이 다르므로 국가별 별도 캘린더를 운영한다.

| 국가 | 상표청 | 갱신 주기 | 유예 기간 | 사용선언 의무 | 불사용 취소 기준 |
|------|--------|-----------|-----------|--------------|-----------------|
| 브라질 | INPI | 10년 | 6개월 (추가 비용) | 없음 | 등록 후 5년 불사용 |
| 멕시코 | IMPI | 10년 | 6개월 (추가 비용) | **있음** (등록일로부터 3년 내) | 3년 불사용 |
| 콜롬비아 | SIC | 10년 | 6개월 (추가 비용) | 없음 | 3년 불사용 |
| 아르헨티나 | INPI-AR | 10년 | 6개월 (추가 비용) | **있음** (갱신 시 사용 입증) | 5년 불사용 |
| 칠레 | INAPI | 10년 | 30일 (추가 비용) | 없음 | 5년 불사용 |
| 페루 | INDECOPI | 10년 | 6개월 (추가 비용) | 없음 | 3년 불사용 |
| 에콰도르 | SENADI | 10년 | 6개월 (추가 비용) | 없음 | 3년 불사용 |
| 파나마 | DIGERPI | 10년 | 6개월 (추가 비용) | 없음 | 5년 불사용 |

> **운영 원칙**: 모든 기한은 **등록 완료일 기준**으로 계산한다.
> 내부 마감은 법정 갱신 기한 **-180일(6개월 전)**, 사용선언은 법정 기한 **-90일(3개월 전)** 로 설정한다.
> 칠레는 유예 기간이 30일로 매우 짧으므로 내부 마감을 **-270일(9개월 전)** 로 설정한다.

## 3. 주요 리스크

### 3-1. 갱신 기한 누락 리스크

| 리스크 유형 | 설명 | 영향 국가 |
|-------------|------|-----------|
| **갱신 기한 미등록** | 등록 완료 후 갱신 일정을 캘린더에 입력하지 않음 | 전 국가 |
| **대리인 교체 시 이력 단절** | 대리인 변경 후 기존 갱신 일정이 새 대리인에게 인계되지 않음 | 전 국가 |
| **갱신 비용 예산 미반영** | 재무팀이 갱신 비용을 연간 예산에 포함하지 않아 기한 내 처리 불가 | 전 국가 |
| **유예 기간 오인** | 칠레 유예 기간(30일)을 타국(6개월)과 동일하게 오인 | 칠레 |
| **다수 국가 동시 갱신 집중** | 동시 진출한 국가들의 갱신이 동시에 도래 → 처리 역량 부족 | 멕시코·브라질 동시 진출 기업 |

**실행 지침**: 등록 완료 즉시 갱신 기한을 중앙 캘린더에 입력한다. 기한은 3단계로 등록한다: ① 법정 마감일, ② 내부 마감일(법정 -6개월), ③ 착수 알림일(법정 -9개월).

### 3-2. 사용선언 누락 리스크

| 리스크 유형 | 설명 | 영향 국가 |
|-------------|------|-----------|
| **사용선언 제도 미인지** | 멕시코·아르헨티나의 사용선언 의무를 모르고 출원함 | 멕시코, 아르헨티나 |
| **기한 오인** | 멕시코 사용선언 기한(등록일로부터 3년째 되는 해의 등록월)을 잘못 계산 | 멕시코 |
| **사용 실적 미축적** | 사용선언 기한이 도래했을 때 제출할 사용증거가 없음 | 멕시코, 아르헨티나 |
| **사용선언 대상 표장 누락** | 멀티 브랜드 운영 기업에서 일부 표장의 사용선언을 누락 | 멕시코 |
| **아르헨티나 갱신 시 사용 입증 준비 부족** | 갱신 시 사용 실적을 증명해야 한다는 사실을 갱신 직전에야 인지 | 아르헨티나 |

**실행 지침**: 멕시코 등록 즉시 사용선언 기한을 캘린더에 입력하고 담당자를 지정한다. 사용선언 기한은 등록일로부터 **3년째 연도의 해당 등록월 마지막 날**이다. 내부 마감은 **3개월 전**으로 설정한다.

### 3-3. 불사용 취소 리스크

| 리스크 유형 | 설명 | 영향 국가 |
|-------------|------|-----------|

| **사용 형태 변경 미반영** | 등록 표장과 실제 사용 표장이 달라져 사용 증거로서 효력 문제 | 전 국가 |

## 4. Decision Box

### 4-1. 갱신 전략 판단

```
[Decision Box A: 갱신 진행 여부 결정]

IF (해당 국가에서 현재 판매 중이거나 12개월 내 판매 계획이 있음)
    → 전액 갱신 진행
      갱신 시 지정 상품·서비스 범위를 현재 사업 범위에 맞게 재검토

IF (해당 국가에서 판매 중단 후 5년 이상 경과 + 재진입 계획 없음)
    → 갱신 포기 검토
      포트폴리오 관리 대장에 포기 사유를 기재하고 IP 총괄 승인을 받는다
      단, 타사에 의한 선점 리스크가 남아 있으면 방어 등록 유지 여부를 별도 결정

IF (해당 국가에서 판매 없으나 경쟁사 선점 리스크가 높음)
    → 방어 등록 유지 목적으로 갱신 진행
      단, 불사용 취소 리스크를 방지하기 위해 소규모 사용 또는 라이선스 구조 검토

IF (갱신 예산 부족으로 전 국가 갱신이 불가)
    → 시장 규모 × 리스크 매트릭스로 우선순위 결정
      브라질·멕시코는 예산 부족과 무관하게 갱신 최우선 처리

IF (지정 상품·서비스 범위가 현재 사업과 불일치)
    → 갱신 시 범위 조정 여부를 현지 대리인과 협의
      범위 축소 시 향후 추가 류 출원 비용 발생 가능성 검토
```

### 4-2. 사용선언 전략 판단 (멕시코·아르헨티나 해당)

```
[Decision Box B: 멕시코 사용선언 대응]

IF (등록일로부터 3년째 연도 진입 + 실제 사용 실적 있음)
    → 사용선언 즉시 준비 개시
      현지 대리인에게 사용선언 기한 확정값을 서면으로 수령
      사용 증거 목록을 국내 IP팀이 준비하고 현지 대리인에게 전달

IF (등록일로부터 3년째 연도 진입 + 실제 사용 실적 없음)
    → 사용선언 기한까지 사용 개시 가능 여부를 영업팀과 즉시 확인
      사용 개시 불가 → 해당 표장의 포트폴리오 존치 여부를 결정
      사용선언 기한 도과 시 등록 소멸 → 재출원 시 우선일 소멸

IF (멕시코에서 라이선스 구조로 사용 중)
    → 라이선시의 사용 실적을 사용선언 근거로 활용
      라이선스 계약서 + 라이선시의 사용 증거를 패키지로 준비

[Decision Box C: 아르헨티나 갱신 시 사용 입증]

IF (갱신 기한 도래 + 아르헨티나에서 실제 판매 중)
    → 사용 증거(인보이스, 광고, 판매 기록)를 갱신 6개월 전부터 정리

IF (갱신 기한 도래 + 아르헨티나 판매 없음)
    → 갱신 포기 또는 라이선스 구조 도입으로 사용 실적 생성 여부 검토
      포기 결정 시 IP 총괄 승인 후 포트폴리오 대장에 기록
```

### 4-3. 포트폴리오 정리 판단

```
[Decision Box D: 표장 유지 vs 포기 결정]

IF (해당 표장이 현재 사용 중이거나 3년 내 사용 예정)
    → 유지 (갱신 진행)

IF (해당 표장이 사용 중이나 등록 표장과 실제 사용 표장이 다름)
    → 현지 대리인에게 사용 형태 변경의 영향 여부 확인
      영향 있으면 표장 보정 출원 또는 현행 표장으로 추가 출원 검토

IF (해당 표장이 완전히 사용 중단되었고 재사용 계획 없음)
    → 포기 전 타사 선점 가능성 평가
      선점 리스크 낮음 → 포기 진행 (관리 비용 절감)
      선점 리스크 높음 → 방어 등록 유지 또는 타사에 매각/이전 검토

IF (포트폴리오 전체 점검이 필요한 시점 → 연 1회 이상)
    → Decision Box D 전체를 포트폴리오 대장 기준으로 표장별 순환 실행
```

## 5. 실행 절차

### Step 1. 등록 완료 즉시 캘린더 등록 (등록 완료 후 5영업일 이내)

등록증을 수령하면 5영업일 이내에 다음 절차를 완료한다.

1. 등록 정보(등록번호, 등록일, 만료일, 지정 상품류)를 포트폴리오 관리 대장에 입력한다.
2. 중앙 캘린더에 다음 3개 일정을 등록한다:
   - **갱신 착수 알림일**: 만료일 기준 -9개월
   - **내부 마감일**: 만료일 기준 -6개월
   - **법정 만료일**: 정확한 날짜
3. 멕시코 등록인 경우 추가로 사용선언 기한을 등록한다:
   - **사용선언 착수 알림일**: 법정 기한 기준 -3개월
   - **사용선언 내부 마감일**: 법정 기한 기준 -1개월
   - **사용선언 법정 기한**: 등록일로부터 3년째 연도의 해당 등록월 마지막 날
4. 아르헨티나 등록인 경우 갱신 시 사용 입증 준비 시작일을 추가로 등록한다(만료일 기준 -12개월).

**담당**: 국내 IP팀(R), IP 총괄(A), 현지 대리인(C)

---

### Step 2. 사용선언 준비 및 제출 (멕시코·아르헨티나 해당, 법정 기한 -3개월부터)

사용선언 착수 알림일이 도래하면 즉시 준비를 시작한다.

1. 현지 대리인에게 사용선언 기한 확정값과 제출 요건을 서면으로 확인한다.
2. 국내 IP팀은 사용증거 목록을 작성한다: 인보이스, 수출신고서, 포장재, 광고물, 납품확인서.
3. 사용증거를 스페인어로 번역하고 필요 시 공증 또는 아포스티유를 진행한다.
4. 현지 대리인에게 사용증거 패키지를 법정 기한 -30일 이전에 전달한다.
5. 제출 완료 후 접수 확인서를 수령하고 증거 저장소에 보관한다.

**담당**: 국내 IP팀(R), 현지 대리인(R), IP 총괄(A), 영업팀(C)

| 사용증거 유형 | 번역 필요 | 공증·아포스티유 | 준비 주체 |
|---------------|-----------|----------------|-----------|
| 인보이스 (Invoice) | 스페인어 번역 필요 | 일부 국가 공증 필요 | 국내 IP팀 + 영업팀 |
| 수출신고서 | 스페인어 번역 필요 | 아포스티유 필요 | 국내 IP팀 + 물류팀 |
| 제품 포장재·라벨 | 해당 없음 | 해당 없음 | 국내 IP팀 |
| 광고·홍보 자료 | 스페인어·포르투갈어 번역 필요 | 해당 없음 | 마케팅팀 |
| 온라인 판매 캡처 | 해당 없음 | 해당 없음 | 국내 IP팀 |
| 현지 판매 계약서 | 현지어 원본 우선 | 일부 국가 공증 필요 | 법무팀 + 영업팀 |

---

### Step 3. 분기 포트폴리오 점검 (분기 1회)

분기마다 포트폴리오 전체를 점검하여 누락·이상 없음을 확인한다.

1. 포트폴리오 관리 대장에서 향후 12개월 내 도래하는 갱신·사용선언 기한 목록을 추출한다.
2. 각 국가별 현지 대리인에게 등록 현황을 확인하고 이상 사항을 보고받는다.
3. 사용 중단된 표장이 있는지 영업팀·마케팅팀에 확인한다.
4. Decision Box D를 실행하여 포기·유지·추가 출원 여부를 결정한다.
5. 점검 결과를 IP 총괄에게 보고하고 결과를 포트폴리오 대장에 기록한다.

**담당**: 국내 IP팀(R), IP 총괄(A), 현지 대리인(C), 영업·마케팅팀(C), 재무팀(I)

---

### Step 4. 갱신 준비 및 실행 (만료일 기준 -9개월부터)

갱신 착수 알림일이 도래하면 즉시 준비를 시작한다.

1. 현지 대리인에게 갱신 절차 요건과 비용을 서면으로 확인한다.
2. 갱신 대상 표장의 지정 상품·서비스 범위를 현재 사업 범위와 대조한다.
3. 범위 조정이 필요하면 Decision Box A를 실행한다.
4. 아르헨티나는 사용 실적 증거를 이 단계에서 최종 정리한다.
5. 갱신 수수료를 재무팀에 사전 요청하고 예산을 확보한다.
6. 갱신 신청서를 내부 마감일(-6개월) 이전에 현지 대리인에게 전달한다.
7. 갱신 완료 후 새 등록증을 수령하고 포트폴리오 대장의 만료일을 업데이트한다.

**담당**: 국내 IP팀(R), 현지 대리인(R), IP 총괄(A), 재무팀(C)

---

### Step 5. 갱신 완료 후 재등록 및 알림 재설정 (갱신 완료 후 5영업일 이내)

갱신이 완료되면 새 만료일 기준으로 Step 1을 반복 실행한다.

1. 새 등록증의 만료일을 포트폴리오 대장에 입력한다.
2. 중앙 캘린더의 기존 갱신 일정을 삭제하고 새 만료일 기준으로 재등록한다.
3. 갱신 완료 사실을 IP 총괄과 관련 팀(영업, 물류)에 통보한다.

**담당**: 국내 IP팀(R), IP 총괄(A), 현지 대리인(I)

### 멕시코(IMPI) 한정 추가 확인

- [ ] 등록 완료 즉시 사용선언 기한을 현지 대리인에게 서면으로 확인했다
- [ ] 사용선언 기한을 중앙 캘린더에 입력했다 (착수 알림 -3개월, 내부 마감 -1개월, 법정 기한)

- [ ] 사용선언 후 다음 사용선언 기한(갱신 주기와 별도)이 있는지 재확인했다

- [ ] 아르헨티나 인플레이션을 반영한 갱신 비용 최신값을 갱신 6개월 전 재확인했다
- [ ] 갱신 시 제출하는 사용 증거를 스페인어로 번역하고 공증 여부를 확인했다

- [ ] 갱신 대상 표장의 지정 상품류를 현재 브라질 사업 범위와 대조했다

## 9. 실패 사례 및 주의사항

### 실패 사례 1: 멕시코 사용선언 기한 오계산으로 등록 소멸

**상황**: 한국 소비재 기업 A사가 멕시코 IMPI에 상표를 등록했다. 담당자는 사용선언 기한을 "등록일로부터 정확히 3년"으로 계산하여 캘린더에 등록했다. 그러나 실제 기한은 "등록일로부터 3년째 연도의 해당 등록월 마지막 날"이었다. 담당자가 계산한 날짜보다 실제 기한이 3주 일찍 도래했고, 그 사이에 사용선언이 제출되지 않았다. IMPI는 해당 등록을 취소했다.

**원인**: 기한 계산을 국내 IP팀이 독자적으로 수행하고 현지 대리인에게 기한 확정값을 서면으로 재확인하지 않았다.

**재발 방지**: 멕시코 등록 즉시 현지 대리인에게 사용선언 기한 확정값을 서면으로 요청하고, 해당 값을 캘린더에 입력한다. 국내 IP팀이 독자적으로 기한을 계산하지 않는다.

---

### 실패 사례 2: 칠레 유예 기간(30일) 오인으로 권리 소멸

**상황**: 한국 화장품 기업 B사가 칠레 INAPI 등록 표장의 갱신 일정을 타 국가와 동일하게 "만료일 -6개월"로 내부 마감을 설정했다. 갱신 수수료 납부 지연이 발생했고, 법정 만료일 이후 유예 기간(30일)도 소화하지 못해 등록이 소멸되었다. 재출원 시 동일 표장을 현지 경쟁사가 먼저 선점하여 3년간 분쟁이 발생했다.

**원인**: 칠레의 유예 기간이 30일로 다른 국가(6개월)와 달리 짧다는 사실을 담당자가 인지하지 못했다.

**재발 방지**: 칠레 갱신 내부 마감은 만료일 -9개월로 설정한다. 칠레 전용 캘린더 알림을 별도로 운영하여 타 국가와 혼동되지 않도록 한다.

---

### 실패 사례 3: 아르헨티나 갱신 시 사용 입증 준비 부재

**상황**: 한국 전자기업 C사가 아르헨티나 INPI 등록 표장 갱신을 진행하면서 사용 실적 제출 요건을 갱신 신청 직전에야 인지했다. 당시 아르헨티나 법인의 판매 기록 정리가 되어 있지 않아 적절한 증거를 제출하지 못했다. 갱신은 수수료를 납부했으나 사용 입증 서류 미비로 갱신이 지연되었고, 유효 기간 공백이 발생하는 동안 경쟁사가 유사 표장을 출원했다.

**원인**: 아르헨티나 갱신이 단순 수수료 납부가 아닌 사용 입증을 포함한다는 사실을 갱신 착수 시점에야 파악했다.

**재발 방지**: 아르헨티나 등록 시점부터 갱신 시 사용 입증 요건을 포트폴리오 대장 비고란에 기재한다. 갱신 준비는 만료일 기준 12개월 전부터 시작한다.

---

### 실패 사례 4: 사용 중단 표장 방치로 불사용 취소 인용

**상황**: 한국 식품 기업 D사가 콜롬비아에서 특정 제품 라인을 단종했다. 해당 제품명 상표는 갱신이 계속되었으나 사용이 전혀 없었다. 4년 후 경쟁사가 불사용 취소를 청구했고, D사는 사용 증거를 제출하지 못해 취소 결정을 받았다. 이후 경쟁사가 동일 표장을 출원하여 시장에 혼란이 발생했다.

**원인**: 제품 단종 시 상표 포트폴리오 정리 절차를 거치지 않았다. 영업팀과 IP팀 간 정보 공유가 없었다.

**재발 방지**: 제품 단종·사업 철수 결정 시 IP팀에 즉시 통보하는 프로세스를 내부에 수립한다. 분기 점검 시 사용 중단 여부를 영업팀에 반드시 확인한다.
