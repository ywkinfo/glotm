# 제11장. Enforcement: 침해 대응 전략 (행정 중심)

> 이 장의 목적: 중남미 8개국에서 브랜드 침해가 확인된 이후, 단계적 집행 절차를 실행한다.
> 경고에서 시작해 행정 단속, 민사 가처분, 형사 고발, 세관 차단으로 이어지는 집행 경로를 국가별 집행 역량에 맞게 설계한다.
> 10장(모니터링)에서 침해가 확인되면 이 장으로 즉시 연결한다. 통관 차단은 12장, 온라인 플랫폼 대응은 13장, 무효·취소·소송은 14장에서 다룬다.

---

## 1. 핵심 원칙

> "침해 대응은 단계적으로 설계한다. 첫 번째 무기가 소송이면 비용이 전략을 이긴다."

중남미에서 브랜드 침해에 대응할 때 가장 흔한 실패는 두 가지다.

첫째, 침해를 발견하자마자 민사 소송부터 시작하는 경우다. 중남미 민사 소송은 평균 3~7년이 소요된다. 그 사이 침해자는 영업을 계속하고, 소송 비용은 누적된다. 행정 단속·경고·가처분을 통해 침해를 조기에 차단할 수 있었음에도 가장 무거운 수단을 먼저 꺼낸 결과다.

둘째, 경고장만 발송하고 후속 조치 없이 방치하는 경우다. 경고장은 첫 번째 단계이지, 유일한 단계가 아니다. 침해자가 응하지 않으면 다음 단계로 즉시 전환해야 한다. 후속 조치 없는 경고는 침해자에게 "계속해도 된다"는 신호를 보내는 것과 같다.

중남미 집행 전략의 설계 원칙은 세 가지다.

- 단계성: 경고 → 행정 → 민사·형사 순으로 단계를 높인다. 단, 침해 규모가 크거나 증거 인멸 위험이 있으면 초기부터 행정 단속과 가처분을 병행한다.
- 국가별 역량 반영: 집행 수단의 실효성은 국가마다 다르다. 브라질·멕시코·칠레는 행정·민사·형사 병행이 가능하다. 아르헨티나·콜롬비아·페루는 행정 중심으로 설계한다. 에콰도르·파나마는 세관 차단과 플랫폼 신고가 현실적으로 더 효과적이다.
- 증거 선행: 모든 집행 단계에서 증거 확보가 절차보다 먼저다. 증거 없이 신고하면 처리가 중단되거나 각하된다.

## 2. 국가 범위 및 적용 조건

이 장은 자사 상표가 등록되어 있거나 출원 중인 국가에서 침해가 확인된 경우에 적용한다.
상표 등록이 없는 국가에서는 집행 수단이 제한되므로, 긴급 출원 또는 선사용 입증 전략을 병행해야 한다.

| 국가 | 행정 집행기관 | 민사 집행 | 형사 집행 | 세관 차단 | 집행 역량 |
|------|--------------|-----------|-----------|-----------|-----------|
| 브라질 | INPI (행정), PROCON (소비자보호), 연방검찰 | 연방법원 가처분 가능 | 검사 고발 가능 | RECEITA FEDERAL | 높음 |
| 멕시코 | IMPI (행정 단속), COFEPRIS (보건 관련) | 민사법원 가처분 | 검사 고발 가능 | SAT (세관) | 높음 |
| 칠레 | INAPI (행정), 소비자보호청 (SERNAC) | 민사법원 | 검찰 고발 | ADUANA | 높음 |
| 콜롬비아 | SIC (행정 단속·과징금) | 민사법원 | 검찰 고발 | DIAN | 중간 |
| 아르헨티나 | INPI-AR (행정), 경찰 | 민사법원 가처분 | 형사법원 | AFIP | 중간 |
| 페루 | INDECOPI (행정 단속·과징금) | 민사법원 | 검찰 고발 | SUNAT | 중간 |
| 에콰도르 | SENADI (행정) | 민사법원 | 제한적 | SENAE | 낮음 |
| 파나마 | DIGERPI (행정) | 민사법원 | 제한적 | ANA (관세청) | 낮음 |

> 운영 원칙: 집행 역량이 낮은 국가(에콰도르, 파나마)는 행정 단속보다 세관 차단(12장)과 플랫폼 신고(13장)를 우선한다.

## 3. 주요 리스크

### 3-1. 집행 비용 과다 리스크

| 리스크 유형 | 설명 | 영향 국가 |
|-------------|------|-----------|
| 소송 직진 | 경고·행정 단속 없이 민사 소송부터 시작해 초기 비용이 과도하게 발생 | 전 국가 |
| 다국가 동시 소송 | 예산 계획 없이 여러 국가에서 동시에 법적 절차를 개시해 비용 초과 | 3개국 이상 동시 침해 상황 |
| 성과 없는 경고 반복 | 경고장을 여러 차례 발송했지만 후속 단계 없이 반복해 침해자에게 대응 의지가 없다는 신호를 줌 | 전 국가 |
| 집행 기관 역량 과신 | 현지 집행 기관의 실제 처리 역량을 과신해 수개월을 기다린 후 진전 없음을 확인 | 에콰도르, 파나마, 일부 중미 국가 |

### 3-2. 증거 부족 리스크

| 리스크 유형 | 설명 | 영향 국가 |
|-------------|------|-----------|
| 침해 증거 미확보 | 신고 전 공증된 증거를 확보하지 않아 행정 기관이 절차를 각하 | 브라질, 멕시코 |
| 자사 등록 증거 미제출 | 자사 상표 등록증·번역본·공증본을 제때 준비하지 못해 신고가 지연 | 전 국가 |
| 구매 증거 누락 | 위조품 구매 시 영수증·사진·구매 내역을 보관하지 않아 침해 행위 입증 불가 | 전 국가 |
| 온라인 증거 소멸 | 침해 페이지 URL을 캡처하지 않은 사이 침해자가 페이지를 삭제 | 전 국가 |

### 3-3. 침해자 대응 리스크

| 리스크 유형 | 설명 | 영향 국가 |
|-------------|------|-----------|
| 도주·재등록 | 경고장 수령 후 침해자가 사업장을 이전하거나 유사 브랜드로 재등록 | 전 국가 |
| 반격 무효 청구 | 침해자가 자사 상표의 불사용 취소 청구로 반격 | 브라질, 멕시코, 콜롬비아 |
| 침해자가 선등록자인 경우 | 침해자가 오히려 해당 국가에서 유사 표장의 선등록자임을 주장 | 등록 전 진출한 경우 |
| 복수 침해자 동시 출현 | 단속 후 유사 침해자가 다수 등장해 개별 대응이 불가능한 상황 | 브라질(온라인), 멕시코 |

## 4. Decision Box: 침해 대응 경로 판단

```
[Decision Box A: 집행 수단 선택 — 침해 유형별 분기]

IF (공보에서 유사 표장 공고 발견)
    → 이의신청 절차 즉시 개시 (06장 참조)
    → 이의 기간이 남아 있으면 집행보다 이의신청이 우선
    → 이미 이의 기간이 경과했으면 → 무효·취소 절차 (14장 참조)

IF (온라인 플랫폼에서 위조품·무단 표장 발견)
    → 플랫폼 신고 즉시 개시 (13장 참조)
    → 동시에 침해 증거(URL·스크린샷·구매내역) 확보
    → 판매자 정보 파악 후 → 경고장 발송 또는 행정 신고 병행

IF (오프라인 시장에서 위조품 발견)
    → 샘플 구매 + 공증된 증거 확보
    → 현지 대리인과 함께 행정 단속 신청 검토
    → 침해 규모가 크면 → 세관 차단 병행 (12장 참조)

IF (침해자가 자사 브랜드를 유통에 무단 사용)
    → 유통 계약 위반 여부 확인 (09장 참조)
    → 계약 위반이면 → 계약 해지 + 손해배상 청구
    → 계약 관계 없는 제3자이면 → 경고장 발송 후 행정·민사 단계

IF (도메인·소셜 계정 선점)
    → 13장 온라인 플랫폼 대응 절차로 이동
    → UDRP 또는 현지 ADR 절차 검토

IF (세관에서 위조품 통관 시도 발견)
    → 12장 통관 차단 절차로 즉시 이동
    → 상표 세관 등록이 없으면 긴급 등록 신청 병행
```

```
[Decision Box B: 국가별 집행 경로 선택]

IF (브라질에서 침해 발생)
    → 1단계: 경고장 발송 (현지 변호사 명의, 포르투갈어)
    → 2단계: INPI 또는 연방법원에 행정·가처분 신청
    → 3단계: 침해 규모·반복성에 따라 형사 고발 병행
    → 온라인 침해이면 → MercadoLibre / Shopee Brasil 브랜드 프로텍션 동시 활용

IF (멕시코에서 침해 발생)
    → 1단계: 경고장 발송 (현지 변호사 명의)
    → 2단계: IMPI에 Infracción Administrativa 신청
    → 3단계: FGR 형사 고발 또는 민사 가처분 병행
    → 국경 위조품이면 → SAT 세관 차단 병행 (12장)

IF (칠레에서 침해 발생)
    → 1단계: 경고장 발송
    → 2단계: INAPI 행정 신고 또는 민사 가처분
    → 3단계: 형사 고발 (지식재산권 침해 형사처벌 가능)
    → 집행 역량이 높으므로 3단계 전략 전부 활용 가능

IF (콜롬비아에서 침해 발생)
    → 1단계: 경고장 발송
    → 2단계: SIC 행정 단속 신청 (조사·과징금 부과 가능)
    → 3단계: 필요 시 민사 가처분 병행
    → 형사 고발은 제한적으로 활용 (처리 지연 가능성 높음)

IF (아르헨티나에서 침해 발생)
    → 1단계: 경고장 발송 (공증인 발송 Carta Documento 권장)
    → 2단계: INPI-AR 또는 사법경찰 고발
    → 3단계: 민사법원 가처분 신청
    → 경제 불안정으로 장기 소송 비용 리스크 고려

IF (페루에서 침해 발생)
    → 1단계: 경고장 발송
    → 2단계: INDECOPI 행정 단속 신청 (과징금·사용금지 명령 가능)
    → 3단계: 행정 결정 후 민사 집행 또는 형사 고발 병행
    → INDECOPI 처리 기간: 통상 3~9개월 예상

IF (에콰도르 또는 파나마에서 침해 발생)
    → 1단계: 플랫폼 신고 + 세관 차단 우선 (집행 역량 낮음)
    → 2단계: 경고장 발송
    → 3단계: SENADI / DIGERPI 행정 신고 (처리 지연 감안)
    → 소송은 비용 대비 실효성을 신중히 검토
```

```
[Decision Box C: 경고장 이후 후속 조치 판단]

IF (경고장 발송 후 침해자가 즉시 중단을 약속)
    → 서면 합의서 수취 (중단 확인, 재발 방지, 재고 처리 조항 포함)
    → 이행 여부를 30일 후 재확인
    → 합의 후 재발 시 → 행정 단속 + 형사 고발 즉시 전환

IF (경고장 발송 후 침해자가 무시 또는 무응답)
    → 2주 이내 행정 신고 또는 민사 가처분 신청 개시
    → 경고장 발송 사실을 증거로 첨부 (발송 내역, 수신 확인서)

IF (경고장 발송 후 침해자가 협상을 요청)
    → 합의 범위(합의금, 라이선스 전환, 브랜드 변경)를 사전 결정 후 대응
    → 협상 중에도 증거 수집과 신고 준비를 병행 (협상 결렬 대비)
    → 합의 협상은 현지 변호사가 Responsible, IP 총괄이 Accountable

IF (침해자가 자사 상표 무효 청구로 반격)
    → 등록 상표 사용증거 즉시 준비 (08장 증거 관리 시스템 참조)
    → 무효 청구 대응과 침해 대응을 별개 트랙으로 동시 진행
    → 14장(분쟁 전략) 절차 병행 개시
```

## 5. 실행 절차

### Step 1. 침해 유형 분류 및 증거 확보 (발견 즉시)

침해가 발견되면 즉시 유형을 분류하고 증거를 확보한다. 증거 확보 전 경고장을 발송하면 침해자가 증거를 인멸할 수 있다.

침해 유형별 증거 확보 방법:

| 침해 유형 | 확보할 증거 | 확보 방법 | 기한 |
|-----------|-------------|-----------|------|
| 오프라인 위조품 판매 | 위조품 샘플, 영수증, 판매처 사진, 판매자 정보 | 현장 구매 또는 조사기관 활용 | 발견 즉시 |
| 온라인 위조품·무단 사용 | 페이지 URL, 화면 캡처(날짜 포함), 판매자 ID, 구매 증거 | 웹 아카이브 저장 + 화면 캡처 | 발견 즉시 |
| 유사 표장 사용 | 침해 표장 사진·URL, 사용 장소·범위, 등록 여부 확인 | 상표청 DB 조회 + 현장 조사 | 발견 후 7일 이내 |
| 무단 라이선스 사용 | 계약서 원본, 사용 범위 초과 증거, 수익 증거 | 계약 기록 + 현장 자료 | 발견 즉시 |

자사 권리 증거 준비:

- 해당 국가 상표 등록증 (공증본 또는 인증본)
- 등록증 스페인어 또는 포르투갈어 번역본 (공증 필요 여부 국가별 확인)
- 상표 사용 증거 (08장 증거 관리 시스템 참조)
- 자사와 침해자 표장의 유사도 비교 자료

담당: 국내 IP팀(R), 현지 대리인(R), IP 총괄(A)

---

### Step 2. 경고장 발송 (증거 확보 후 14일 이내)

경고장은 법적 절차 개시 전 침해자에게 공식적으로 중단을 요구하는 첫 번째 단계다.
경고장 발송 자체가 증거가 되므로, 내용과 발송 방법을 정확히 관리한다.

경고장 필수 포함 사항:

1. 자사 상표 등록 정보 (등록번호, 지정 류, 등록일)
2. 침해 행위의 구체적 내용 (침해 표장, 사용 장소, 사용 기간)
3. 침해 중단 요청 내용 (사용 중단, 재고 폐기, 침해 관련 자료 제출)
4. 응답 기한 (통상 14~30일)
5. 응답 없을 경우의 후속 조치 예고 (행정 신고, 민사 소송 등)

국가별 경고장 발송 방법:

| 국가 | 권장 발송 방법 | 언어 | 유의사항 |
|------|--------------|------|----------|
| 브라질 | 내용증명우편(AR) 또는 공증인 발송 | 포르투갈어 | 현지 변호사 명의 사용 권장 |
| 멕시코 | 등기우편 또는 공증인 발송 | 스페인어 | IMPI 신고 병행 가능 |
| 콜롬비아 | 등기우편 또는 이메일(수신 확인 포함) | 스페인어 | SIC 신고 전 발송 권장 |
| 아르헨티나 | 공증인 발송(Carta Documento) 권장 | 스페인어 | 법적 효력을 위해 공증인 발송 사용 |
| 칠레 | 등기우편 또는 공증인 발송 | 스페인어 | 전자 발송도 유효 |
| 페루 | 등기우편 + 이메일 병행 | 스페인어 | INDECOPI 신고 전 발송 권장 |
| 에콰도르 | 등기우편 | 스페인어 | 현지 대리인을 통한 발송 권장 |
| 파나마 | 등기우편 또는 이메일(수신 확인) | 스페인어 | DIGERPI 신고 병행 가능 |

> 중남미 공통 주의사항: 경고장은 반드시 현지 변호사 또는 대리인이 검토한 후 발송한다. 잘못된 내용이나 과도한 주장은 역으로 법적 문제가 될 수 있다.

담당: 국내 IP팀(R), 현지 대리인(R), IP 총괄(A)

---

### Step 3. 행정 단속 신청 (경고 무시 또는 침해 지속 시)

경고장 발송 후 응답이 없거나 침해가 지속되면 해당 국가 행정 기관에 신고한다.
행정 단속은 민사 소송보다 빠르고 비용이 적으며, 대부분의 중남미 국가에서 가장 현실적인 1차 집행 수단이다.

국가별 행정 단속 신청 절차:

| 국가 | 신청 기관 | 신청 가능 조치 | 평균 처리 기간 | 신청 시 필요 서류 |
|------|-----------|--------------|---------------|-----------------|
| 브라질 | INPI (행정 절차) 또는 연방경찰 | 침해 중단 명령, 과징금, 재고 압수 | 6~18개월 | 등록증, 침해 증거, 포르투갈어 번역본 |
| 멕시코 | IMPI (Infracción Administrativa) | 침해 중단 명령, 과징금, 영업정지 | 6~12개월 | 등록증, 침해 증거, 대리인 위임장 |
| 칠레 | INAPI 또는 민사법원 | 침해 금지 명령, 손해배상 | 3~9개월 | 등록증, 침해 증거 |
| 콜롬비아 | SIC | 침해 금지 명령, 과징금, 재고 압수 | 6~12개월 | 등록증, 침해 증거 |
| 아르헨티나 | INPI-AR 또는 사법경찰 | 침해 중단 명령, 재고 압수 | 6~18개월 | 등록증, 침해 증거, 공증본 |
| 페루 | INDECOPI | 침해 금지 명령, 과징금, 재고 압수 | 3~9개월 | 등록증, 침해 증거 |
| 에콰도르 | SENADI | 침해 금지 명령 | 6~24개월 | 등록증, 침해 증거 |
| 파나마 | DIGERPI | 침해 금지 명령 | 6~18개월 | 등록증, 침해 증거 |

행정 단속 신청 시 공통 준비 사항:

1. 자사 상표 등록증 (공증본 또는 인증본, 현지어 번역 포함)
2. 침해 행위 증거 (Step 1에서 확보한 자료 일체)
3. 경고장 발송 증거 (발송 내역, 수신 확인서)
4. 현지 대리인 위임장 (각국 형식에 맞게 준비)
5. 신청 수수료 (국가별 상이, 현지 대리인에게 사전 확인)

담당: 현지 대리인(R), 국내 IP팀(R), IP 총괄(A)

### Step 4. 민사 가처분 신청 (긴급 차단이 필요한 경우)

침해 규모가 크거나 빠른 차단이 필요한 경우 민사 가처분을 행정 단속과 병행 또는 우선 신청한다.
가처분은 본안 소송의 결과와 관계없이 즉각적인 침해 중단 효과를 노릴 수 있다.

가처분 적합 상황:

- 행사·이벤트·성수기에 위조품이 대량 유통되는 경우
- 침해자가 대규모 생산·유통망을 보유한 경우
- 공보에서 유사 표장이 등록 직전 상태인 경우
- 침해자가 재고 이전·증거 인멸 징후를 보이는 경우

국가별 가처분 특징:

| 국가 | 가처분 신청 기관 | 처리 기간 | 보증금 요건 | 유의사항 |
|------|----------------|-----------|-------------|----------|
| 브라질 | 연방법원 | 수일~수주 | 요구될 수 있음 | 신속처리 신청 가능 |
| 멕시코 | 민사법원 | 1~4주 | 요구될 수 있음 | IMPI 단속과 병행 가능 |
| 칠레 | 민사법원 | 1~2주 | 요구될 수 있음 | 집행 역량 높음 |
| 콜롬비아 | 민사법원 | 2~6주 | 요구될 수 있음 | SIC 행정 절차와 병행 |
| 아르헨티나 | 민사법원 | 2~6주 | 요구될 수 있음 | 경제 리스크 고려 |
| 페루 | 민사법원 | 4~8주 | 요구될 수 있음 | INDECOPI 절차 병행 권장 |

> 중남미 특칙: 브라질과 멕시코는 가처분 신청 시 보증금을 요구하는 경우가 있다. 신청 전 현지 변호사와 보증금 규모를 사전 협의한다.

담당: 현지 변호사(R), 국내 IP팀(R), IP 총괄(A)

---

### Step 5. 형사 고발 (반복·고의·대규모 침해의 경우)

형사 고발은 침해 억지력이 가장 강하지만, 처리 기간이 길고 결과 예측이 어렵다.
반복적·고의적 침해 또는 위조품 대규모 유통에 한해 행정·민사 절차와 병행한다.

형사 고발 적합 상황:

- 경고·행정 단속 이후에도 침해가 반복되는 경우
- 조직적 위조품 생산·유통망이 확인된 경우
- 소비자에게 위해를 줄 수 있는 위조품(의약품, 화장품, 식품류)인 경우
- 침해 규모가 사업적 손해 수준에 달하는 경우

국가별 형사 고발 가능 여부:

| 국가 | 형사 처벌 가능 여부 | 담당 기관 | 유의사항 |
|------|------------------|-----------|----------|
| 브라질 | 가능 | 연방검찰청 | 위조품 관련 형법 적용 가능 |
| 멕시코 | 가능 | 검사청(FGR) | 상표 침해 형사 처벌 규정 존재 |
| 칠레 | 가능 | 검찰청 | 형사 집행 역량 높음 |
| 콜롬비아 | 제한적 | 검찰청 | 처리 지연 가능성 높음, 병행 활용 |
| 아르헨티나 | 가능 | 연방검찰청 | 경제 상황에 따라 집행력 변동 |
| 페루 | 가능 | 검찰청 | INDECOPI 행정 결과 활용 권장 |
| 에콰도르 | 제한적 | 검찰청 | 실효성 낮음, 행정 중심으로 운영 |
| 파나마 | 제한적 | 검찰청 | 실효성 낮음 |

담당: 현지 형사변호사(R), 국내 IP팀(C), IP 총괄(A)

### Step 6. 합의 및 사후 관리

침해 대응 절차가 진행되는 동안 합의 제안이 들어오거나 절차 결과로 합의가 가능한 경우, 합의 범위와 조건을 사전에 설정한다.

합의 가능 범위:

| 합의 유형 | 내용 | 주의사항 |
|-----------|------|----------|
| 즉시 중단 합의 | 침해 중단 확인서 + 재고 폐기 확인 | 재발 방지 조항 필수 포함 |
| 유상 라이선스 전환 | 침해자에게 정식 라이선스 부여 (09장 참조) | 로열티 구조·기간·해지 조항 명확히 설계 |
| 합의금 수취 | 침해에 대한 손해배상으로 합의금 수취 | 합의 후 재침해 시 소송 가능 조항 포함 |
| 브랜드 변경 합의 | 침해자가 유사 표장을 폐기하고 다른 브랜드로 변경 | 변경 시한 및 위반 시 제재 조항 포함 |

합의 후 이행 관리:

- 합의 내용 이행 여부를 30일, 90일, 180일 단위로 점검한다.
- 디지털 자산(플랫폼 계정, 소셜 채널, 도메인) 이전 여부를 확인한다.
- 이행 완료 후 행정·사법 절차를 공식 취하한다.

담당: 국내 IP팀(R), 현지 대리인(R), IP 총괄(A)

## 7. 체크리스트

### 7-1. 공통 체크리스트 (전 국가 적용)

침해 발견 즉시:
- [ ] 침해 유형을 분류했다 (공보·온라인·오프라인·계약 위반)
- [ ] 침해 증거를 확보했다 (샘플, 스크린샷, URL, 구매 영수증)
- [ ] 자사 상표 등록증 사본(공증본)을 준비했다
- [ ] 현지 대리인에게 침해 사실을 통보했다

경고장 발송 전:
- [ ] 경고장 내용을 현지 변호사가 검토했다
- [ ] 침해 행위의 구체적 내용을 경고장에 명시했다
- [ ] 응답 기한과 후속 조치 예고를 포함했다
- [ ] 발송 방법(등기, 공증인)을 국가별 기준에 맞게 선택했다
- [ ] 발송 내역과 수신 확인서를 보관했다

행정 신고 전:
- [ ] 경고장 발송 증거를 첨부 자료로 준비했다
- [ ] 자사 등록증의 현지어 번역본과 공증본을 준비했다
- [ ] 현지 대리인 위임장을 작성했다
- [ ] 신청 수수료를 확인하고 준비했다

합의 협상 전:
- [ ] 수용 가능한 합의 범위(중단, 라이선스, 합의금)를 사전에 결정했다
- [ ] 합의 조건을 IP 총괄이 승인했다
- [ ] 합의서에 재발 방지 조항을 포함했다

### 7-2. 국가별 추가 체크리스트

브라질 한정:
- [ ] 경고장을 포르투갈어로 작성하고 현지 변호사가 서명했다
- [ ] INPI 행정 신청 또는 연방법원 가처분 여부를 결정했다
- [ ] 형사 고발이 필요한 경우 연방검찰청 관할 지역을 확인했다
- [ ] MercadoLibre / Shopee Brasil 브랜드 프로텍션 등록 여부를 확인했다 (13장 참조)

멕시코 한정:
- [ ] IMPI Infracción Administrativa 신청 서류(등록증·침해 증거·위임장)를 준비했다
- [ ] 형사 고발은 FGR에 해야 함을 확인했다
- [ ] SAT 세관 등록 여부를 확인했다 (12장 참조)

콜롬비아 한정:
- [ ] SIC 신고 전 경고장 발송 기록을 확보했다
- [ ] SIC 행정 절차와 민사 가처분 병행 여부를 결정했다

아르헨티나 한정:
- [ ] 경고장 발송 시 공증인 발송(Carta Documento) 방식을 사용했다
- [ ] 집행 기간 중 경제 환경 변화에 따른 비용 변동 리스크를 검토했다

페루 한정:
- [ ] INDECOPI 신고 서류를 현지 대리인이 준비했다
- [ ] INDECOPI 결정까지 통상 3~9개월이 소요됨을 내부 일정에 반영했다

## 9. 실패 사례 및 주의사항

### 실패 사례 1: 경고장만 반복하고 후속 조치 없음

상황: 멕시코 시장에서 위조품 유통을 발견하고 경고장을 3회 발송했다. 침해자는 무시했고, 매번 경고장만 발송하다 1년이 경과했다.

결과: 침해자는 1년간 위조품을 계속 유통했다. 대응 없이 경고만 반복한 사실이 오히려 "묵인"으로 해석될 소지가 생겼다. IMPI 행정 단속 신청 시 "이미 1년이 지난 침해"로 판단 기준이 달라졌다.

교훈: 경고장 발송 후 2주 이내에 응답이 없으면 행정 신고 또는 가처분 절차를 즉시 개시한다. 경고는 단계의 시작이지 유일한 도구가 아니다.

---

### 실패 사례 2: 증거 없이 행정 신고

상황: 브라질에서 온라인 위조품을 발견하고 즉시 INPI에 신고했다. 그러나 침해 페이지 URL을 캡처하지 않았고, 구매 증거도 없었다.

결과: INPI가 증거 불충분으로 절차를 보류했다. 증거를 다시 수집하는 동안 침해자는 해당 판매 페이지를 삭제했다. 결국 절차를 재개하지 못했다.

교훈: 신고 전 반드시 증거를 확보한다. 온라인 침해는 페이지 삭제 위험이 있으므로 발견 즉시 URL·화면 캡처·구매 증거를 수집하고 저장한다.

---

### 실패 사례 3: 합의 조건 사전 결정 없이 협상 진행

상황: 콜롬비아에서 침해자가 합의 의사를 밝혔다. 담당자가 합의 범위를 사전에 결정하지 않은 채 협상을 시작했고, 내부 결재를 반복하는 사이 협상이 지연됐다.

결과: 침해자는 협상 지연을 이용해 재고를 처분하고 사업을 다른 법인명으로 전환했다. 협상이 최종 결렬됐을 때는 이미 집행 가능한 재고가 없었다.

교훈: 합의 협상 개시 전 수용 가능한 조건(중단·라이선스·합의금 범위)을 사전에 IP 총괄이 승인한다. 협상 진행 중에도 행정·민사 절차를 병행해 침해자의 대응 여지를 줄인다.

---

### 실패 사례 4: 상표 미등록 상태에서 집행 시도

상황: 아르헨티나에 진출했지만 상표 등록 전이었다. 유사 브랜드 사용자를 발견하고 INPI-AR에 행정 신고를 시도했다.

결과: INPI-AR은 자사 등록 상표가 없다는 이유로 신고를 수리하지 않았다. 긴급 출원을 시작했지만 심사 기간 동안 침해자의 영업이 계속됐다. 결국 침해자가 유사 표장을 먼저 등록하는 상황이 발생했다.

교훈: 집행의 전제는 등록이다. 시장 진입 전 해당 국가 상표 등록을 완료한다. 긴급 상황이면 출원 즉시 출원 번호 기반의 임시 보호 가능 여부를 현지 대리인에게 확인한다.
