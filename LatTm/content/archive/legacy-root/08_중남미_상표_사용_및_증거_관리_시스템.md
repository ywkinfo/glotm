# 제08장. 상표 사용 및 증거 관리 시스템

> **이 장의 목적**: 등록된 상표권을 유지·방어하기 위해 "사용 사실"을 증거로 구축하고 관리하는 시스템을 운영한다.
> 중남미에서 상표권은 등록으로 완성되지 않는다. 사용 증거가 없으면 권리는 언제든 취소될 수 있다.
> 이 장은 07장(등록 후 유지관리) 완료 후, 10장(모니터링 시스템)과 병행 운영한다.

---

## 1. 핵심 원칙

> **"증거 없는 권리는 취약하다. 사용하는 것만큼 중요한 것이 사용을 증명하는 것이다."**

중남미 대부분의 국가는 **선출원주의(First-to-file)**를 따른다. 그러나 선출원이 완료된 이후에도 권리는 사용에 의해 유지된다. 불사용 취소 제도가 전 국가에 적용되며, 사용 입증 책임은 권리자에게 있다.

세 가지 원칙을 항상 지킨다.

첫째, 상표를 실제 사용하는 순간부터 증거를 수집한다. 사용 후 소급 수집은 한계가 있다.
둘째, 증거는 국가별로 분리해 저장한다. 한 나라에서 수집한 증거가 다른 나라의 분쟁에서 유효하지 않을 수 있다.
셋째, 증거 관리는 IP팀 단독 업무가 아니다. 영업·마케팅·물류팀이 증거 생산자로 참여해야 한다.

---

## 2. 국가 범위 및 적용 조건

이 장은 아래 8개 대상국에 모두 적용된다. 단, 국가별로 불사용 취소 기준 연수, 사용선언 의무 여부, 증거 요건이 다르므로 국가별 유의사항(7절)을 반드시 병행 확인한다.

| 국가 | 상표청 | 불사용 취소 기준 | 사용선언 의무 | 증거 언어 요건 |
|------|--------|-----------------|--------------|----------------|
| 브라질 | INPI | 등록 후 5년 경과 시 청구 가능 | 없음 | 포르투갈어 번역 필요 |
| 멕시코 | IMPI | 3년 불사용 시 취소 가능 | **등록 후 3년 이내 필수** | 스페인어 |
| 콜롬비아 | SIC | 3년 불사용 시 취소 가능 | 없음 | 스페인어 |
| 아르헨티나 | INPI | 5년 불사용 시 취소 가능 | **5~6년차 중간 + 갱신 시** | 스페인어 |
| 칠레 | INAPI | 불사용 취소 사유 도입 (법 개정) | 없음 | 스페인어 |
| 페루 | INDECOPI | 3년 불사용 시 취소 가능 | 없음 | 스페인어 |
| 에콰도르 | SENADI | 3년 불사용 시 취소 가능 | 없음 | 스페인어 |
| 파나마 | DIGERPI | 5년 불사용 시 취소 가능 | 없음 | 스페인어 |

> **운영 원칙**: 불사용 취소 청구에 대비하기 위한 증거 수집은 취소 기준 연수보다 최소 1년 이전부터 시작한다. 취소 기준 직전에 사용을 시작하는 것은 법적으로 유효하지 않을 수 있다.

## 3. 주요 리스크

### 3-1. 불사용 취소 청구 리스크

불사용 취소는 경쟁사, 제3자, 전 파트너 누구나 청구할 수 있다. 청구인은 별도 증거를 제출할 의무가 없으며, 권리자가 사용 사실을 입증해야 한다.

| 리스크 유형 | 설명 | 해당 국가 | 대응 방향 |
|-------------|------|-----------|-----------|
| **사용 증거 미비** | 실제 판매는 했으나 증거를 체계적으로 보관하지 않은 경우 | 전 국가 | 판매 개시 즉시 증거 수집 루틴 구축 |
| **증거 단절** | 특정 기간(예: 1~2년) 판매 공백으로 연속 사용 입증 불가 | 안데스 권역(3년 기준) | 비수기에도 최소한의 사용 활동 유지 |
| **현지 사용 아닌 수출 주장** | 한국에서 수출했다고 해서 해당 국가의 "사용"으로 인정되지 않는 경우 | 전 국가 | 현지 판매·광고·유통 증거를 별도 확보 |
| **유통사 명의 사용** | 유통사가 상표를 사용하는 구조에서 권리자의 직접 사용이 없다는 주장 | 전 국가 | 라이선스 계약서 + 사용 통제 증거 확보 |
| **사용 범위 불일치** | 지정 상품류와 실제 사용 상품이 일치하지 않는 경우 | 전 국가 | 지정 류별 사용 증거를 분리 관리 |

### 3-2. 멕시코 사용선언 특수 리스크

멕시코는 사용 증거 제출이 "불사용 취소 방어"가 아니라 **"등록 유지 의무"**다. 사용선언을 기한 내 제출하지 않으면 권리가 자동 소멸된다.

| 리스크 시나리오 | 결과 | 내부 마감 |
|-----------------|------|-----------|
| 사용선언 기한 누락 | 등록 권리 소멸 — 재출원 필수 | 등록일 +2년 9개월에 내부 알림 설정 |
| 사용선언 제출했으나 증거 불충분 | IMPI 보완 요청 또는 선언 인정 거부 | 제출 30일 전에 증거 패키지 완성 |
| 사용선언 일부 류만 가능 | 나머지 류는 소멸 → 인접 류 보호 공백 | 출원 시 사용 가능 범위만 지정 (02장 원칙 재확인) |

### 3-3. 증거 보관 리스크

| 리스크 유형 | 설명 |
|-------------|------|
| **분산 보관** | 각 팀이 각자 보관해 분쟁 시 통합 제출 불가 |
| **형식 미비** | 날짜, 국가, 상품명, 상표 표기가 불명확한 증거는 효력 의심 |
| **언어 미처리** | 한국어 문서를 번역 없이 제출할 경우 현지 절차에서 인정 거부 |
| **기간 공백** | 특정 연도의 증거가 없어 연속 사용 입증 불가 |
| **디지털 증거 미캡처** | 온라인 판매·광고는 캡처하지 않으면 소멸 |

## 4. Decision Box: 증거 관리 방식 판단 기준

```
[Step 1] 현재 사용 중인 상표와 해당 국가를 확인한다.

[Step 2] 아래 분기로 증거 관리 방식을 결정한다.

IF (멕시코에 등록 상표 보유)
    → 사용선언 캘린더를 즉시 생성한다 (등록일 기준 +3년 마감, 내부 마감 +2년 9개월)
      사용선언 패키지(증거 묶음)를 기한 3개월 전에 완성한다
      단일 류 미제출도 권리 소멸 위험 — 류별로 증거를 분리 관리한다

IF (안데스 권역 3개국 — 콜롬비아·페루·에콰도르 — 에 등록 상표 보유)
    → 3년 불사용 취소 기준 적용 — 최소 매 2년마다 사용 증거 갱신 수집
      연간 1회 국가별 증거 파일 점검을 정기 업무로 설정한다
      3년 연속 사용을 입증할 수 있는 증거 구조(연도별 파일)를 유지한다

IF (브라질에 등록 상표 보유)
    → 5년 불사용 취소 기준 적용 — 4년차에 증거 완비 여부를 점검한다
      포르투갈어 번역 증거를 별도 준비한다
      유통사를 통해 판매하는 경우 라이선스 구조를 증거로 남긴다

IF (아르헨티나에 등록 상표 보유)
    → 5~6년차 중간 사용선언 기한을 캘린더에 입력한다 (등록 즉시)
      갱신 시 최근 5년 사용선언 제출 요건에 맞춰 연도별 증거 파일을 유지한다
      중간 사용선언 제출 3개월 전에 증거 패키지를 완성한다

IF (유통사·파트너가 현지에서 상표를 사용하는 구조)
    → 라이선스 계약서를 증거의 기반으로 삼는다
      유통사가 사용 보고서와 판매 증거를 정기 제출하도록 계약서에 명시한다 (09장 참조)
      권리자의 "사용 통제" 사실이 입증되어야 라이선시 사용이 권리자의 사용으로 인정된다

IF (현재 해당 국가에서 판매 공백 기간이 발생 중)
    → 공백 기간이 취소 기준 연수(3년 또는 5년)에 근접하면 즉시 대응 방안을 수립한다
      최소 규모의 판매 또는 광고 활동을 통해 사용 사실을 생성한다
      공백 이유와 복귀 계획을 문서화해 향후 분쟁 대비 자료로 보관한다

IF (복수 국가에서 증거 관리 예산이 제한적인 경우)
    → 우선순위: 사용선언 의무 국가(멕시코·아르헨티나) > 취소 기준 3년 국가(안데스 권역) > 취소 기준 5년 국가(브라질·파나마)
      최소 증거 세트(아래 5절 참조)를 국가별로 연 1회 이상 확보한다
```

## 5. 실행 절차

### Step 1. 증거 수집 대상 확인 (등록 완료 직후 또는 판매 개시 전)

등록이 완료되거나 판매가 시작되면 즉시 아래를 확인하고 수집을 시작한다.

1. 포트폴리오 관리 대장에서 해당 국가의 등록 정보를 확인한다: 등록 번호, 지정 류, 등록일, 불사용 취소 기준 연수.
2. 사용선언 의무 국가(멕시코·아르헨티나)인 경우 캘린더 기한을 즉시 확인·등록한다.
3. 해당 국가에서 상표가 실제로 사용되는 방식을 파악한다: 직접 판매, 유통사 판매, 온라인 판매, 광고 전용 등.
4. 증거 저장소에 국가별 폴더를 생성한다: `/증거저장소/[국가명]/[표장명]/[연도]/`

**담당**: 국내 IP팀(R), IP 총괄(A), 영업팀(C)

---

### Step 2. 증거 유형 결정 및 수집 루틴 구축

상표 사용 방식에 따라 수집해야 할 증거 유형이 다르다. 아래 표에서 해당 유형을 확인하고 수집 담당자를 지정한다.

| 증거 유형 | 내용 | 수집 담당 | 수집 주기 | 보관 형식 |
|-----------|------|-----------|-----------|-----------|
| **판매 송장·영수증** | 상표 표기된 상품의 현지 판매 기록 | 영업팀 | 거래 발생 즉시 | PDF + 원본 사본 |
| **포장·라벨 실물** | 상표가 표기된 상품 포장재 실물 또는 사진 | 영업팀/물류팀 | 포장 변경 시마다 | 사진 파일 + 실물 보관 |
| **광고·마케팅 자료** | 신문·잡지·SNS·옥외광고·카탈로그 등 상표 사용 광고 | 마케팅팀 | 캠페인 발행 즉시 | 원본 파일 + 스크린샷 |
| **온라인 판매 캡처** | 자사 쇼핑몰·플랫폼(MercadoLibre 등)의 상품 페이지 | 마케팅팀/IP팀 | 월 1회 이상 | 날짜·URL 포함 스크린샷 |
| **전시·박람회 참가** | 현지 전시회 참가 증명 (부스 사진, 참가 확인서) | 영업팀 | 참가 직후 | 사진 + 공식 확인서 |
| **수입·통관 서류** | 해당 국가에 상품이 반입된 통관 서류 | 물류팀 | 선적 시마다 | PDF + 원본 사본 |
| **유통사 판매 보고서** | 현지 유통사가 제출하는 정기 판매 보고서 | 영업팀 | 분기 1회 이상 | PDF (유통사 공식 문서) |
| **언론 보도·인터뷰** | 현지 미디어에 상표 또는 제품이 노출된 기사 | 마케팅팀 | 발행 즉시 | URL + 스크린샷 + PDF |

> **최소 증거 세트**: 불사용 취소 방어를 위한 최소한의 증거는 아래 3가지다. 모든 국가·표장에 대해 최소 이 3가지는 연도별로 확보한다.
> 1. 판매 송장 (날짜·수량·가격·상표명 포함)
> 2. 상표가 표기된 포장 사진
> 3. 해당 국가 내 광고 또는 온라인 판매 캡처

**담당**: 국내 IP팀(R), 영업팀(R), 마케팅팀(R), 물류팀(C), IP 총괄(A)

---

### Step 3. 증거 저장소 운영

모든 증거는 아래 기준에 따라 중앙 저장소에 보관한다.

**폴더 구조 (표준)**:
```
/증거저장소
  /브라질
    /[표장명_A]
      /2024
        - 송장_20240315.pdf
        - 포장사진_20240315.jpg
        - 온라인캡처_20240315_MercadoLivre.png
      /2025
        - ...
  /멕시코
    /[표장명_A]
      /사용선언패키지_2026
        - 사용선언서_초안.docx
        - 송장묶음.pdf
        - 광고자료.pdf
      /2024
        - ...
```

**저장 기준**:
- 파일명에 날짜(YYYYMMDD)를 포함한다.
- 날짜가 불명확한 증거는 기재 불가 문서로 분리 관리한다.
- 한국어 문서는 번역 버전을 동일 폴더에 함께 보관한다.
- 원본 디지털 파일은 삭제하지 않는다. 수정본은 별도 파일로 저장한다.

**담당**: 국내 IP팀(R), IP 총괄(A)

---

### Step 4. 사용선언 패키지 준비 (멕시코·아르헨티나 전용)

멕시코와 아르헨티나는 사용선언을 공식 제출해야 한다. 아래 절차를 기한 3개월 전에 시작한다.

**멕시코 사용선언 (Declaración de Uso)**:

| 단계 | 내용 | 기한 |
|------|------|------|
| 증거 패키지 완성 | 류별 사용 증거 묶음 완성 | 기한 D-90 |
| 현지 대리인 검토 | 제출 가능 여부 및 보완 항목 확인 | D-60 |
| 사용선언서 초안 작성 | IMPI 양식 기준, 스페인어 | D-45 |
| 위임장 공증 | 필요 서류 공증 처리 | D-30 |
| 최종 제출 | IMPI e-MARCANET 전자 제출 | **내부 마감: 법정 기한 D-30** |

**아르헨티나 중간 사용선언 (Acta de Renovación)**:

| 단계 | 내용 | 기한 |
|------|------|------|
| 5~6년차 기한 확인 | 등록일 기준 5년 도래 6개월 전 점검 | 등록 후 4년 6개월 |
| 증거 패키지 완성 | 최근 5년 사용 증거 묶음 완성 | 기한 D-90 |
| 현지 대리인 의뢰 | INPI-AR 제출 절차 위임 | D-60 |
| 위임장 공증·아포스티유 | 처리 기간 10영업일 이상 확보 | D-45 |

**담당**: 국내 IP팀(R), 현지 대리인(R: 서류 작성), IP 총괄(A), 재무팀(I)

### Step 5. 연간 증거 파일 점검

연 1회 이상 전 국가 증거 파일을 점검한다. 이 점검은 포트폴리오 분기 점검(15장)과 연계한다.

**점검 항목**:
1. 각 국가별 최신 연도(직전 12개월) 증거가 존재하는가?
2. 최소 증거 세트(송장·포장·광고)가 각 국가별로 갖춰져 있는가?
3. 취소 기준 연수가 도래하기 전에 증거가 충분한가?
4. 사용선언 의무 기한이 12개월 이내로 도래하는 국가는 없는가?
5. 유통사 구조인 경우 유통사로부터 판매 보고서를 수취했는가?

**점검 결과에 따른 후속 조치**:

| 점검 결과 | 후속 조치 |
|-----------|-----------|
| 증거 공백 발견 | 해당 팀에 즉시 보완 요청 + 캘린더 재설정 |
| 사용선언 기한 임박 | Step 4 절차 즉시 개시 |
| 유통사 보고서 미수취 | 계약 조항 기준으로 유통사에 공식 요청 |
| 해당 국가 판매 중단 예정 | 방어 유지 가능 여부 검토 (취소 기준 연수 확인) |

**담당**: 국내 IP팀(R), IP 총괄(A), 영업팀(C), 마케팅팀(C)

---

### Step 6. 불사용 취소 청구 수신 시 긴급 대응

불사용 취소 청구서를 수신하면 아래 절차를 즉시 실행한다.

| 시점 | 조치 |
|------|------|
| 청구 수신 즉시 | 국내 IP팀 + IP 총괄 공동 수신 확인, 현지 대리인 즉시 통보 |
| 수신 후 48시간 이내 | 현지 대리인에게 답변 기한 확인 요청 |
| 수신 후 7영업일 이내 | 증거 저장소에서 해당 국가 + 표장 + 지정 류의 사용 증거 전수 확인 |
| 수신 후 14영업일 이내 | 사용 증거 패키지 완성 및 현지 대리인 전달 |
| 법정 답변 기한 내부 마감 | 기한 −21일 (대리인 검토 완료), 기한 −14일 (최종 제출) |

> **중남미 특칙**: 불사용 취소 절차에서 답변을 제출하지 않으면 청구가 자동 인용될 수 있다. 수신 즉시 대응 체계를 가동한다.

**담당**: 국내 IP팀(R), IP 총괄(A), 현지 대리인(R: 답변서 작성), 법무팀(C)

- [ ] 내부 마감(등록일 +2년 9개월)에 알림을 설정했다

- [ ] 연도별 사용 증거 폴더가 5개 연도 이상 연속으로 구성되어 있다
- [ ] 중간 사용선언 패키지 준비를 기한 D-90에 시작하도록 캘린더에 입력했다

- [ ] CAN 상표 보유 기업은 3개국 각각의 증거를 별도 폴더에 보관했다
- [ ] 판매 공백 기간이 3년에 근접하는 경우 즉시 사용 활동을 재개했다

### 브라질 해당 기업 추가 확인

- [ ] 포르투갈어 번역 증거를 동일 폴더에 함께 보관했다
- [ ] 등록 후 4년차에 증거 완비 여부를 점검하는 일정을 설정했다
- [ ] 유통사 구조인 경우 라이선스 계약서를 증거 폴더에 포함했다

- [ ] 칠레는 법 개정 이후 불사용 취소 가능성을 인지하고 증거 수집을 시작했다
- [ ] 파나마는 실판매가 없는 방어 출원인 경우 최소한의 상업적 사용 활동 계획을 수립했다

## 10. 실패 사례 및 주의사항

### 실패 사례 1: 멕시코 사용선언 — 증거 부족으로 일부 류 권리 소멸

**상황**: 한국 소비재 기업 A사가 멕시코에서 5개 류로 등록 후 3년이 지나 사용선언을 제출했다. 영업팀과 협의 없이 IP팀이 단독으로 준비했고, 제출 직전에야 3개 류의 판매 증거가 없다는 사실을 발견했다. 시간 부족으로 보완 불가 — 3개 류의 사용선언을 제출하지 못했다.

**결과**: 사용선언 미제출 3개 류의 등록 권리 자동 소멸. 경쟁사가 해당 류에 유사 표장을 출원. 핵심 사업 영역 중 일부가 노출됨.

**교훈**: 사용선언 준비는 기한 D-90에 시작한다. 류별 증거 보유 여부를 D-90 시점에 확인하고, 미비 류는 즉시 영업팀과 협의해 보완한다. 사용 불가 류는 자발적 해제를 검토한다.

---

### 실패 사례 2: 콜롬비아 불사용 취소 — 유통사 사용이 권리자 사용으로 불인정

**상황**: 한국 화장품 기업 B사가 콜롬비아 유통사를 통해 2년간 제품을 판매했다. 전 경쟁사가 불사용 취소를 청구하면서 B사의 "직접 사용" 부재를 주장했다. B사는 유통사 판매 증거를 제출했으나, 라이선스 계약서가 없어 "유통사 사용이 권리자 사용과 별개"라는 주장에 반박하지 못했다.

**결과**: SIC에서 취소 청구 일부 인용. 일부 류의 등록 소멸.

**교훈**: 유통사 구조에서는 반드시 라이선스 계약서를 작성하고 SIC에 등록을 권장한다. 유통사가 계약 아래 사용한다는 사실을 입증할 수 있어야 권리자 사용으로 인정된다.

---

### 실패 사례 3: 브라질 불사용 취소 — 포르투갈어 번역 미비로 증거 인정 거부

**상황**: 한국 전자기업 C사가 브라질에서 불사용 취소 청구를 받고, 7년간의 판매 송장과 마케팅 자료를 제출했다. 자료가 한국어와 영어로만 작성되어 있어 INPI 절차에서 포르투갈어 번역 공증 자료 제출을 요구받았다. 번역 준비에 8주가 소요되면서 답변 기한 내 충분한 자료를 제출하지 못했다.

**결과**: 증거 부족으로 취소 청구가 일부 인용됨.

**교훈**: 브라질 증거는 포르투갈어 번역 공증본을 평소에 준비한다. 분쟁 발생 후 번역을 시작하면 기한을 충족하기 어렵다. 핵심 증거 문서는 연간 번역본을 갱신한다.

---

### 실패 사례 4: 아르헨티나 중간 사용선언 — 기한 미인식으로 권리 위험

**상황**: 한국 제조기업 D사가 아르헨티나 등록 6년 후 갱신을 시도했다. 그 시점에야 5~6년차 중간 사용선언을 제출하지 않았다는 사실을 파악했다. 중간 사용선언 미제출로 INPI-AR이 등록에 이의를 제기했고, 권리 유지를 위한 별도 절차가 필요해졌다.

**결과**: 갱신 지연 + 추가 절차 비용 발생 + 일부 류 권리 위험 노출.

**교훈**: 아르헨티나 사용선언 의무는 등록 즉시 캘린더에 입력한다. "갱신만 하면 된다"고 인식하면 중간 사용선언을 놓친다. 포트폴리오 관리 대장에 5~6년차 중간 사용선언을 별도 항목으로 기재한다.

---

### 실패 사례 5: 증거 저장소 미운영 — 분쟁 발생 시 7년치 증거 소멸

**상황**: 한국 식품 기업 E사가 페루에서 7년간 판매했으나, 증거를 체계적으로 보관하지 않았다. 담당자가 교체되면서 이전 담당자의 개인 PC에 있던 증거 자료 다수가 소실됐다. 취소 청구를 받고 증거를 수집하려 했으나, 3년 이내의 연속 사용을 입증할 문서가 부족했다.

**결과**: 불사용 취소 청구 일부 인용. 신규 출원으로 권리 재건 필요.

**교훈**: 증거는 개인 PC나 로컬 드라이브에 보관하지 않는다. 반드시 중앙 저장소(공유 드라이브)에 저장하고, 담당자 교체 시 인수인계 항목에 증거 저장소 접근 권한을 포함한다.

## 11. 운영 시스템 연계

이 장의 결과는 아래 시스템에 즉시 반영한다.

| 운영 도구 | 반영 내용 | 시점 |
|-----------|-----------|------|
| **증거 저장소** | 국가별·표장별·연도별 폴더 생성 및 수집 증거 즉시 저장 | 판매 개시 즉시 |
| **중앙 캘린더** | 사용선언 기한, 내부 마감, 연간 점검 일정 | 등록 완료 즉시 |
| **포트폴리오 관리 대장** | 사용선언 제출 완료 여부, 불사용 취소 청구 현황 | 제출·수신 즉시 |
| **현지 대리인 DB** | 사용선언·취소 대응 수임 기록, 증거 제출 이력 | 절차 진행 시마다 |
