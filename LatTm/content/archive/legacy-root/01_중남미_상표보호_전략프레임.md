# 제01장. 중남미 상표 보호 전략 프레임 (전체 구조)

> 이 장의 목적: 중남미 8개국 브랜드 보호 운영의 전체 구조를 설계한다.
> 각 장의 상세 내용을 읽기 전에, 이 장을 먼저 읽고 전체 흐름을 파악한다.

---

## 1. 핵심 원칙

> "출원이 아니라 운영이다."

상표 등록은 시작이지, 목표가 아니다. 중남미에서 브랜드를 보호한다는 것은
출원 → 등록 → 사용증거 관리 → 모니터링 → 집행이 끊기지 않고 돌아가는
운영 시스템을 구축하는 것이다.

중남미는 단일 법제가 없다. 국가마다 출원 경로, 사용 요건, 이의 기간, 집행 수준이 다르다.
따라서 "어느 나라에서 어떤 규칙으로 싸울지"를 먼저 결정해야 나머지 절차가 설계된다.

---

## 2. 전략 설계의 출발점: 국가 클러스터

중남미 8개 대상국은 법제·언어·유통 구조에 따라 4개 클러스터로 묶인다.
클러스터 단위로 전략을 설계하면 자원 낭비 없이 보호 범위를 확장할 수 있다.

| 클러스터 | 대상 국가 | 통합 출원 시스템 | 언어 | 핵심 특징 |
|----------|-----------|-----------------|------|-----------|
| 대형 단독 시장 | 브라질, 멕시코 | 없음 (개별 출원 필수) | 포르투갈어 / 스페인어 | 시장 규모 최대, 리스크도 최고 |
| 안데스 규범권 | 콜롬비아, 페루, 에콰도르 | CAN 상표 (Decision 486) 활용 가능 | 스페인어 | 3년 불사용 취소 공통 적용 |
| 남미 남부권 | 아르헨티나, 칠레 | 없음 (개별 출원) | 스페인어 | 중간 사용선언 요건 (아르헨티나) |
| 허브형 | 파나마 | 없음 | 스페인어 | 운하·자유무역지대 경유 유통 리스크 |

> 운영 원칙: 8개국 동시 출원은 원칙이 아니다. 시장 진입 계획과 리스크 수준에 따라 우선순위를 결정하고 단계적으로 확장한다.

## 3. 전체 프로세스 맵

모든 실무 절차는 아래 5단계 흐름 안에 위치한다. 각 단계는 이 가이드의 해당 장과 연결된다.

```
[STAGE 1] 설계 (Pre-filing)
    브랜드 구조 결정 → 국가 우선순위 결정 → 표장 언어 리스크 점검
    → 담당 장: 02장(포트폴리오 설계), 03장(검색 및 충돌 분석)

[STAGE 2] 권리화 (Filing)
    출원 경로 선택 → 출원서 작성 및 제출 → 이의·거절 대응
    → 담당 장: 04장(출원 경로), 05장(출원 실무), 06장(이의·거절 대응)

[STAGE 3] 운영 (Post-registration)
    등록 후 유지관리 → 사용증거 관리 → 계약·라이선스 통제
    → 담당 장: 07장(유지관리), 08장(증거 관리), 09장(계약 구조)

[STAGE 4] 감시 (Monitoring)
    공보 모니터링 → 온라인 감시 → 시장·유통 감시
    → 담당 장: 10장(모니터링 시스템)

[STAGE 5] 집행 (Enforcement)
    경고 → 행정 단속 → 통관 차단 → 플랫폼 차단 → 소송
    → 담당 장: 11장(침해 대응), 12장(통관), 13장(온라인), 14장(분쟁)
```

> 중남미 특칙: 집행(STAGE 5)은 국가별 역량 차이가 크다.
> 브라질·멕시코·칠레는 행정·민사·형사 병행이 가능하지만,
> 일부 국가는 세관 차단과 플랫폼 신고가 현실적으로 더 효과적이다.

## 4. 국가 선택 Decision Box

이 Decision Box는 가이드 전체에서 가장 먼저 실행해야 하는 판단이다.
국가를 결정하지 않으면, 이후 어떤 절차도 설계할 수 없다.

```
[Step 1] 현재 판매 중이거나 12개월 내 진입 예정인 국가를 목록화한다.

[Step 2] 아래 분기로 출원 우선순위를 결정한다.

IF (브라질만 해당)
    → INPI 단독 출원 진행 (포르투갈어 표장 검토 필수)

IF (멕시코만 해당)
    → IMPI 단독 출원 진행 + 3년 사용선언 캘린더 즉시 생성

IF (콜롬비아·페루·에콰도르 중 2개국 이상 동시 필요)
    → CAN 안데스 공동체 상표 출원 우선 검토
      단, 볼리비아 포함 여부와 현지 대리인 비용 비교 후 결정

IF (아르헨티나·칠레 동시 필요)
    → 개별 국가 출원 2건 진행 (통합 시스템 없음)
      아르헨티나는 중간 사용선언 일정 즉시 캘린더화

IF (파나마가 포함된 경우)
    → 허브 기능 여부 확인: 파나마에서 실판매 vs 유통 경유만인지 구분
      유통 경유만이라도 방어용 출원 권장 (위조품 재유통 리스크)

IF (8개국 중 3개국 이상 + 예산 제약)
    → 시장 규모 × 리스크 매트릭스(아래 표)로 우선순위 결정
      최소 보호선: 브라질·멕시코는 반드시 포함

[Step 3] 우선순위 결정 후 → 02장(포트폴리오 설계)으로 이동
```

### 국가 우선순위 결정 매트릭스

| 국가 | 시장 규모 | 브랜드 리스크 | 운영 복잡도 | 최소 보호 포함 여부 |
|------|-----------|--------------|------------|------------------------|
| 브라질 | ●●●●● | ●●●●● | ●●●●○ | 반드시 포함 |
| 멕시코 | ●●●●○ | ●●●●● | ●●●○○ | 반드시 포함 |
| 콜롬비아 | ●●●○○ | ●●●●○ | ●●●○○ | 시장 진입 시 포함 |
| 아르헨티나 | ●●●○○ | ●●●●○ | ●●●●○ | 시장 진입 시 포함 |
| 칠레 | ●●○○○ | ●●●○○ | ●●○○○ | 시장 진입 시 포함 |
| 페루 | ●●○○○ | ●●●○○ | ●●○○○ | 안데스 묶음으로 처리 |

## 5. RACI 구조: 전체 운영 역할 정의

중남미 브랜드 보호는 IP팀 혼자 할 수 없다. 아래 역할 구조를 전사적으로 합의하고 고정한다.

### 5-1. 역할 정의

| 역할 | 담당 | 중남미 특칙 |
|------|------|------------|
| Responsible (실행) | 국내 IP팀 또는 브랜드 담당자 | 현지 대리인에게 단독 위임 금지 |
| Accountable (최종 책임) | IP 총괄 또는 법무 책임자 | 외부 대리인은 Accountable 불가 |
| Consulted (협의) | 현지 대리인 (각국), 영업·마케팅팀 | 대리인은 반드시 Consulted에 위치 |
| Informed (통보) | 재무팀, 물류팀, 경영진 | 기한·비용 변경 시 즉시 통보 |

> 브라질 예외: INPI 제출 서류 작성은 브라질 현지 변호사(advogado)가
> Responsible을 수행할 수 있다. 단, 전략 결정권은 본사가 유지한다.

### 5-2. 단계별 RACI 표

| 단계 | 업무 | 국내 IP팀 | IP 총괄 | 현지 대리인 | 영업팀 | 재무팀 |
|------|------|-----------|---------|------------|--------|--------|
| Pre-filing | 국가 우선순위 결정 | R | A | C | C | I |
| Pre-filing | 표장 언어 리스크 점검 | R | A | C | C | — |
| Pre-filing | 상표 검색 (각국 DB) | R | A | C | — | — |
| Filing | 출원서 작성·제출 | R | A | R (브라질) / C | — | I |
| Filing | 이의·거절 대응 | R | A | C | — | I |
| Post-reg | 사용선언 제출 (멕시코 등) | R | A | C | I | — |
| Post-reg | 증거 수집·저장 | R | A | — | R | — |
| Post-reg | 계약서 상표 조항 검토 | R | A | C | C | — |
| Monitoring | 공보·온라인 모니터링 | R | A | C | — | — |
| Enforcement | 경고장 발송 | R | A | R | — | I |
| Enforcement | 세관 차단 신청 | R | A | R | — | I |

*R=Responsible, A=Accountable, C=Consulted, I=Informed*

## 6. 기한 관리 원칙: 전체 캘린더 구조

모든 기한은 아래 2가지 원칙으로 관리한다.

- 내부 마감 = 법정 기한 −30일: 법정 기한을 직접 쫓지 않는다.
- 캘린더 등록 시점 = 해당 절차 완료 직후: 등록 후 사용선언 기한은 등록 즉시 캘린더에 넣는다.

### 국가별 핵심 기한 요약표

| 국가 | 출원→등록 (평균) | 갱신 주기 | 사용 요건 기한 | 불사용 취소 기준 |
|------|-----------------|-----------|----------------|-----------------|
| 브라질 | 24~36개월 | 10년 | 별도 사용선언 없음 | 등록 후 5년 경과 시 청구 가능 |
| 멕시코 | 4~6개월 | 10년 | 등록 후 3년 내 사용선언 제출 | 3년 불사용 시 취소 청구 가능 |
| 콜롬비아 | 12~18개월 | 10년 | 별도 사용선언 없음 | 3년 불사용 시 취소 청구 가능 |
| 아르헨티나 | 18~30개월 | 10년 | 5~6년차 중간 사용선언 + 갱신 시 사용선언 | 5년 불사용 시 취소 청구 가능 |
| 칠레 | 12~18개월 | 10년 | 별도 사용선언 없음 | 불사용 취소 사유 도입 (법 개정) |
| 페루 | 12~18개월 | 10년 | 별도 사용선언 없음 | 3년 불사용 시 취소 청구 가능 |
| 에콰도르 | 12~18개월 | 10년 | 별도 사용선언 없음 | 3년 불사용 시 취소 청구 가능 |
| 파나마 | 12~18개월 | 10년 | 별도 사용선언 없음 | 5년 불사용 시 취소 청구 가능 |

> 멕시코 주의: 사용선언 기한(등록일로부터 3년 이내)을 놓치면 권리가 자동 소멸될 수 있다.
> 등록 즉시 캘린더에 입력하고, 내부 마감을 등록일 +2년 9개월로 설정한다.

## 7. 출원 경로 선택 원칙 (요약)

상세 내용은 04장에서 다루며, 이 장에서는 기본 원칙만 정리한다.

| 경로 | 활용 조건 | 주의사항 |
|------|-----------|----------|
| 마드리드 국제출원 | 한국 기본출원/등록 기반, 다수 국가 동시 보호 필요 시 | 중남미 일부 국가는 마드리드 비가맹 (파나마 포함 확인 필수) |
| 직접 출원 (개별국) | 브라질·멕시코 최우선 → 개별 출원 권장 | 현지 대리인 필수, 포르투갈어/스페인어 명세서 |
| CAN 안데스 공동체 상표 | 콜롬비아·페루·에콰도르·볼리비아 2개국 이상 동시 보호 필요 시 | 현지 대리인 선임 필수, 운영 복잡도 고려 |

## 8. 운영 시스템 연계 구조

이 가이드는 아래 4가지 운영 도구와 함께 사용한다. 도구 없이 가이드만 읽으면 실행으로 이어지지 않는다.

| 운영 도구 | 역할 | 최초 설정 시점 | 담당 장 |
|-----------|------|--------------|---------|
| 포트폴리오 관리 대장 | 국가별 출원·등록·갱신 현황 한눈에 파악 | 첫 출원 전 | 15장, 19장 |
| 중앙 캘린더 | 국가별 기한(갱신·사용선언·이의기간) 통합 관리 | 각 등록 완료 직후 | 07장, 19장 |
| 증거 저장소 | 국가별·연도별 사용증거(송장·광고·캡처 등) 폴더 보관 | 판매 시작 즉시 | 08장, 19장 |
| 현지 대리인 DB | 국가별 대리인 연락처·계약 조건·과거 대응 이력 | 대리인 선임 즉시 | 09장, 19장 |

## 9. 체크리스트: 전략 프레임 수립 완료 기준

이 체크리스트를 모두 완료한 후 02장(포트폴리오 설계)으로 이동한다.

### 공통 (모든 기업)

- [ ] 진입 대상 국가 목록을 확정했다
- [ ] 국가 클러스터(대형/안데스/남부/허브)를 기준으로 묶었다
- [ ] 브라질·멕시코를 최소 보호선에 포함했다 (또는 제외 사유를 문서화했다)
- [ ] 전체 5단계 프로세스(설계→권리화→운영→감시→집행)를 팀 내 공유했다
- [ ] RACI 표를 작성하고 담당자를 지정했다
- [ ] 포트폴리오 관리 대장 양식을 준비했다
- [ ] 중앙 캘린더를 생성했다

### 멕시코 해당 기업 추가 확인

- [ ] 등록 예정일 기준 사용선언 마감일(+3년)을 캘린더에 입력했다
- [ ] 내부 마감(+2년 9개월)을 별도 알림으로 설정했다

### 안데스 권역(콜롬비아·페루·에콰도르) 해당 기업 추가 확인

- [ ] CAN 안데스 공동체 상표 출원 여부를 검토했다 (개별 출원 대비 비용·기간 비교)
- [ ] 3년 불사용 취소에 대비한 사용증거 수집 루틴을 정했다

### 파나마 해당 기업 추가 확인

- [ ] 파나마에서 실판매인지 유통 경유인지 구분했다
- [ ] 위조품 재유통 리스크를 감안해 방어용 출원 여부를 결정했다

## 10. 주요 실패 사례 및 주의사항

### 실패 패턴 1: "브라질·멕시코를 나중에 하겠다"

상황: 예산 절감을 이유로 중소 시장부터 출원하고, 핵심 시장을 후순위로 미룬 경우.

결과: 그 사이 제3자(딜러 포함)가 브라질 또는 멕시코에서 선등록.
취소 절차에 1~3년 이상 소요되고, 그 기간 동안 시장 진입이 막힘.

교훈: 브라질·멕시코는 예산과 관계없이 최우선 출원 대상이다.
나머지 국가는 단계적으로 추가해도 되지만, 이 두 나라는 동시에 출원한다.

---

### 실패 패턴 2: "대리인이 알아서 하겠지"

상황: 현지 대리인에게 출원과 갱신을 전적으로 위임하고, 본사는 결과만 받아보는 구조.

결과: 사용선언 기한 누락, 갱신 누락, 대리인 교체 시 이력 단절.
일부 사례에서 대리인 자신이 상표를 선등록하는 경우도 발생.

교훈: 현지 대리인은 Consulted(협의) 위치다. 기한 관리와 전략 결정은 반드시 본사가 직접 한다.
대리인 계약서에 상표 출원 금지 조항과 기밀유지 조항을 명시한다.

---

### 실패 패턴 3: "출원 단계에서 국가를 너무 많이 잡았다"

상황: 초기에 8개국 전체 출원을 동시에 진행하다 예산 소진.
등록 후 관리 비용을 감당 못해 갱신을 포기하거나 대리인을 교체하면서 운영 공백 발생.

결과: 등록은 됐지만 사용증거 관리·모니터링이 방치됨. 3~5년 후 불사용 취소 청구에 무방비.

교훈: 출원 수보다 운영 가능한 수를 기준으로 국가를 결정한다.
운영 불가능한 국가는 출원하지 않는 것이 낫다.
