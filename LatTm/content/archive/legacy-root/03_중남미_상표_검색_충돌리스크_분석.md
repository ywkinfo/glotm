# 제03장. Pre-filing: 상표 검색 및 충돌 리스크 분석

> 이 장의 목적: 출원 전에 "내 표장이 충돌 없이 등록될 수 있는가"를 판단한다.
> 검색을 건너뛰면 이의신청·거절·분쟁 비용이 출원 비용의 수십 배에 달할 수 있다.
> 이 장을 완료한 후 04장(출원 경로 선택)으로 이동한다.

---

## 1. 핵심 원칙

> "검색은 전략이다. 빠른 출원보다 정확한 검색이 먼저다."

중남미에서 상표 검색을 소홀히 하면 세 가지 문제가 발생한다.

첫째, 선행 상표와 충돌해 거절 또는 이의를 받는다. 재출원 비용과 지연 기간이 발생하고, 그 사이 제3자가 유사 표장을 추가 출원할 수 있다.

둘째, 현지 딜러나 제3자가 이미 선등록한 상태를 뒤늦게 발견한다. 이 경우 무효·취소 절차에 수년이 소요되고, 그 기간 동안 시장 진입이 제한된다.

셋째, 검색 없이 출원하면 충돌 표장 발견 후 브랜드 전략 전체를 재설계해야 한다.

중남미 상표 검색은 국가별로 데이터베이스가 분산되어 있고, 일부 국가는 검색 시스템이 불완전하다. "검색해봤더니 없었다"는 판단은 해당 국가 공식 DB 검색을 완료한 후에만 유효하다.

---

## 2. 검색 대상 확정

검색을 시작하기 전에 아래 네 가지를 확정한다.

| 검색 요소 | 결정 내용 | 확인 방법 |
|-----------|-----------|-----------|
| 검색 표장 | 02장에서 확정한 보호 표장 전체 목록 | 문자·도형·복합 유형별 구분 |
| 검색 국가 | 01장 Decision Box에서 결정한 출원 대상국 | 클러스터 단위로 묶어 처리 |
| 검색 상품·서비스류 | 02장에서 확정한 지정 류 + 인접 류(상하 1~2개 류) | 니스 분류 기준 |
| 검색 범위 | 동일 표장 + 유사 표장 + 번역·음역 표장 | 현지어 변형 포함 |

> 운영 원칙: 검색 범위를 좁게 잡아 "문제없다"는 결론을 내리는 것은 위험하다.
> 인접 류와 유사 표장까지 포함해 검색한다.

---

## 3. 주요 리스크

### 3-1. 데이터베이스 분산 및 정확도 리스크

중남미 각국 상표청의 온라인 검색 시스템은 완성도 차이가 크다. 검색 결과를 그대로 신뢰하면 안 된다.

| 리스크 유형 | 설명 | 대상 국가 |
|-------------|------|-----------|
| DB 업데이트 지연 | 출원 후 공보 게재까지 수개월 지연 → 검색 결과에 미반영 | 브라질, 아르헨티나 |
| 검색 시스템 불완전 | 일부 도형상표·복합상표 검색 기능 미지원 | 에콰도르, 파나마 |
| 구 데이터 누락 | 오래된 등록상표가 온라인 DB에 미수록 | 아르헨티나, 에콰도르 |
| 유사 표장 미탐지 | 자동 유사성 검색 기능 부재로 수동 확인 필요 | 대부분 국가 |
| 국가별 독립성 | 안데스 권역도 각국 DB 독립 운영 → 통합 검색 불가 | 콜롬비아, 페루, 에콰도르 |

실행 지침: 온라인 자체 검색을 "1차 스크리닝"으로만 활용한다. 현지 대리인에게 정밀 검색을 별도 의뢰하고 결과를 서면으로 수취한다.

---

### 3-2. 유사성 판단 리스크

| 리스크 유형 | 설명 | 실무 예시 |
|-------------|------|-----------|
| 발음 유사 | 스페인어·포르투갈어 발음 기준 혼동 가능성 판단 필요 | 영문 표기가 현지어로 다르게 발음됨 |
| 의미 유사 | 표기는 달라도 의미가 동일한 번역 표장 충돌 | 한국어 브랜드의 스페인어 번역이 기존 상표와 동일 |
| 외관 유사 | 로고 디자인·색채 조합이 기존 등록 도형상표와 유사 | 국가별로 외관 유사성 판단 기준 상이 |
| 약칭·머리글자 | 풀네임은 달라도 약칭이 선등록 표장과 동일 | 3자 이니셜 조합이 기존 표장과 충돌 |
| 현지어 번역 충돌 | 브랜드명의 스페인어 번역이 기존 상표와 혼동 가능 | "신선"을 번역한 "Fresco"가 기존 상표와 충돌 |

실행 지침: 유사성 판단은 현지 대리인의 의견을 반드시 서면으로 수취한다. "충돌 없음" 의견도 근거 포함 서면만 유효하다.

---

### 3-3. 선점 상표 발견 시 리스크

| 발견 상황 | 의미 | 다음 행동 |
|-----------|------|-----------|
| 동일 표장 + 동일 류 | 출원 불가 (거절 확실) | 표장 변경 또는 무효·취소 절차 검토 |
| 유사 표장 + 동일 류 | 이의 또는 거절 가능성 높음 | 리스크 평가 후 출원 여부 결정 |
| 동일 표장 + 인접 류 | 향후 사업 확장 시 충돌 가능 | 인접 류 선점 여부 확인 후 출원 범위 조정 |
| 딜러·파트너 명의 등록 | 즉각 대응 필요 | 06장 이의 절차 또는 09장 계약 조항 강화 |
| 미사용 상태의 선등록 | 불사용 취소 청구 가능성 | 사용 실적 확인 후 취소 절차 검토 (14장 참조) |

---

### 3-4. 도메인·소셜 미디어 선점 리스크

상표 검색과 별개로 온라인 선점 리스크를 병행 확인한다.

| 확인 항목 | 확인 방법 | 비고 |
|-----------|-----------|------|
| 도메인(.com / 국가 도메인) | WHOIS 조회 | .com.br / .com.mx / .com.co 등 국가 도메인 포함 |
| 소셜 핸들(@브랜드명) | 각 플랫폼 검색 | Instagram·TikTok·Facebook 중남미 주요 플랫폼 |
| 앱스토어 앱명 | 구글 플레이·앱스토어 검색 | 앱 출시 예정 기업 한정 |
| 주요 이커머스 판매자명 | MercadoLibre 등 검색 | 위조업자 선점 여부 확인 |

> 운영 원칙: 도메인·소셜 선점은 상표권과 별개이므로 상표 등록과 동시에 확보 조치를 취한다.
> 상표 등록이 완료돼도 도메인은 별도 분쟁 절차가 필요하다.

## 4. Decision Box: 검색 범위 및 대응 판단 기준

### 4-1. 검색 방식 선택

```
[Decision Box A: 검색 범위 결정]

IF (브라질·멕시코: 핵심 시장 단독 출원)
    → 공식 DB 자체 검색 (1차) + 현지 대리인 정밀 검색 의뢰 (2차) 필수
      브라질 INPI는 검색 결과 지연 가능 → 대리인 확인 필수

IF (안데스 권역 3개국 이상 동시 검색)
    → 각국 DB 개별 검색 (통합 DB 없음)
      CAN 상표 출원 검토 시에도 국가별 충돌 여부는 개별 확인

IF (예산 제약으로 검색 범위 축소 필요)
    → 최우선: 동일 표장 + 핵심 류 검색
      차선: 유사 표장 검색 (발음·외관·의미 유사)
      최소 검색 후 출원 결정 시 리스크를 IP 총괄이 서면 승인

IF (브랜드명 변경 이력 있거나 이전 현지 파트너가 있는 경우)
    → 구 브랜드명·파트너 명의 선등록 여부 추가 확인 필수

IF (이미 시장에서 제품 판매 중인 경우)
    → 현지 유통 중인 상태에서 제3자 선점 가능성 높음
      긴급 검색 후 즉시 출원 절차 진입
```

### 4-2. 충돌 발견 시 대응 판단

```
[Decision Box B: 충돌 발견 시 대응 선택]

IF (동일·유사 표장이 등록 상태 + 사용 중)
    → 시나리오 A: 표장 변경 후 재출원 (가장 확실한 방법)
    → 시나리오 B: 무효·취소 절차 병행 후 출원 (기간·비용 고려)
      판단 기준: 브랜드 자산 가치 > 변경 비용이면 무효 절차 진행

IF (동일·유사 표장이 등록 상태 + 미사용 3년 이상)
    → 불사용 취소 청구 검토 (국가별 취소 가능 기간 확인)
      취소 인용 후 출원 진행 (동시 진행 불가 여부 확인)

IF (동일·유사 표장이 출원 상태 + 미등록)
    → 이의신청 기간 확인 후 이의 제기 준비 (06장 참조)
      공고 즉시 이의 기산

IF (딜러·파트너 명의로 등록된 경우)
    → 계약서 조항 검토 후 계약 기반 이전 청구 우선
      계약 근거 없으면 무효 절차 진행 (14장 참조)
      분쟁 기간 중 해당 국가 사업 계획 조정 필요

IF (인접 류에만 충돌 표장 존재)
    → 핵심 류는 출원 진행
      인접 류는 충돌 리스크 감안해 지정 여부 결정
      향후 사업 확장 시 재검토 일정 캘린더에 입력
```

### 4-3. 검색 결과 신뢰도 평가

```
[Decision Box C: 검색 결과 신뢰도 확인]

IF (현지 대리인 정밀 검색 완료 + 충돌 없음 서면 수취)
    → 출원 진행 (04장으로 이동)

IF (자체 온라인 검색만 완료 + 충돌 없어 보임)
    → 반드시 현지 대리인 정밀 검색 추가 의뢰 후 출원
      자체 검색만으로 출원 결정 금지

IF (현지 대리인이 "유사 표장 있으나 문제없을 것"이라고 구두 의견 제시)
    → 서면 의견서 재요청 (구두 의견은 증거 효력 없음)
      서면 수취 후 출원 여부 IP 총괄 승인
```

## 5. 국가별 검색 데이터베이스 및 실행 절차

### 5-1. 국가별 공식 검색 DB

| 국가 | 공식 검색 시스템 | URL | 검색 가능 항목 | 신뢰도 |
|------|----------------|-----|----------------|--------|
| 브라질 | INPI Busca de Marcas | busca.inpi.gov.br | 문자·도형·복합 | 중간 (업데이트 지연) |
| 멕시코 | IMPI MARCANET | marcanet.impi.gob.mx | 문자·도형 | 높음 |
| 콜롬비아 | SIC Signos Distintivos | sic.gov.co | 문자·복합 | 중간 |
| 아르헨티나 | INPI Marcas | inpi.gob.ar | 문자 중심 | 중간 (구 데이터 누락) |
| 칠레 | INAPI Búsqueda | inapi.cl | 문자·도형 | 높음 |
| 페루 | INDECOPI Signos | indecopi.gob.pe | 문자·복합 | 중간 |
| 에콰도르 | SENADI | senadi.gob.ec | 문자 중심 | 낮음 |
| 파나마 | DIGERPI | mici.gob.pa | 문자 중심 | 낮음 |

> 운영 원칙: DB 신뢰도가 "낮음"인 국가는 반드시 현지 대리인 검색을 병행한다.
> "중간" 이상이라도 현지 대리인 검색을 권장한다.

---

### 5-2. 실행 절차

#### Step 1. 자체 온라인 검색 (D-75 이전)

출원 예정 표장 전체에 대해 각국 공식 DB에서 1차 스크리닝을 실시한다.

검색 항목:
- 동일 표장 (영문·한글·현지어 표기 모두)
- 발음 유사 표장 (스페인어·포르투갈어 발음 기준)
- 의미 유사 표장 (브랜드명의 번역어·유사어)
- 도형 유사 표장 (로고 보유 기업)

검색 기록 방식:
- 검색일, 검색 DB, 검색어, 검색 결과(스크린샷 또는 PDF) 저장
- 결과가 "없음"인 경우도 "검색 완료 + 미발견" 기록 보관
- 증거 저장소 내 "03_검색/[국가명]/[검색일]" 폴더에 저장

담당: 국내 IP팀(R), IP 총괄(A)

---

#### Step 2. 현지 대리인 정밀 검색 의뢰 (D-60 이전)

자체 검색 결과를 현지 대리인에게 공유하고 정밀 검색을 의뢰한다.

의뢰 시 전달 사항:
- 출원 예정 표장 (이미지 파일 + 텍스트 형태)
- 지정 상품·서비스류 (전체 목록)
- 인접 류 검색 포함 여부 (권장: 포함)
- 딜러·파트너 등 특정 명의 선점 여부 확인 요청

수취 서류:
- 정밀 검색 결과 보고서 (충돌 표장 목록 포함)
- 유사성 평가 의견서 (충돌 가능성 등급: 높음/중간/낮음/없음)
- 의견 근거 (해당 상표 등록일·권리자·사용 여부)

담당: 국내 IP팀(R), 현지 대리인(R), IP 총괄(A)

---

#### Step 3. 충돌 리스크 평가 및 의사결정 (D-45 이전)

검색 결과를 바탕으로 충돌 리스크를 평가하고 다음 행동을 결정한다.

| 평가 항목 | 판단 기준 | 담당 |
|-----------|-----------|------|
| 충돌 표장 수 및 유사도 | 높음·중간·낮음 3단계 | 현지 대리인 + 국내 IP팀 |
| 선등록 표장의 사용 상태 | 실사용 여부 및 사용 증거 | 현지 대리인 |
| 권리자 신원 | 딜러·파트너 또는 제3자 | 국내 IP팀 |
| 대응 비용·기간 추정 | 무효·취소 vs 표장 변경 비교 | 국내 IP팀 + IP 총괄 |
| 최종 출원 여부 결정 | 출원 진행 / 표장 변경 / 출원 보류 | IP 총괄(A) |

결정 결과는 포트폴리오 관리 대장에 기록하고 다음 행동 일정을 캘린더에 입력한다.

담당: 국내 IP팀(R), IP 총괄(A), 현지 대리인(C)

---

#### Step 4. 도메인·소셜 미디어 선점 확인 (D-45 이전)

상표 검색과 병행해 온라인 선점 여부를 확인한다.

확인 항목:
- 주요 도메인: .com / .com.br / .com.mx / .com.co / .com.ar / .cl / .com.pe 등
- 소셜 플랫폼: Instagram·TikTok·Facebook·YouTube 핸들
- 중남미 이커머스: MercadoLibre·Rappi 판매자명 검색

선점 발견 시:
- 상표권 기반 도메인 분쟁 가능성 검토 (UDRP·현지 ADR)
- 소셜 선점은 플랫폼 신고 채널 활용 (13장 참조)
- 즉각 대응 어려운 경우 대안 핸들명 확보 검토

담당: 국내 IP팀(R), 마케팅팀(C)

---

#### Step 5. 검색 결과 정리 및 포트폴리오 관리 대장 업데이트 (D-30 이전)

모든 검색 결과와 의사결정 내용을 포트폴리오 관리 대장에 기록한다.

기록 항목:
- 국가별 검색 완료 일자
- 충돌 표장 유무 및 리스크 등급
- 현지 대리인 의견서 보관 위치
- 출원 진행 여부 최종 결정 결과
- 충돌 발견 시 대응 방향 및 담당자

이후 04장(출원 경로 선택)으로 이동한다.

담당: 국내 IP팀(R), IP 총괄(A)

## 9. 실패 사례 및 주의사항

### 실패 패턴 1: "온라인 검색에서 안 나왔으니 없는 것이다"

상황: 브라질 INPI 온라인 DB에서 자체 검색 후 충돌 없음을 확인하고 출원을 진행했다. 출원 공고 후 선출원 표장을 보유한 권리자가 이의신청을 제기했다.

원인: INPI DB 업데이트 지연으로 해당 선출원이 검색 시점에 반영되지 않은 상태였다. 현지 대리인 정밀 검색을 생략한 것이 결정적 원인이었다.

결과: 이의신청 대응에 8개월 소요. 최종 협상을 통해 해당 류에서 조건부 등록을 허가받았으나 추가 비용 발생. 사업 일정이 1년 이상 지연됐다.

교훈: 온라인 자체 검색은 1차 스크리닝일 뿐이다. 브라질·아르헨티나 등 DB 신뢰도가 낮은 국가는 반드시 현지 대리인 정밀 검색을 별도 의뢰한다.

---

### 실패 패턴 2: "발음이 조금 다르니까 괜찮을 것이다"

상황: 멕시코 출원 시 자체 판단으로 "발음은 유사하지만 표기가 달라 문제없다"고 결론을 내렸다. 현지 대리인 유사성 의견서를 생략했다.

원인: 심사 과정에서 심사관이 발음 유사성을 근거로 거절 결정을 내렸다. 스페인어 발음 기준 혼동 가능성이 한국어 기준과 달랐다.

결과: 거절 대응 절차에 6개월 추가 소요. 최종 보정 후 등록됐으나, 초기에 현지 대리인 의견서를 받았다면 표장을 소폭 수정해 더 빠르게 처리할 수 있었다.

교훈: 유사성 판단은 현지 언어·발음 기준으로 해야 한다. 현지 대리인의 서면 유사성 평가 의견서는 생략하지 않는다.

---

### 실패 패턴 3: "딜러 명의 선등록을 계약서로 막을 수 있다"

상황: 콜롬비아 딜러와 계약 체결 후 사업을 시작했다. 계약서에 상표 선등록 금지 조항이 있었으나, 딜러가 계약 이전에 유사 표장을 이미 등록한 상태였다는 사실을 2년 후에 발견했다.

원인: 딜러 계약 전 딜러 명의 선등록 여부 검색을 실시하지 않았다. 검색 시 "회사명" 기준만 확인하고 딜러 대표자 개인 명의는 확인하지 않았다.

결과: 무효 절차 진행 중 딜러와 관계가 악화됐다. 해당 국가 사업이 약 18개월간 불안정한 상태에서 운영됐다.

교훈: 딜러·파트너 선임 전 해당 인물 및 법인 명의의 선등록 여부를 반드시 확인한다. 법인명 외에 대표자 개인 명의도 포함해 검색한다.

---

### 실패 패턴 4: "핵심 류만 검색하면 충분하다"

상황: 아르헨티나 출원 시 핵심 상품류(25류 의류)만 검색하고 인접 류(24류 직물)는 생략했다. 등록 후 인접 류에 유사 표장을 보유한 경쟁사로부터 확장 사업 관련 경고장을 수취했다.

원인: 인접 류 검색 생략으로 향후 사업 확장 시의 충돌 리스크를 사전에 파악하지 못했다.

결과: 경쟁사와 협상에 4개월 소요. 인접 류 추가 출원을 진행했으나 협상 기간 동안 제품 라인 확장이 지연됐다.

교훈: 핵심 류 외에 인접 류(상하 1~2개 류)도 검색에 포함한다. 향후 3년 이내 진입 예정인 상품·서비스류도 사전에 확인한다.
