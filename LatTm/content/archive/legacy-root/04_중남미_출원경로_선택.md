# 제04장. Filing 전략: 출원 경로 선택 (마드리드 vs 직접출원)

> 이 장의 목적: 중남미 8개국에 대한 출원 경로를 결정한다.
> 경로 선택이 잘못되면 비용·기간·보호 범위 모두에서 손실이 발생한다.
> 03장(상표 검색) 완료 후 이 장을 진행하고, 완료 후 05장(출원 실무)으로 이동한다.

---

## 1. 핵심 원칙

> "중남미는 국가별로 분리해서 설계한다. 단일 경로는 없다."

마드리드 국제출원 하나로 중남미 전체를 커버하려는 시도는 실패로 이어질 가능성이 높다.
브라질·멕시코는 마드리드 경로보다 직접 출원이 실무적으로 유리하다.
안데스 권역은 CAN 공동체 상표가 비용 효율 면에서 검토 가치가 있다.
파나마는 마드리드 미가맹국이므로 직접 출원이 유일한 경로다.

출원 경로는 "가장 편한 경로"가 아니라 "국가별 최적 경로"를 조합해서 설계한다.

---

## 2. 출원 경로 개요

중남미 8개국에 적용 가능한 출원 경로는 세 가지다.

| 경로 | 설명 | 활용 가능 국가 |
|------|------|----------------|
| 마드리드 국제출원 (WIPO) | 한국 기본출원/등록 기반, WIPO를 통해 복수 국가에 동시 출원 | 브라질, 멕시코, 콜롬비아, 아르헨티나, 칠레, 페루, 에콰도르 (파나마 제외) |
| 직접 출원 (국가별 개별 출원) | 각국 상표청에 직접 출원서 제출 | 8개국 전체 |
| CAN 안데스 공동체 상표 | 안데스 공동체 사무국을 통해 4개 회원국에 동시 출원 | 콜롬비아, 페루, 에콰도르, 볼리비아 |

> 파나마 주의: 파나마는 마드리드 협정 미가맹국이다. 파나마 출원은 반드시 직접 출원으로 진행한다.

---

## 3. 주요 리스크

### 3-1. 마드리드 의존 리스크

| 리스크 유형 | 설명 | 영향 국가 |
|-------------|------|-----------|
| 중앙 공격(Central Attack) | 기본출원/등록이 5년 이내에 취소·포기되면 마드리드 등록 전체가 소멸 위험 | 마드리드 경로 전체 |
| 심사 기간 불이익 | 일부 국가에서 마드리드 경로 심사가 직접 출원보다 오래 걸리는 경우 발생 | 브라질, 멕시코 |
| 거절 통보 기간 제한 | 각국이 거절 이유를 지정 기간 내 통보하지 않으면 자동 등록되나, 이후 대응이 복잡해질 수 있음 | 전체 |
| 명세서 범위 제한 | 마드리드 출원 명세서가 기본출원 범위를 초과할 수 없어 현지 특화 범위 설계 어려움 | 전체 |
| 현지 대리인 연계 약화 | 마드리드 경로에서 현지 대리인 개입이 줄어 이의·거절 대응이 늦어질 수 있음 | 전체 |

### 3-2. 직접 출원 리스크

| 리스크 유형 | 설명 | 영향 국가 |
|-------------|------|-----------|
| 다국가 관리 복잡도 | 국가별 기한·비용·대리인을 개별 관리해야 해 누락 가능성 증가 | 3개국 이상 동시 출원 시 |
| 초기 비용 집중 | 국가별 출원료·대리인 비용이 개별 발생해 초기 자금 부담이 마드리드 대비 클 수 있음 | 전체 |
| 언어 장벽 | 국가별 스페인어·포르투갈어 서류 작성에 현지 대리인 의존도 높음 | 브라질(포르투갈어), 전 스페인어권 |

### 3-3. CAN 안데스 공동체 상표 리스크

| 리스크 유형 | 설명 |
|-------------|------|
| 볼리비아 포함 강제 | CAN 상표는 4개 회원국 전체에 효력 발생 (볼리비아 제외 불가) |
| 단일 심사 결과 구속 | 한 회원국의 거절 이유가 전체 출원에 영향을 줄 수 있음 |
| 운영 복잡도 | CAN 사무국 경유 절차로 직접 출원보다 커뮤니케이션 단계가 많음 |

## 4. Decision Box: 출원 경로 선택

이 Decision Box는 국가 확정 후 가장 먼저 실행한다.
국가별로 독립적으로 판단하고, 결과를 조합해 최종 출원 경로 믹스를 설계한다.

```
[Step 1] 출원 대상 국가 목록을 확인한다 (01장 Decision Box 결과 기준).

[Step 2] 파나마 포함 여부를 먼저 분리한다.

  IF (파나마 포함)
      → 파나마 = 직접 출원 확정 (마드리드 미가맹)
        나머지 국가는 아래 Step 3으로 진행

[Step 3] 브라질·멕시코 경로를 결정한다.

  IF (브라질 포함)
      → 직접 출원 (INPI) 우선 권장
        사유: 브라질 INPI의 마드리드 심사 기간이 직접 출원 대비 실익이 불분명하고,
              포르투갈어 명세서 현지 최적화가 등록 가능성을 높임

  IF (멕시코 포함)
      → 직접 출원 (IMPI) 우선 권장
        사유: 멕시코 IMPI의 직접 출원 처리 기간(4~6개월)이 마드리드 대비 빠름
              딜러 선점 리스크 대응을 위해 조기 출원이 우선

  IF (브라질·멕시코 모두 포함 + 한국 기본출원이 아직 출원 전)
      → 한국 출원 먼저 완료 → 이후 브라질·멕시코 직접 출원 병행
        마드리드 경로는 이 두 나라에서 시간상 이점이 없음

[Step 4] 안데스 권역(콜롬비아·페루·에콰도르) 경로를 결정한다.

  IF (콜롬비아·페루·에콰도르 중 2개국 이상 동시 필요)
      → CAN 안데스 공동체 상표 출원 우선 검토
        조건: 볼리비아 포함 4개국 효력이 사업에 문제없을 것
              현지 대리인 수임 가능 여부 확인

  IF (CAN 상표가 부적합한 경우)
      → 조건: 볼리비아 제외 필요 / 국가별 명세서 범위를 달리해야 하는 경우
      → 국가별 직접 출원으로 전환

  IF (안데스 3개국 중 1개국만 필요)
      → 해당 국가 직접 출원 (CAN 비용 효율 없음)

[Step 5] 아르헨티나·칠레 경로를 결정한다.

  IF (아르헨티나·칠레 포함)
      → 개별 직접 출원 (통합 시스템 없음)
        아르헨티나: 중간 사용선언 일정 즉시 캘린더화
        칠레: 불사용 취소 제도 도입 이후 사용증거 관리 필수

[Step 6] 마드리드 경로 활용 여부를 최종 결정한다.

  IF (한국 기본출원/등록 완료 + 비(非)브라질·멕시코 국가 3개 이상 동시 보호 필요)
      → 마드리드 출원으로 관리 단일화 검토
        단, 각국 거절 대응을 위한 현지 대리인 사전 선임 필수

  IF (브라질·멕시코가 주요 출원 대상이고 나머지는 부가적)
      → 마드리드 경로 불필요, 브라질·멕시코 직접 출원 + 필요 국가 추가 직접 출원

  IF (예산 제약으로 관리 단일화가 최우선)
      → 마드리드 출원 검토 가능
        단, 파나마는 별도 직접 출원, 브라질·멕시코는 직접 출원 권장 유지
```

| 파나마 | 직접 출원 (DIGERPI) | 없음 | 마드리드 미가맹, 직접 출원 유일 |

## 6. 비용·기간 비교표

### 6-1. 마드리드 vs 직접 출원 비교 (브라질·멕시코 기준)

| 항목 | 마드리드 경로 | 직접 출원 |
|------|--------------|-----------|
| 출원 기반 | 한국 기본출원/등록 필요 | 불필요 (독립 출원 가능) |
| 초기 비용 | WIPO 기본료 + 지정국 추가료 | 국가별 출원료 + 대리인 비용 |
| 심사 기간 (브라질) | 직접 출원과 유사 또는 더 긴 경우 발생 | 24~36개월 (INPI 처리 기준) |
| 심사 기간 (멕시코) | 6~12개월 | 4~6개월 |
| 명세서 유연성 | 기본출원 범위 내로 제한 | 국가별 최적화 가능 |
| 거절 대응 | WIPO 통해 각국에 전달, 대응 창구 분산 | 현지 대리인 직접 대응, 빠른 처리 |
| 중앙 공격 리스크 | 있음 (5년 이내 기본등록 소멸 시 전체 위험) | 없음 (독립 등록) |
| 관리 복잡도 | 단일 창구 (WIPO) | 국가별 개별 관리 필요 |

### 6-2. CAN 안데스 공동체 상표 vs 직접 출원 비교 (안데스 권역 기준)

| 항목 | CAN 안데스 공동체 상표 | 직접 출원 (3개국) |
|------|----------------------|------------------|
| 효력 국가 | 콜롬비아·페루·에콰도르·볼리비아 (4개국 고정) | 지정 국가만 |
| 초기 비용 | 단일 출원료 (3~4개국 동시 효율) | 국가별 출원료 × 3 |
| 심사 기간 | 12~18개월 | 12~18개월 (국가별 유사) |
| 명세서 유연성 | 단일 명세서로 4개국 적용 | 국가별 개별 최적화 가능 |
| 볼리비아 포함 | 강제 (제외 불가) | 볼리비아 선택 가능 |
| 현지 대리인 | 1인 또는 CAN 사무국 경유 가능 | 국가별 개별 대리인 필요 |
| 관리 복잡도 | 단일 등록으로 이후 관리 효율 | 국가별 개별 관리 필요 |

| 국가 | 상표청 | 심사 기간 (평균) | 갱신 주기 | 특이사항 |
|------|--------|-----------------|-----------|----------|

| 콜롬비아 | SIC | 12~18개월 | 10년 | CAN 상표 병행 가능 |
| 아르헨티나 | INPI-AR | 18~30개월 | 10년 | 중간 사용선언 의무 |
| 칠레 | INAPI | 12~18개월 | 10년 | 전자출원 가능 |

## 7. 실행 절차

### Step 1. 출원 경로 믹스 설계 (출원 D-60 이전)

1. 01장 국가 우선순위 목록과 02장 포트폴리오 설계 결과를 기반으로 출원 대상 국가를 확정한다.
2. 파나마 포함 여부를 먼저 분리한다 (직접 출원 고정).
3. 브라질·멕시코에 대해 직접 출원 진행 여부를 결정한다.
4. 안데스 권역(콜롬비아·페루·에콰도르) 2개국 이상 해당 시 CAN 상표 출원 비용·기간을 비교한다.
5. 잔여 국가에 대해 마드리드 또는 직접 출원을 결정한다.
6. 최종 출원 경로 믹스를 문서화하고 IP 총괄의 승인을 받는다.

담당: 국내 IP팀(R), 현지 대리인(C), IP 총괄(A)

---

### Step 2. 현지 대리인 선임 또는 확인 (출원 D-45 이전)

출원 경로별로 필요한 현지 대리인을 확인하고 선임한다.

| 출원 경로 | 현지 대리인 필요 여부 | 선임 기준 |
|-----------|---------------------|-----------|
| 직접 출원 (모든 국가) | 필수 | 해당 국가 산업재산권 전문 자격 보유자 |
| 마드리드 경로 | 거절 통보 시 필수 (사전 선임 권장) | 거절 대응 가능 현지 전문가 |
| CAN 안데스 공동체 상표 | 필수 | 안데스 공동체 상표 경험 보유자 |

> 대리인 선임 원칙: 계약서에 상표 출원 금지 조항과 기밀유지 조항을 반드시 포함한다. 대리인에게 전략 결정권을 위임하지 않는다 (09장 참조).

담당: 국내 IP팀(R), IP 총괄(A)

---

### Step 3. 마드리드 경로 선택 시 — 한국 기본출원 상태 확인 (출원 D-40 이전)

마드리드 경로를 선택한 국가가 있을 경우, 아래 사항을 확인한다.

1. 한국 기본출원 또는 기본등록의 상태를 특허청에서 확인한다.
2. 기본출원이 출원 중인 경우: 마드리드 출원 가능하나, 5년 중앙 공격 기간 관리를 캘린더화한다.
3. 기본등록이 5년 이상 경과한 경우: 중앙 공격 리스크 소멸, 마드리드 경로 안정성 확보.
4. 마드리드 출원 명세서가 기본출원 명세서 범위를 초과하지 않는지 검토한다.
5. WIPO 마드리드 시스템에 등록된 한국 기본출원/등록 정보를 확인한다.

담당: 국내 IP팀(R), IP 총괄(A)

---

### Step 4. CAN 안데스 공동체 상표 경로 선택 시 — 추가 확인 (출원 D-40 이전)

CAN 안데스 공동체 상표를 선택한 경우, 아래 사항을 추가로 확인한다.

1. 볼리비아 포함 4개국 효력이 사업에 문제없는지 검토한다 (볼리비아 제외 불가).
2. CAN 경험이 있는 현지 대리인을 선임한다.
3. 단일 명세서로 4개국을 커버하는 것이 적합한지 국가별 사업 계획을 대조한다.
4. 비용 비교표(CAN vs 직접 3개국)를 작성해 IP 총괄 보고 후 최종 결정한다.

담당: 국내 IP팀(R), 현지 대리인(C), IP 총괄(A)

---

### Step 5. 출원 경로 확정 및 포트폴리오 관리 대장 업데이트 (출원 D-30 이전)

1. 국가별 출원 경로를 포트폴리오 관리 대장에 기록한다.
2. 출원 예정일, 대리인 정보, 경로 선택 사유를 함께 기록한다.
3. 마드리드 경로 선택 국가에 대해 5년 중앙 공격 기간 종료일을 캘린더에 입력한다.
4. 각국 심사 예상 완료 시점을 기준으로 이의·거절 대응 준비 일정을 설정한다.
5. 출원 경로 확정 사실을 재무팀(비용 집행 기준)과 경영진(진행 현황)에 통보한다.

담당: 국내 IP팀(R), IP 총괄(A), 재무팀(I), 경영진(I)

- [ ] 출원 예정일 기준 등록 후 3년 사용선언 기한을 사전 계산해 캘린더에 입력했다

- [ ] 안데스 3년 불사용 취소에 대비한 사용증거 수집 루틴을 설정했다

### 아르헨티나 해당 기업 추가 확인

- [ ] 출원 예정일 기준 중간 사용선언 기한(등록 후 5~6년차)을 캘린더에 입력했다
- [ ] 갱신 시 사용선언 요건을 확인하고 증거 수집 계획을 병행해 수립했다

### 파나마 해당 기업 추가 확인

- [ ] 허브 기능(유통 경유) 목적인지 실판매 목적인지 구분해 류 수를 결정했다
- [ ] 파나마 5년 불사용 취소 기준을 고려해 실사용 가능 범위만 지정했다

## 10. 실패 사례 및 주의사항

### 실패 패턴 1: "마드리드 하나로 중남미 전체를 처리하려 했다"

상황: 예산 절감을 이유로 마드리드 경로만으로 브라질·멕시코·안데스 권역을 모두 지정.
브라질 INPI에서 명세서 범위 문제로 거절 통보 수취.
멕시코에서 심사 기간이 길어지는 동안 딜러가 직접 출원으로 선등록.

결과: 브라질 재출원 시 직접 출원 비용 추가 발생.
멕시코 딜러 선등록 문제로 취소 절차 진행 중.

교훈: 브라질·멕시코는 처음부터 직접 출원으로 설계한다.
마드리드 경로의 관리 편의성이 이 두 나라에서는 오히려 리스크를 만든다.

---

### 실패 패턴 2: "CAN 상표 선택 시 볼리비아 포함을 간과했다"

상황: 콜롬비아·페루·에콰도르 3개국 동시 보호를 위해 CAN 상표 출원 진행.
볼리비아에서 사업 계획이 없었으나 CAN 상표 효력이 자동 발생.
볼리비아 현지 기업이 CAN 상표 등록에 이의를 제기하면서 전체 등록에 영향.

결과: CAN 상표 심사 지연. 볼리비아 이의 대응 비용 추가 발생.

교훈: CAN 상표 선택 전 볼리비아 사업 계획 유무를 반드시 확인한다.
볼리비아에서 리스크가 있으면 CAN 대신 3개국 직접 출원으로 전환한다.

---

### 실패 패턴 3: "파나마를 마드리드로 지정했다가 미가맹 사실을 뒤늦게 확인"

상황: 중남미 8개국 전체를 마드리드로 지정해 출원서 작성 완료.
WIPO 제출 직전 파나마가 마드리드 협정 미가맹임을 확인.
파나마 출원이 누락된 채로 다른 국가 출원이 진행됨.

결과: 파나마에 대한 별도 직접 출원을 뒤늦게 시작.
그 사이 제3자가 파나마에서 유사 표장을 선출원.

교훈: 출원 경로 설계 첫 단계에서 파나마 마드리드 미가맹을 확인하고 분리한다.
파나마 직접 출원은 마드리드 출원과 동시에 착수한다.

---

### 실패 패턴 4: "마드리드 기본출원 취하 후 중남미 등록 전체 위험"

상황: 한국 기본출원에 대한 이의신청이 제기돼 기본출원을 취하.
마드리드 경로로 지정된 중남미 5개국 출원이 5년 이내 기본출원 취하로 인해 중앙 공격 대상이 됨.
중남미 각국에서 개별 직접 출원으로 전환하는 절차 진행 필요.

결과: 전환 절차 비용과 공백 기간 중 제3자 출원 리스크 발생.

교훈: 마드리드 경로 선택 시 기본출원의 안정성을 먼저 점검한다.
중남미 핵심 시장(브라질·멕시코)은 마드리드 기본출원 상태와 무관하게 독립적으로 직접 출원한다.
