# 제02장. Pre-filing: 상표 포트폴리오 설계

> **이 장의 목적**: 출원 전에 "무엇을, 어디에, 어떻게" 등록할지 결정한다.
> 출원 국가와 표장 구조를 잘못 설계하면, 이후 모든 절차가 비효율적이 된다.
> 이 장을 완료한 후 03장(상표 검색)으로 이동한다.

---

## 1. 핵심 원칙

> **"브랜드 구조를 먼저 설계하라. 표장 하나가 나라마다 다르게 작동한다."**

중남미에서 상표 포트폴리오를 잘못 설계하는 실수는 세 가지로 압축된다.

첫째, 한국에서 쓰는 브랜드명을 그대로 출원했다가 현지 언어에서 부정적 의미가 발견된다.
둘째, 표장을 단일로 설계했다가 국가별 표기·발음 차이로 소비자 인지가 분산된다.
셋째, 지정 상품·서비스 범위를 너무 좁게 또는 너무 넓게 잡아 실제 사업과 불일치가 발생한다.

포트폴리오 설계는 "출원서 작성"이 아니라 "사업 보호 구조 설계"다.
국가, 표장 유형, 상품 범위, 브랜드 계층 네 가지를 먼저 결정해야 출원 절차가 시작된다.

---

## 2. 설계 대상 확정

포트폴리오를 설계하기 전에 아래 네 가지를 확정한다.

| 설계 요소 | 결정 내용 | 확인 방법 |
|-----------|-----------|-----------|
| **보호 표장** | 문자상표, 도형상표, 복합상표, 색채상표 중 선택 | 사업에서 실제 사용하는 표장 유형 기준 |
| **보호 국가** | 01장 Decision Box 결과 기준 | 시장 진입 계획 + 리스크 매트릭스 |
| **지정 상품·서비스** | 니스 분류(Nice Classification) 기준 상품·서비스류 | 현재 + 3년 이내 사업 계획 반영 |
| **브랜드 계층** | 단일 브랜드 vs 멀티 브랜드 (기업명/제품명/서브브랜드) | 제품 포트폴리오 구조와 유통 전략 기준 |

> **운영 원칙**: 위 네 가지 중 하나라도 미확정이면 출원을 진행하지 않는다.
> 미확정 상태의 출원은 나중에 보정 비용과 분쟁 리스크를 만든다.

---

## 3. 주요 리스크

### 3-1. 언어·발음·속어 리스크

중남미는 스페인어권(7개국)과 포르투갈어권(브라질)으로 나뉜다.
같은 스페인어권이라도 국가별로 의미가 달라지는 단어가 존재한다.

| 리스크 유형 | 설명 | 실무 예시 |
|-------------|------|-----------|
| **직역 의미** | 브랜드명이 현지어로 부정적 단어와 동음이의 | 영문명이 멕시코 속어와 일치하는 경우 |
| **발음 유사** | 현지어 발음 기준 기존 등록상표와 혼동 | 자음·모음 발음 규칙이 한국과 상이 |
| **문자 표기** | 영문 표기가 스페인어·포르투갈어 철자 규칙과 충돌 | 악센트 부호 유무에 따른 의미 차이 |
| **포르투갈어 전용** | 브라질에서만 다른 의미로 인식 | 스페인어권에서는 무의미한 단어가 포르투갈어권에서 속어 |

**실행 지침**: 출원 전 원어민(스페인어·포르투갈어) 검토를 반드시 거친다.
현지 대리인에게 "의미 검토 의견서"를 서면으로 요청하고 파일에 보관한다.

---

### 3-2. 상품·서비스 범위 설계 리스크

| 리스크 유형 | 설명 | 결과 |
|-------------|------|------|
| **과소 지정** | 실제 판매 상품을 포함하지 못한 범위로 출원 | 경쟁사가 인접 상품류에 유사 표장 등록 가능 |
| **과다 지정** | 실제 사용하지 않는 상품류를 포함한 범위로 출원 | 불사용 취소 청구 시 해당 부분 소멸 위험 |
| **미래 사업 미반영** | 현재 사업만 기준으로 설계 | 3년 내 신제품 출시 시 추가 출원 비용 발생 |
| **국가별 분류 차이** | 동일 상품이 국가마다 다른 니스 류에 속하는 경우 | 국가별 개별 검토 없이 일괄 적용하면 누락 발생 |

**실행 지침**: 현재 판매 상품 + 3년 이내 출시 예정 상품·서비스를 함께 목록화한다.
영업팀과 마케팅팀이 Consulted로 참여해 사업 계획을 반영한다.

---

### 3-3. 브랜드 계층 설계 리스크

| 시나리오 | 리스크 | 대응 방향 |
|----------|--------|-----------|
| 기업명과 제품명이 다름 | 별도 출원 필요 여부 미판단 | 사용 방식 기준으로 각각 출원 검토 |
| 로컬 서브브랜드 사용 계획 | 서브브랜드 선점 리스크 | 진입 전 서브브랜드도 출원 대상 포함 |
| 유통사가 현지 브랜드 병기 | 유통사 명의 출원 가능성 | 계약서에 상표 소유 조항 명시 (09장 참조) |
| 라이선스 구조 예정 | 라이선시 명의 출원 리스크 | 라이선스 계약 전 본사 선등록 필수 |

---

## 4. Decision Box: 포트폴리오 설계 판단 기준

### 4-1. 브랜드 구조 선택

```
[Decision Box A: 단일 브랜드 vs 멀티 브랜드]

IF (전 제품에 동일 브랜드명 사용 + 국가별 표기 차이 없음)
    → 단일 표장 출원 전략 적용
      단, 문자+도형 복합의 경우 문자 단독 출원도 병행 권장

IF (국가별 현지화 브랜드명 사용 예정)
    → 현지명을 별도 표장으로 출원 (본사 소유)
      현지 파트너 명의 출원 금지 조항을 계약서에 명시

IF (기업 브랜드 + 제품 브랜드 + 서브브랜드 3단계 구조)
    → 3개 모두 독립 출원 대상으로 분류
      우선순위: 기업 브랜드 > 핵심 제품 브랜드 > 서브브랜드

IF (온라인 전용 브랜드명 또는 소셜 핸들명 별도 사용)
    → 해당 명칭도 상표 출원 대상에 포함
      도메인·소셜 계정은 상표 등록과 별개이므로 별도 보호 조치 필요
```

### 4-2. 지정 상품·서비스 범위 선택

```
[Decision Box B: 광범위 출원 vs 최소범위 출원]

IF (브라질·멕시코·아르헨티나: 사용선언 또는 불사용 취소 제도 적용)
    → 실제 사용 가능한 범위만 지정 (과다 지정 금지)
      사용선언 제출 시 미사용 항목 포함 시 리스크 증가

IF (안데스 권역: 콜롬비아·페루·에콰도르 3년 불사용 취소 적용)
    → 3년 내 실제 사용 예정 상품·서비스류만 지정
      사용 예정이 불분명한 류는 별도 추가 출원으로 처리

IF (예산 제약으로 류 수를 줄여야 하는 경우)
    → 핵심 상품류(실판매 기준) 우선 + 인접 류 방어 출원 순서로 단계적 진행
      핵심 류 누락 시 사업 보호 불가이므로 핵심 류는 예산과 무관하게 포함

IF (신규 시장으로 사업 범위가 불확실한 경우)
    → 사업 계획서에서 확인된 상품·서비스류만 지정
      "혹시 몰라서 많이 지정"은 불사용 취소 빌미 제공
```

### 4-3. 표장 유형 선택

```
[Decision Box C: 표장 유형 선택]

IF (문자만 사용하는 브랜드)
    → 문자 표장 단독 출원
      스탠더드 캐릭터(표준문자) 출원 우선 검토

IF (로고가 브랜드 핵심)
    → 도형 포함 복합 표장 출원 + 문자 단독 출원 병행
      도형만의 출원도 검토 (로고 단독 사용 가능성 고려)

IF (특정 색채가 브랜드 정체성)
    → 색채상표 등록 가능 여부를 국가별로 확인 후 출원
      중남미 국가별 색채상표 등록 요건 상이

IF (슬로건·광고 문구 보호 필요)
    → 별도 표장으로 출원 검토 (식별력 인정 여부 사전 검토 필수)
```

## 5. 실행 절차

### Step 1. 보호 표장 목록 작성 (D-90 이전)

사업에서 실제 사용 중이거나 사용 예정인 모든 표장을 목록화한다.

- 현재 사용 중인 표장: 문자·로고·슬로건 전부 수집
- 3년 이내 사용 예정 표장: 신제품 브랜드명·서브브랜드 포함
- 온라인·소셜 미디어에서 사용 중인 표기 변형 포함
- 현지 광고에서 번역·변형해 사용할 예정인 명칭 포함

**담당**: 국내 IP팀(R), 마케팅팀(C), 현지 대리인(C)

---

### Step 2. 언어 리스크 점검 (D-75 이전)

보호 표장 목록이 완성되면 즉시 언어 리스크 점검을 시작한다.

1. 스페인어 원어민(또는 현지 대리인)에게 의미·발음 검토 요청
2. 포르투갈어 원어민(브라질 대리인)에게 별도 검토 요청
3. 국가별 속어·비속어 포함 여부 확인
4. 검토 결과를 서면으로 수취해 파일 보관 (증거 저장소에 저장)
5. 문제 발견 시 대안 표장 목록 작성 후 Step 1로 부분 반복

**담당**: 국내 IP팀(R), 현지 대리인(R), IP 총괄(A)

---

### Step 3. 지정 상품·서비스 초안 작성 (D-60 이전)

1. 영업팀·마케팅팀에서 현재 판매 상품·서비스 전체 목록 수령
2. 3년 이내 출시 예정 상품·서비스 목록 추가
3. 니스 분류 기준으로 상품·서비스류 분류
4. 국가별 분류 차이 확인 (특히 브라질 INPI 분류 기준)
5. 불사용 취소 제도 적용 국가(멕시코·안데스 권역 등) 기준으로 범위 재검토
6. 최종 지정 상품·서비스 목록을 영업팀 확인 후 확정

**담당**: 국내 IP팀(R), 영업팀(C), 현지 대리인(C), IP 총괄(A)

---

### Step 4. 국가별 출원 대상 표장 확정 (D-45 이전)

| 확인 항목 | 내용 |
|-----------|------|
| 국가별 출원 표장 | 동일 표장 또는 국가별 현지화 표장 여부 확정 |
| 국가별 류 수 | 국가별 비용·사용 요건 고려해 류 수 결정 |
| 우선 출원 순서 | 브라질·멕시코 최우선 → 안데스 권역 → 남부·허브 순 |
| 마드리드 경로 활용 여부 | 04장 기준으로 판단 (브라질·멕시코는 직접 출원 권장) |

---

### Step 5. 포트폴리오 관리 대장 등록 (출원 전 완료)

확정된 출원 계획을 포트폴리오 관리 대장에 전부 입력한다.

- 출원 예정 표장별: 국가, 표장 유형, 지정 류, 예정 출원일, 담당 대리인
- 각 국가 기준 사용선언 예정일·갱신 예정일 사전 계산
- 캘린더 시스템에 주요 기한 입력 (이 단계에서 입력, 등록 후 갱신)

---

## 6. 체크리스트

### 공통 (모든 기업)

- [ ] 보호 대상 표장 전체 목록을 작성했다
- [ ] 문자·도형·복합·색채 등 표장 유형을 분류했다
- [ ] 스페인어 의미·발음 리스크 점검을 완료했다
- [ ] 포르투갈어(브라질) 의미·발음 리스크 점검을 완료했다
- [ ] 니스 분류 기준 지정 상품·서비스 목록을 완성했다
- [ ] 3년 이내 출시 예정 상품·서비스를 목록에 포함했다
- [ ] 국가별 출원 대상 표장과 류를 확정했다
- [ ] 포트폴리오 관리 대장에 출원 계획을 입력했다
- [ ] 현지 대리인에게 의미 검토 의견서를 서면으로 수취했다

### 브라질 해당 기업 추가 확인

- [ ] 포르투갈어 표기 및 발음 검토를 브라질 대리인에게 별도 의뢰했다
- [ ] INPI 분류 기준으로 지정 상품·서비스를 재검토했다
- [ ] 복합 표장인 경우 문자 단독 출원 병행 여부를 결정했다
- [ ] 불사용 취소(카두시다지) 기준인 5년을 고려해 사용 범위를 설계했다

### 멕시코 해당 기업 추가 확인

- [ ] 등록일 3주년 후 3개월 사용선언 창을 고려해 지정 범위를 재검토했다
- [ ] 스페인어(멕시코 지역 속어 포함) 검토를 완료했다
- [ ] 등록 후 사용선언 기한을 포트폴리오 관리 대장에 사전 입력했다

### 안데스 권역(콜롬비아·페루·에콰도르) 해당 기업 추가 확인

- [ ] 3년 불사용 취소 기준으로 지정 범위를 재검토했다
- [ ] 안데스 공통 규범을 전제로 하되 출원은 국가별로 직접 진행한다는 점을 04장 기준으로 확인했다

## 9. 실패 사례 및 주의사항

### 실패 패턴 1: "한국 브랜드명 그대로 출원했다가 현지어 문제 발생"

**상황**: 한국에서 검증된 브랜드명을 스페인어 검토 없이 멕시코에 출원.
공고 이후 현지 소비자와 경쟁사로부터 해당 명칭이 비속어와 유사하다는 이의 제기.

**결과**: 이의 대응에 6개월 이상 소요. 일부 시장에서는 브랜드명 변경을 검토해야 하는 상황 발생.
변경 시 재출원 비용과 마케팅 비용 추가 발생.

**교훈**: 언어 리스크 점검은 출원 전 의무 절차다. "그냥 영문 표기라 괜찮을 것 같다"는 판단은 금물이다.
스페인어와 포르투갈어는 별도로 각각 검토해야 한다.

---

### 실패 패턴 2: "지정 류를 많이 잡아두면 나중에 편할 것이다"

**상황**: 초기 출원 시 실제 사업과 무관한 20개 류를 지정해 출원.
멕시코 등록 후 3년이 지나 사용선언을 제출했는데, 실제 사용 증거가 있는 류가 3개뿐.

**결과**: 사용 증거 없는 17개 류에 대한 사용선언 제출 불가.
경쟁사가 해당 류에 대해 불사용 취소 청구. 실제 사업에 필요한 류까지 분쟁 대상에 포함됨.

**교훈**: 지정 범위는 "실제 사용할 수 있는 범위"로 설계한다.
멕시코·안데스 권역은 특히 3년 기준으로 설계가 필수다.

---

### 실패 패턴 3: "현지 유통사가 현지 브랜드명을 먼저 등록했다"

**상황**: 멕시코 유통사와 협업하면서 현지어 브랜드명 번역을 유통사에 위임.
유통사가 번역한 현지 브랜드명을 자사 명의로 선출원.

**결과**: 계약 종료 후 현지 브랜드명 사용 불가. 신규 명칭 개발 및 재출원 필요.
소비자 인지도 리셋에 따른 마케팅 비용 재투입.

**교훈**: 현지화 브랜드명은 반드시 본사 명의로 직접 출원한다.
유통사에게는 사용권만 부여하고, 출원 주체는 항상 본사다.
계약서에 "상표 출원 금지 조항"과 "기존 출원 이전 의무"를 명시한다 (09장 참조).

---

### 실패 패턴 4: "도형 표장만 출원하고 문자 표장을 빠뜨렸다"

**상황**: 로고(도형+문자 복합)를 하나의 표장으로만 출원.
경쟁사가 문자 단독 표장을 동일한 류에 먼저 출원.

**결과**: 본사의 로고에 포함된 문자 표기가 경쟁사 상표권과 충돌 발생.
경쟁사로부터 경고장 수취 후 협상 또는 분쟁 절차 진입.

**교훈**: 복합 표장 출원 시 문자 단독 출원을 반드시 병행한다.
도형 단독 출원도 로고 사용 방식에 따라 검토한다.
