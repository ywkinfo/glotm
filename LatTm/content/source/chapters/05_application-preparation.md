# 제05장. Filing 실무: 출원서 작성 및 제출

> 이 장의 목적: 출원 경로가 확정된 후 실제 출원서를 작성하고 각국 상표청에 제출하는 절차를 실행한다.
> 04장(출원 경로 선택) 완료 후 이 장을 진행하고, 완료 후 06장(공고·이의·거절 대응)으로 이동한다.

---

## 1. 핵심 원칙

> "명세가 결과를 결정한다. 출원서는 권리의 경계선이다."

출원서 작성은 단순한 서류 작업이 아니다.
상품·서비스 명세 범위가 곧 보호 범위이며, 잘못된 명세는 등록 후에도 수정이 불가능하거나 극히 제한적이다.
중남미 각국은 출원 언어, 전자출원 여부, 명세 기준이 다르다.
국가별 요건을 확인하지 않고 일괄 작성한 출원서는 거절 또는 보호 공백의 원인이 된다.

출원서 작성에서 반드시 지켜야 할 세 가지 원칙:
1. 실사용 가능한 상품·서비스만 명세에 포함한다.
2. 국가별 언어 요건과 전자출원 방식을 사전에 확인한다.
3. 출원 접수증을 수령하는 즉시 기한 캘린더에 등록한다.

---

## 2. 국가 범위 및 적용 조건

이 장은 04장에서 확정된 출원 경로 기준으로 아래 8개국 출원 실무에 적용된다.

| 국가 | 상표청 | 출원 언어 | 전자출원 여부 | 현지 대리인 필수 여부 |
|------|--------|-----------|--------------|----------------------|
| 브라질 | INPI | 포르투갈어 | 필수 (INPI 전자출원 시스템) | 필수 |
| 멕시코 | IMPI | 스페인어 | 필수 (e-MARCANET) | 권장 |
| 콜롬비아 | SIC | 스페인어 | 가능 (SIPI 시스템) | 권장 |
| 아르헨티나 | INPI-AR | 스페인어 | 가능 | 권장 |
| 칠레 | INAPI | 스페인어 | 가능 | 권장 |
| 페루 | INDECOPI | 스페인어 | 가능 | 권장 |
| 에콰도르 | SENADI | 스페인어 | 가능 | 권장 |
| 파나마 | DIGERPI | 스페인어 | 가능 | 권장 |

> 브라질 주의: 브라질은 현지 대리인 없이 출원이 사실상 불가능하다. INPI 전자출원 시스템은 브라질 현지 대리인 자격증 번호 기반으로 운영된다.

## 4. Decision Box: 출원서 작성 판단 기준

이 Decision Box는 출원서 초안 작성 전에 실행한다.

```
[Step 1] 우선권 주장 여부를 결정한다.

  IF (한국 출원일로부터 6개월 이내 + 파리 협약 가입국에 출원)
      → 우선권 주장 출원으로 진행
        한국 출원 번호·출원일을 출원서에 기재
        한국 출원서 사본 + 공증 준비 (국가별 아포스티유 요건 확인)

  IF (한국 출원일로부터 6개월 초과)
      → 우선권 주장 불가
        현재 일자 기준 신규 출원으로 진행

  IF (마드리드 경로 선택 국가)
      → 우선권은 마드리드 출원서에서 일괄 처리
        각국 직접 출원서 우선권 기재 불필요


[Step 2] 상품·서비스 명세 범위를 결정한다.

  IF (브라질 INPI 출원)
      → INPI 공식 분류 목록 기준으로 포르투갈어 명세 작성
        브라질 대리인이 명세 최종 검토 후 제출
        불사용 취소(카두시다지) 기준 5년을 고려해 실사용 범위만 포함

  IF (멕시코 IMPI 출원)
      → 스페인어 명세 작성 + 등록일 3주년 후 3개월 사용선언이 가능한 범위만 포함
        사용선언 대상이 되는 상품·서비스를 출원 시점에 확정

  IF (안데스 권역 콜롬비아·페루·에콰도르 출원)
      → 3년 내 실사용 예정 범위만 포함
        Decision 486을 고려하되 국가별 직접출원 명세는 각각 점검

  IF (아르헨티나 INPI-AR 출원)
      → 5~6년차 중간 사용선언 대상 범위를 고려해 명세 설계
        갱신 시 사용선언 가능한 범위만 포함

  IF (칠레·파나마·기타)
      → 실사용 가능 범위 기준으로 명세 작성
        파나마: 허브 목적이면 유통 관련 류 포함 검토


[Step 3] 출원 서류 준비 방식을 결정한다.

  IF (브라질 직접 출원)
      → 현지 대리인이 서류 작성 + INPI 전자출원 시스템으로 제출
        위임장(Procuração)을 포르투갈어로 작성 + 공증
        법인 등기부 등본 포르투갈어 번역 + 아포스티유 첨부

  IF (멕시코 직접 출원)
      → e-MARCANET을 통해 현지 대리인이 전자출원
        위임장(Poder Notarial) 스페인어 작성 + 공증

  IF (마드리드 경로 선택 국가)
      → WIPO MM2 양식으로 한국 특허청 경유 제출
        현지 서류 별도 불필요 (거절 통보 시 현지 대리인 선임)

  IF (안데스 권역 직접출원)
      → 각국 대리인을 통해 SIC / INDECOPI / SENADI에 개별 접수
        동일 사업군이라도 국가별 명세 표현은 따로 점검


[Step 4] 표장 파일 요건을 확인한다.

  IF (도형·복합 표장 포함)
      → 국가별 이미지 파일 요건 확인 (해상도·포맷·색상)
        브라질: JPEG 또는 PNG, 최소 300DPI
        멕시코: JPG 포맷, 300DPI 이상

  IF (색채상표 출원)
      → 색상 번호(Pantone 또는 CMYK) 명기 필수
        국가별 색채상표 등록 요건 사전 확인

  IF (한글 또는 한자 포함 표장)
      → 음역(발음) 병기 또는 번역 기재 요건 국가별 확인
        한글 표장은 로마자 표기 병행 권장
```

## 5. 실행 절차

### Step 1. 출원 서류 체크리스트 확인 (출원 D-30 이전)

국가별로 필요한 서류 목록을 확인하고 준비를 시작한다.

1. 04장에서 확정된 출원 경로별로 필요 서류 목록을 작성한다.
2. 위임장 작성 방식(공증 여부, 아포스티유 요건)을 국가별로 확인한다.
3. 법인 출원의 경우 법인 등기부 등본 원본 + 번역 + 아포스티유 준비 일정을 설정한다.
4. 표장 이미지 파일이 각국 요건을 충족하는지 확인한다.
5. 우선권 주장 여부와 우선권 서류 준비 필요성을 결정한다.

담당: 국내 IP팀(R), 현지 대리인(C), IP 총괄(A)

---

### Step 2. 상품·서비스 명세 초안 작성 (출원 D-21 이전)

1. 02장에서 확정된 지정 상품·서비스 목록을 기반으로 국가별 명세 초안을 작성한다.
2. 브라질 출원의 경우 INPI 공식 분류 목록 기준으로 포르투갈어 명세를 작성한다.
3. 스페인어권 국가 출원의 경우 니스 분류 기준 스페인어 명세를 작성한다.
4. 불사용 취소 제도 적용 국가(멕시코·안데스 권역·아르헨티나)는 실사용 가능 범위를 재확인한다.
5. 명세 초안을 현지 대리인에게 검토 의뢰하고 수정 의견을 수령한다.
6. 최종 명세를 IP 총괄에 보고하고 확정한다.

담당: 국내 IP팀(R), 현지 대리인(C), 영업팀(C), IP 총괄(A)

---

### Step 3. 출원서 작성 및 현지 대리인 검토 (출원 D-14 이전)

1. 현지 대리인이 각국 양식에 맞는 출원서 초안을 작성한다.
2. 국내 IP팀이 출원서 내용(표장, 명세, 출원인 정보)을 검토하고 오류를 확인한다.
3. 우선권 주장 출원의 경우 한국 출원 번호·출원일 기재를 확인한다.
4. 표장 이미지 파일이 출원서와 동일하게 첨부됐는지 확인한다.
5. 수정 사항 반영 후 최종 출원서를 확정한다.

담당: 현지 대리인(R), 국내 IP팀(R), IP 총괄(A)

---

### Step 4. 서류 공증 및 아포스티유 처리 (출원 D-10 이전)

국가별 공증·아포스티유 요건에 따라 서류를 처리한다.

| 서류 유형 | 처리 방식 | 해당 국가 |
|-----------|-----------|-----------|
| 위임장 (법인 대표 서명) | 공증 후 아포스티유 첨부 | 브라질, 아르헨티나, 파나마 |
| 위임장 (법인 대표 서명) | 공증만 (아포스티유 불필요) | 멕시코, 콜롬비아, 칠레, 페루, 에콰도르 |
| 법인 등기부 등본 | 번역 + 아포스티유 첨부 | 브라질, 아르헨티나 |
| 우선권 서류 (한국 출원서 사본) | 공증 번역 + 아포스티유 | 우선권 주장 국가 전체 |

> 아포스티유 처리 기간: 한국 외교부 또는 법원 기준 3~5영업일 소요. 출원일 D-10 이전에 착수한다.

담당: 국내 IP팀(R), 법무팀(C), IP 총괄(A)

---

### Step 5. 출원 접수 및 접수증 수령 (출원일)

1. 현지 대리인이 각국 상표청에 출원서를 제출한다.
2. 출원 접수증(번호·일자)을 수령하는 즉시 국내 IP팀에 전달한다.
3. 출원 번호와 출원일을 포트폴리오 관리 대장에 즉시 입력한다.
4. 국가별 심사 예상 완료 시점을 계산해 이의·거절 대응 준비 일정을 설정한다.
5. 멕시코 출원의 경우: 등록 예정 시점 기준으로 사용선언 준비 알림을 가등록하고, 등록 후 실제 등록일 기준으로 `3주년 + 3개월` 창을 재계산한다.
6. 아르헨티나 출원의 경우: 등록 예상일 기준 5~6년차 중간 사용선언 기한을 즉시 캘린더에 등록한다.

담당: 현지 대리인(R), 국내 IP팀(R), IP 총괄(A), 재무팀(I)

---

### Step 6. 우선권 서류 후속 제출 (출원 접수 후 유예 기간 내)

우선권 주장 출원의 경우, 출원 시 우선권 서류를 동시 제출하지 못한 경우 유예 기간 내에 후속 제출한다.

| 국가 | 우선권 서류 제출 유예 기간 | 내부 마감 기준 |
|------|--------------------------|---------------|
| 브라질 | 출원일로부터 60일 이내 | 출원일 +45일 |
| 멕시코 | 출원일로부터 3개월 이내 | 출원일 +60일 |
| 콜롬비아 | 출원일로부터 3개월 이내 | 출원일 +60일 |
| 아르헨티나 | 출원일로부터 3개월 이내 | 출원일 +60일 |
| 칠레 | 출원일로부터 3개월 이내 | 출원일 +60일 |

담당: 국내 IP팀(R), 현지 대리인(R), IP 총괄(A)

## 6. 국가별 출원 서류 요건 비교표

## 10. 실패 사례 및 주의사항

### 실패 패턴 1: "포르투갈어 명세를 스페인어 번역으로 제출했다"

상황: 스페인어권 7개국 출원서를 작성한 후 포르투갈어 번역 없이 동일한 명세를 브라질 INPI에 제출 의뢰.
브라질 현지 대리인이 스페인어 명세를 그대로 사용해 INPI 전자출원 시스템에 접수.
INPI 심사 과정에서 포르투갈어 명세 요건 불비로 보완 요구 발생.

결과: 보완서 제출 지연으로 심사 일정 3~4개월 추가 소요.
보완 과정에서 명세 범위가 원래 의도보다 좁게 수정됨.

교훈: 브라질은 포르투갈어 전용 명세를 별도 작성한다.
스페인어권 국가 명세 작성과 브라질 명세 작성을 별개 업무로 구분한다.

---

### 실패 패턴 2: "위임장 아포스티유 처리를 출원일에 맞추려다 기한을 놓쳤다"

상황: 브라질 출원 예정일 D-3에 아포스티유 신청을 시작.
한국 외교부 발급 소요 기간(3~5영업일)을 간과해 출원일까지 서류 미비.
현지 대리인이 위임장 없이 출원을 선진행 후 유예 기간 내 보완을 시도.

결과: 유예 기간 내 아포스티유 처리 지연 반복.
INPI에서 보완 서류 미접수로 해당 출원이 무효 처리됨.

교훈: 위임장 아포스티유 처리는 출원 D-10 이전에 착수한다.
서류 처리 기간을 항상 역산해 내부 마감을 설정한다.

---

### 실패 패턴 3: "멕시코 출원서에 미사용 상품류를 과다 포함했다"

상황: 한국 내 등록 상표와 동일한 명세(5개 류)로 IMPI에 출원.
멕시코 진출 당시 실제 판매 상품은 1개 류에 해당.
등록 후 사용선언 창에서 4개 류의 사용 실적 입증이 불가했다.

결과: 등록 후 사용선언 창에서 4개 류를 유지하지 못해 권리 공백이 발생.
핵심 상품류 1개만 권리 유지, 인접 류 보호 공백 발생.

교훈: 멕시코 출원 시 등록일 3주년 후 3개월 창 안에 실제 사용을 선언할 수 있는 범위만 지정한다.
사용선언 대상 류를 출원 시점에 영업팀과 협의해 결정한다.

---

### 실패 패턴 4: "접수증 번호를 포트폴리오 관리 대장에 입력하지 않았다"

상황: 7개국 동시 출원을 진행하며 현지 대리인이 이메일로 접수증을 전달.
국내 IP팀이 이메일 수신만 확인하고 포트폴리오 관리 대장 입력을 미룸.
6개월 후 콜롬비아 SIC에서 거절 이유 통지가 이메일로 전달됐으나 출원 번호 미기록으로 문서 연결 불가.

결과: 거절 이유 통지 확인 지연 + 의견서 제출 기한 도과 + 해당 국가 출원 무효.

교훈: 출원 접수증 수령 즉시 포트폴리오 관리 대장에 입력한다.
접수증 수령 확인을 대리인에게 별도 요청하고, 미수령 시 D+3일 이내 재요청한다.

---

### 실패 패턴 5: "안데스 3국 명세를 각국 사업 계획 검토 없이 사실상 동일하게 밀어 넣었다"

상황: 콜롬비아·페루·에콰도르 3개국 동시 보호를 위해 3건을 병행 출원하면서,
사실상 동일 명세를 그대로 복제해 에콰도르에서 실제 판매하지 않는 상품류를 포함했다.
에콰도르 3년 불사용 취소 제도에 따라 제3자가 해당 상품류에 대해 취소 청구.

결과: 에콰도르 등록의 일부가 취약해지고, 권역 전체 운영 계획도 다시 손봐야 했다.

교훈: Decision 486을 참고하더라도 명세는 국가별 사업 계획과 사용 가능성을 기준으로 따로 조정한다.
