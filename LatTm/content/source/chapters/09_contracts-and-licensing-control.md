# 제09장. 계약 구조: 유통·라이선스·프랜차이즈 통제

> **이 장의 목적**: 유통사·라이선시·프랜차이지 등 파트너를 통한 브랜드 운영에서 발생하는 상표 리스크를 계약 구조로 통제한다.
> 파트너가 가장 큰 리스크 발생원이다. 사전 예방 없이 사후 대응은 시간·비용이 수배 이상 소요된다.
> 08장(상표 사용 및 증거 관리)과 병행하고, 09장 완료 후 10장(모니터링 시스템)으로 이동한다.

---

## 1. 핵심 원칙

> **"파트너 리스크가 가장 크다. 계약 전 선등록, 계약서에 통제 조항, 운영 중 증거 관리를 반드시 병행한다."**

중남미에서 상표 분쟁의 가장 흔한 발생 원인은 경쟁사가 아니라 **파트너**다.
이 단계의 실패는 세 가지 형태로 나타난다.

첫째, 유통사·딜러가 계약 체결 전 또는 계약 기간 중 자사 표장과 동일·유사한 상표를 현지에 선출원·선등록한다. 이를 무효화하는 데 통상 1~3년이 소요되며, 그 기간 동안 해당 국가 사업이 마비되거나 협상에서 불리한 위치에 놓인다.

둘째, 라이선스·유통 계약에 상표 관련 조항이 없거나 불명확해 파트너가 상표를 무단으로 변형·확장·재라이선스한다. 불사용 취소 대응 시 파트너 사용을 권리자 사용으로 인정받지 못하는 문제도 발생한다.

셋째, 계약 종료 후 파트너가 상표 사용을 계속하거나, 수집한 고객 데이터·브랜드 자산을 활용해 유사 브랜드를 운영한다.

계약 구조 통제는 **본사 선등록 → 계약서 조항 설계 → 운영 중 모니터링 → 계약 종료 절차** 네 단계가 연결된 시스템으로 운영한다.

---

## 2. 국가 범위 및 적용 조건

이 장의 절차는 파트너를 통해 브랜드를 운영하는 모든 국가에 적용된다.
국가별 계약법·상표법 차이가 있으므로, 계약서는 각국 준거법 기준으로 현지 대리인 검토를 거쳐야 한다.

| 국가 | 파트너 선출원 리스크 | 라이선스 등록 필요성 | 계약 준거법 위험도 | 주요 유의점 |
|------|------------------|------------------|--------------------|------------|
| 브라질 | ●●●●○ | 권장 (INPI 등록) | 높음 | 현지 노동법·대리점법 강행규정 多 |
| 멕시코 | ●●●●● | 권장 (IMPI 등록) | 높음 | 딜러 선출원 빈도 최고, 대리점 보호법 적용 주의 |
| 콜롬비아 | ●●●○○ | 권장 (SIC 등록) | 중간 | 안데스 CAN 규범 적용 |
| 아르헨티나 | ●●●●○ | 필수 (INPI-AR 등록) | 높음 | 대리점 보호법 강행 적용 |
| 칠레 | ●●●○○ | 권장 | 중간 | 상대적으로 예측 가능한 법 환경 |
| 페루 | ●●●○○ | 권장 | 중간 | 안데스 CAN 규범 적용 |
| 에콰도르 | ●●○○○ | 권장 | 낮음 | 안데스 CAN 규범 적용 |
| 파나마 | ●●●○○ | 선택적 | 중간 | 허브 성격, 재수출 경로 통제 필수 |

> **운영 원칙**: 파트너 계약 체결 전 반드시 해당 국가에 본사 명의로 출원이 완료되어 있어야 한다.
> 출원이 완료되지 않은 상태에서 파트너 계약을 체결하는 것은 허용하지 않는다.

## 3. 주요 리스크

### 3-1. 파트너 선출원·선등록 리스크

| 리스크 유형 | 설명 | 주요 발생 국가 |
|-------------|------|----------------|
| **계약 전 선등록** | 계약 체결 전 딜러·배급사가 브랜드명을 자신의 명의로 출원 | 멕시코, 브라질, 아르헨티나 |
| **계약 중 선등록** | 계약 기간 중 파트너가 유사 표장을 별도 출원 | 전 국가 |
| **대표자 개인 명의 등록** | 법인 계약 체결 후 대표자 개인 명의로 표장 출원 | 멕시코, 콜롬비아 |
| **계약 종료 후 재등록** | 계약 종료 후 동일·유사 표장을 신규 출원 | 전 국가 |
| **현지화 표장 무단 출원** | 현지 언어로 변형한 표장을 파트너가 자신의 명의로 출원 | 브라질, 멕시코 |

**실행 지침**: 파트너 선임 전 해당 법인명, 대표자 개인명 기준의 상표 선등록 여부를 반드시 검색한다. 검색 결과는 증거 저장소에 보관한다.

---

### 3-2. 계약서 부재 또는 불완전 조항 리스크

| 리스크 유형 | 설명 | 결과 |
|-------------|------|------|
| **상표 소유권 조항 미포함** | 파트너가 자신이 사용하는 상표에 권리가 있다고 주장 | 불사용 취소·무효 절차에서 불리 |
| **사용 범위 미특정** | 사용 가능 지역·상품류·매체 범위가 불명확 | 파트너의 무단 확장 사용 통제 불가 |
| **재라이선스 금지 미포함** | 파트너가 서브 딜러에게 상표 사용권을 재부여 | 브랜드 통제 불가, 제3자 권리 발생 |
| **표시 방법 미지정** | 파트너가 표장을 변형·색상 변경·축소 사용 | 브랜드 동일성 훼손, 등록 표장과 불일치 |
| **계약 종료 후 처리 미포함** | 계약 종료 후 상표 사용·재고 처리·고객 데이터 활용 기준 없음 | 계약 종료 후 분쟁 지속 |

---

### 3-3. 라이선스 구조 운영 리스크

| 리스크 유형 | 설명 | 결과 |
|-------------|------|------|
| **라이선스 미등록** | 상표청에 라이선스 등록을 하지 않음 | 일부 국가에서 제3자 대항 불가 |
| **파트너 사용이 권리자 사용으로 불인정** | 라이선스 계약 없이 유통사가 상표 사용 | 불사용 취소 대응 시 증거로 인정 거부 |
| **로열티 구조 미설계** | 로열티 미수취로 세무·이전가격 문제 발생 | 조세당국 리스크, 관계사 간 거래 왜곡 |
| **품질 통제 조항 미포함** | 라이선시가 품질 기준을 충족하지 않는 제품에 상표 사용 | 브랜드 가치 훼손, 소비자 피해 |

---

### 3-4. 대리점·프랜차이즈 보호법 리스크

중남미 일부 국가는 대리점·유통사를 보호하는 강행법규를 두고 있다.
이 규정은 계약서 내용보다 우선하며, 파트너 계약 종료 시 보상·통지 의무가 발생할 수 있다.

| 국가 | 대리점 보호 수준 | 주요 내용 |
|------|----------------|-----------|
| 브라질 | 높음 | 일정 기간 이상 독점 유통 계약 종료 시 손해배상 청구 가능 |
| 멕시코 | 높음 | 대리점 계약 종료 시 사전 통지 및 보상 의무 |
| 아르헨티나 | 높음 | 장기 유통 계약 종료 시 경제적 보상 청구권 |
| 콜롬비아 | 중간 | 계약 기간 중 일방적 해지 제한 |
| 칠레 | 낮음 | 비교적 자유로운 계약 설계 가능 |

**실행 지침**: 브라질·멕시코·아르헨티나 파트너 계약은 현지 법무 검토를 반드시 거친다. 계약 종료 조항은 현지 강행법규와 상충하지 않도록 설계한다.

## 4. Decision Box: 계약 구조 판단 기준

```
[Decision Box A: 직접 판매 vs 파트너 의존]

IF (현지 법인을 설립해 직접 판매)
    → 상표는 본사 또는 현지 법인 명의 선택
      현지 법인 명의 선택 시: 본사와의 라이선스 계약 또는 서브라이선스 구조 설계
      본사 명의 유지 시: 현지 법인에 정식 라이선스 계약 체결 후 운영

IF (독점 유통사 1개사를 통해 판매)
    → 본사 명의 선출원 완료 후 유통 계약 체결
      상표 소유권 본사 귀속 조항 필수
      독점 범위(지역·상품류·기간)를 계약서에 명시
      계약 종료 시 상표 사용 즉시 중단 조항 포함

IF (복수 유통사 또는 비독점 채널 활용)
    → 각 파트너와 개별 라이선스 또는 사용허락 계약 체결
      파트너 간 사용 범위가 중첩되지 않도록 지역·채널 분리
      모든 파트너에게 동일한 표시 방법 기준 적용

IF (프랜차이지를 통해 브랜드 운영)
    → 프랜차이즈 계약에 상표 조항 + 품질 통제 조항 필수
      본사 명의 등록 선행 후 계약 체결
      프랜차이지의 하위 사용(서브라이선스) 금지 조항 포함
      표시 매뉴얼(Brand Manual)을 계약 부속서로 첨부

IF (전자상거래 플랫폼 셀러/대리 운영사 활용)
    → 셀러 계약에 상표 사용 허락 범위를 명시
      플랫폼 내 브랜드 등록(Brand Registry)은 본사 명의로 직접 진행
      셀러에게 플랫폼 계정 명의 이전 불가 조항 포함
```

```
[Decision Box B: 라이선스 등록 여부 판단]

IF (브라질에서 파트너가 상표를 사용하는 경우)
    → INPI에 라이선스 등록 권장
      미등록 시에도 계약은 유효하나, 제3자 대항력 및 로열티 송금 처리에 문제 발생 가능

IF (멕시코에서 파트너가 상표를 사용하는 경우)
    → IMPI에 라이선스 등록 권장
      미등록 라이선스도 당사자 간 유효하나, 분쟁 시 입증 부담 가중

IF (콜롬비아·페루·에콰도르에서 파트너가 상표를 사용하는 경우)
    → 안데스 CAN 규범 하에 계약 체결
      SIC/INDECOPI/SENADI 라이선스 등록으로 권리자 사용 입증 기반 구축

IF (아르헨티나에서 파트너가 상표를 사용하는 경우)
    → INPI-AR 라이선스 등록 강력 권장
      불사용 취소 대응 시 라이선스 등록 증거가 유효한 방어 수단

IF (불사용 취소 대응이 예상되는 경우)
    → 라이선스 계약서 + 상표청 등록 + 파트너 사용 증거(송장·광고·판매 실적) 3종 세트 준비
```

```
[Decision Box C: 계약 종료 시 대응]

IF (파트너와 합의 종료)
    → 상표 사용 중단 확인서를 서면으로 수취
      재고 처리 기간·방법 계약서에 명시
      잔존 디지털 자산(소셜 계정·도메인·플랫폼 계정) 이전 절차 포함

IF (파트너가 계약 종료 후 상표 사용 지속)
    → 경고장 발송 (현지 대리인 명의)
      상표청 라이선스 등록 말소 신청
      필요 시 행정 단속 또는 민사 가처분 신청 (11장 참조)

IF (파트너가 계약 종료 후 유사 상표로 경업)
    → 상표 출원 즉시 이의신청 (이의기간 내)
      계약상 경업금지 조항 위반으로 손해배상 청구 검토
      증거 보전(파트너 사용 증거·계약서·경업 행위 증거) 즉시 시작
```

## 5. 실행 절차

### Step 1. 파트너 선임 전 상표 선등록 확인 (계약 체결 D-60 이전)

| 확인 항목 | 내용 | 담당 |
|-----------|------|------|
| 본사 명의 출원·등록 완료 | 해당 국가에 본사 명의로 출원이 완료됐는지 확인 | 국내 IP팀(R), IP 총괄(A) |
| 파트너 법인 명의 선등록 검색 | 파트너 법인명으로 동일·유사 표장 출원·등록 여부 검색 | 국내 IP팀(R), 현지 대리인(C) |
| 파트너 대표자 개인 명의 선등록 검색 | 대표자 개인명으로 동일·유사 표장 출원·등록 여부 검색 | 국내 IP팀(R), 현지 대리인(C) |
| 파트너 관계 법인 검색 | 자회사·특수관계인 명의 검색 | 국내 IP팀(R), 현지 대리인(C) |

> **담당**: 국내 IP팀(R), IP 총괄(A), 현지 대리인(C), 영업팀(I)
> **기한**: 파트너 계약 체결 최소 60일 전

---

### Step 2. 계약서 상표 조항 설계 (계약 체결 D-30 이전)

아래 필수 조항을 계약서에 포함한다. 조항은 현지 대리인 검토 후 해당 국가 법률 기준에 맞게 조정한다.

| 필수 조항 | 핵심 내용 | 국가별 특이사항 |
|-----------|-----------|----------------|
| **상표 소유권 귀속** | 상표 소유권은 본사에 귀속되며 파트너는 출원·등록 권리가 없다 | 전 국가 필수 |
| **사용 허락 범위** | 허락된 지역·상품류·채널·기간을 명시 | 전 국가 필수 |
| **사용 방법 기준** | 표시 방법(크기·색상·위치)은 본사 Brand Manual에 따른다 | 전 국가 필수 |
| **재라이선스 금지** | 파트너는 제3자에게 상표 사용을 재허락할 수 없다 | 전 국가 필수 |
| **출원·등록 금지** | 파트너는 본사 표장과 동일·유사한 표장을 어느 국가에도 출원·등록할 수 없다 | 전 국가 필수 |
| **변형 사용 금지** | 파트너는 표장을 변형·축약·번역해 사용할 수 없다 | 전 국가 필수 |
| **품질 통제** | 파트너는 본사가 정한 품질 기준을 유지하며, 본사의 품질 감사를 수용한다 | 전 국가 필수 |
| **증거 협조 의무** | 파트너는 본사의 불사용 취소 대응을 위한 사용 증거 제공에 협조한다 | 전 국가 필수 |
| **계약 종료 후 사용 금지** | 계약 종료 후 즉시 상표 사용을 중단하며, 재고 처리 기간을 명시 | 전 국가 필수 |
| **경업 금지** | 계약 종료 후 일정 기간·지역 내 유사 브랜드 운영을 금지 | 브라질·멕시코·아르헨티나 현지법 검토 |
| **분쟁 해결** | 준거법 및 중재 조항 명시 | 현지 강행법규 확인 필수 |

> **담당**: 국내 IP팀(R), IP 총괄(A), 현지 대리인(R), 법무팀(C), 영업팀(C)

## 9. 필수 계약 조항 템플릿

아래는 핵심 상표 관련 조항의 표준 문안이다. 현지 대리인 검토 후 계약서에 통합한다.

### 조항 1: 상표 소유권 귀속

> "본 계약에 따라 허락된 상표(이하 "허락 상표")의 소유권은 [본사명](이하 "허락자")에 단독으로 귀속된다. [파트너명](이하 "수허락자")은 본 계약에 따른 사용 허락 외에 허락 상표에 대한 어떠한 권리도 취득하지 아니하며, 허락 상표와 동일하거나 유사한 상표를 어느 국가에서도 직접 또는 제3자를 통해 출원·등록하지 아니한다."

### 조항 2: 사용 허락 범위

> "허락자는 수허락자에게 [지역], [상품류], [채널], [기간] 범위에 한해 허락 상표의 비독점적[또는 독점적] 사용을 허락한다. 수허락자는 이 범위를 초과하여 허락 상표를 사용하지 아니한다."

### 조항 3: 재라이선스 및 변형 금지

> "수허락자는 허락자의 사전 서면 동의 없이 허락 상표의 사용권을 제3자에게 재부여하거나 양도하지 아니한다. 수허락자는 허락 상표를 변형·축약·번역하거나 다른 상표와 결합하여 사용하지 아니한다."

### 조항 4: 증거 협조 의무

> "수허락자는 허락자의 요청이 있는 경우, 허락 상표 사용에 관한 판매 송장, 광고 자료, 판매 실적 보고서 등의 자료를 30일 이내에 제공한다. 수허락자는 허락자가 불사용 취소 또는 상표 분쟁에 대응하는 데 필요한 증거 수집에 협조한다."

### 조항 5: 계약 종료 후 처리

> "본 계약 종료 후 수허락자는 즉시 허락 상표의 사용을 중단한다. 단, 기존 재고의 처분을 위해 [○개월]의 유예 기간을 부여하며, 유예 기간 중 사용은 허락자가 서면으로 확인한 재고에 한정한다. 수허락자는 계약 종료일로부터 [○일] 이내에 허락 상표가 포함된 모든 디지털 자산(도메인·소셜 계정·플랫폼 계정 포함)을 허락자에게 이전하거나 삭제한다."

## 10. 실패 사례 및 주의사항

### 실패 사례 1: 멕시코 딜러 선출원 — 계약서 조항이 있어도 사전 출원이 없었다

**상황**: 한국 소비재 기업 A사가 멕시코 유통 딜러와 계약 체결 시 상표 출원 금지 조항을 계약서에 포함했다. 그러나 본사 명의의 멕시코 IMPI 출원이 완료되지 않은 상태였다. 계약 체결 6개월 후, 딜러가 계약 체결 이전 시점에 이미 유사 표장을 IMPI에 출원했다는 사실이 검색을 통해 확인됐다.

**원인**: 파트너 계약 전 본사 출원이 완료되지 않았다. 파트너 명의 선등록 여부 검색을 계약 체결 전에 실시하지 않았다.

**결과**: 무효 절차 진행 중 딜러와의 협상이 장기화됐다. 딜러가 상표권을 '볼모'로 추가 보상을 요구하는 협상 구조에 놓였다. 약 18개월간 멕시코 사업 불안정.

**교훈**: 파트너 계약 전 본사 명의 출원 완료가 선행조건이다. 계약서 조항만으로 사전 등록을 대체할 수 없다.

---

### 실패 사례 2: 브라질 유통사 — 라이선스 계약 없이 사용, 불사용 취소 대응 실패

**상황**: 한국 화장품 기업 B사가 브라질 유통사를 통해 3년간 제품을 판매했다. 유통 계약서에는 상표 사용 조항이 없었으며, 별도 라이선스 계약도 체결하지 않았다. 전 경쟁사가 브라질 INPI에 B사 상표의 불사용 취소를 청구했고, B사의 유통사 판매가 "권리자의 사용"으로 인정받지 못한다는 주장을 제기했다.

**원인**: 유통 계약에 상표 라이선스 구조가 설계되지 않았다. INPI 라이선스 등록이 없었다.

**결과**: INPI 절차에서 취소 청구 일부 인용. B사 일부 류의 등록 소멸. 신규 출원으로 권리 재건 필요.

**교훈**: 유통사가 상표를 사용하는 경우 반드시 라이선스 계약을 체결하고 상표청에 등록한다. "유통 계약 = 상표 사용 허락"은 성립하지 않는다.

---

### 실패 사례 3: 아르헨티나 파트너 — 계약 종료 후 유사 브랜드 창업

**상황**: 한국 식품 기업 C사가 아르헨티나 유통 파트너와 5년 계약 종료 후, 파트너가 C사 브랜드와 유사한 표장으로 독자 사업을 시작했다. 계약서에 경업 금지 조항이 없었고, 상표 출원 금지 조항도 포함되지 않았다. 전 파트너가 유사 표장을 INPI-AR에 출원했다.

**원인**: 계약 종료 후 파트너의 경업을 제한하는 조항이 없었다. 계약 종료 전 파트너 명의 출원 모니터링이 없었다.

**결과**: 이의신청 및 무효 절차 동시 진행. 아르헨티나 시장에서 소비자 혼동 발생. 절차 완료까지 약 2년 소요.

**교훈**: 계약서에 경업 금지 조항과 계약 종료 후 상표 출원 금지 조항을 반드시 포함한다. 계약 종료 후 6개월간 파트너 명의 상표 출원 모니터링을 지속한다.

---

### 실패 사례 4: 콜롬비아 전 대리점 — 사용선언 없이 대리점 명의로 실질 운영

**상황**: 한국 전자기업 D사가 콜롬비아 대리점과 계약 해지 후 SIC에 직접 상표를 출원했다. 전 대리점이 이의신청을 제기하며, 자신이 D사 표장 아래 장기 사업을 운영해온 사실을 이유로 선사용권을 주장했다. 계약서에 상표 소유권 귀속 조항이 없었다.

**원인**: 계약서에 상표 소유권 조항이 없어 전 대리점의 "사실상 사용자" 주장이 설득력을 가졌다.

**결과**: SIC 이의 절차에서 D사 출원이 심사 지연. 대리점 선사용 주장에 반박하는 계약 이력과 증거 부재로 어려움 발생.

**교훈**: 계약서에 상표 소유권 귀속 조항을 반드시 포함한다. 대리점이 상표 아래 사업을 운영하더라도 소유권은 본사에 있음을 명문화해야 한다.

---

### 실패 사례 5: 파나마 허브 경로 — 계약 없는 재수출로 위조 상품 혼입

**상황**: 한국 소비재 기업 E사가 파나마 유통사와 계약을 체결했으나 재수출 경로 통제 조항을 포함하지 않았다. 파나마 유통사가 콜롬비아·에콰도르 하위 딜러에게 제품을 공급하면서 위조품을 혼입해 유통했다. E사는 재수출 경로를 추적하지 못해 위조품 유통 사실을 늦게 발견했다.

**원인**: 파나마 허브 기능에 따른 재수출 경로 통제가 계약에 없었다. 서브 딜러 구조 파악 체계도 없었다.

**결과**: 콜롬비아·에콰도르 시장에서 위조품 혼입으로 브랜드 평판 훼손. 3개국에서 동시 대응이 필요해진 상황.

**교훈**: 파나마 계약에는 재수출 범위 제한, 서브 딜러 사전 승인, 허브 경로 추적 의무를 명시한다.
