# 제06장. Examination / Opposition: 공고·이의·거절 대응

> **이 장의 목적**: 출원 후 심사·공고·이의·거절 단계에서 발생하는 모든 절차에 즉시 대응한다.
> 이의신청 기간을 놓치거나 거절 이유에 적시 대응하지 못하면 권리화 기회를 영구히 잃는다.
> 05장(출원 실무) 완료 후 이 장을 진행하고, 등록 결정 후 07장(등록 후 유지관리)으로 이동한다.

---

## 1. 핵심 원칙

> **"이의 단계에서 승부가 결정된다. 공고일을 모르면 권리를 잃는다."**

중남미에서 출원 후 가장 많은 권리 손실이 발생하는 단계는 심사·공고·이의 단계다.
이 단계의 실패는 세 가지 형태로 나타난다.

첫째, 공보 모니터링을 하지 않아 제3자의 유사 표장 공고를 발견하지 못한다.
이의신청 기간이 경과하면 해당 표장은 등록되고, 이후 무효 절차만 남는다.

둘째, 자사 출원에 대한 거절 이유통지를 받고도 대응 기한을 놓친다.
중남미 국가들은 출원인에게 별도 알림을 보내지 않는 경우가 많다. 대리인에게만 통보가 가고 내부 보고가 지연되면 기한을 그대로 소멸시킨다.

셋째, 이의 또는 거절에 직면했을 때 "방어할지, 협상할지, 포기할지"를 결정하지 못해 기회비용이 발생한다.

이 단계는 **공격(타사 이의신청)**과 **방어(자사 출원 대응)** 두 축으로 동시에 운영한다.

---

## 2. 국가 범위 및 적용 조건

이 장의 절차는 출원이 진행 중인 모든 국가에 적용된다. 국가별로 공고 방식, 이의 기간, 거절 대응 기한이 상이하므로 반드시 국가별로 별도 캘린더를 운영한다.

| 국가 | 상표청 | 공보 매체 | 이의신청 기간 | 거절 대응 기한 |
|------|--------|-----------|--------------|----------------|
| 브라질 | INPI | Revista da Propriedade Industrial (RPI) | 공고일로부터 60일 | 거절통지일로부터 60일 |
| 멕시코 | IMPI | Gaceta de la Propiedad Industrial | 공고일로부터 30일 | 거절통지일로부터 2개월 |
| 콜롬비아 | SIC | Gaceta de Propiedad Industrial | 공고일로부터 30일 | 거절통지일로부터 30일 |
| 아르헨티나 | INPI | Boletín de Marcas | 공고일로부터 30일 | 거절통지일로부터 30일 |
| 칠레 | INAPI | Diario Oficial | 공고일로부터 30일 | 거절통지일로부터 30일 |
| 페루 | INDECOPI | Gaceta Oficial | 공고일로부터 30일 | 거절통지일로부터 30일 |
| 에콰도르 | SENADI | Gaceta de la Propiedad Intelectual | 공고일로부터 30일 | 거절통지일로부터 30일 |
| 파나마 | DIGERPI | Gaceta Oficial Digital | 공고일로부터 30일 | 거절통지일로부터 30일 |

> 브라질은 이의 기간이 60일로 다른 국가보다 길지만, 그만큼 상대방도 이의신청을 준비할 시간이 많다는 점을 유의한다.

### 3-1. 공보 모니터링 누락 리스크

| 리스크 유형 | 설명 | 영향 국가 |

| **공보 미수령** | 각국 공보를 정기 구독하지 않아 유사 표장 공고를 발견하지 못함 | 전 국가 |
| **온라인 공보 접근 어려움** | 브라질 RPI는 유료 구독 또는 INPI 직접 접속 필요 | 브라질 |
| **외국어 공보 해석 지연** | 스페인어·포르투갈어 공보를 내부에서 직접 확인하기 어려워 대리인에게 의존 | 전 국가 |
| **공고일 기준 오인** | 공보 발행일과 온라인 게재일이 다를 경우 이의 기간 계산 오류 발생 | 콜롬비아, 에콰도르 |

**실행 지침**: 공보 모니터링은 출원 접수일부터 현지 대리인에게 의뢰한다. 모니터링 범위(표장 유형, 상품류)를 서면으로 명시한 위임 범위에 포함시킨다.

|-------------|------|------|
| **대리인 통보 지연** | 현지 대리인이 거절통지를 수령하고도 국내 IP팀에 지연 보고 | 기한 내 대응 서류 준비 불가 |
| **내부 의사결정 지연** | 거절 이유 분석 후 "대응 vs 포기" 결정이 늦어져 기한 도과 | 권리 소멸 |

| **증거 준비 지연** | 거절 이유가 선행 상표와의 유사성인 경우 공존합의서·구별 입증이 기한 내 어려움 | 브라질, 멕시코 |

**실행 지침**: 거절통지 수령일로부터 48시간 이내에 국내 IP팀이 현지 대리인으로부터 내용을 전달받는 보고 프로세스를 수립한다. 연락 체계를 대리인 계약서에 명시한다.

## 4. Decision Box

### 4-1. 이의신청 전략 선택

```
[Decision Box A: 타사 공고에 대한 이의신청 여부]

IF (공고된 표장이 자사 출원 표장과 동일 + 동일 류)
    → 즉시 이의신청 준비 개시
      출원번호·공고일·이의 기한을 캘린더에 등록하고 현지 대리인에게 의뢰

IF (공고된 표장이 자사 표장과 유사 + 동일 류 또는 인접 류)
    → 현지 대리인에게 충돌 리스크 평가 의뢰 (3영업일 이내)
      리스크 "높음" → 이의신청 진행
      리스크 "중간" → 이의 기한 내 재평가 후 결정
      리스크 "낮음" → 이의신청 보류, 지속 모니터링

IF (이의신청인이 자사 파트너·대리점 또는 전 파트너)
    → 계약 위반 여부를 09장 계약 구조 체크리스트로 확인
      계약 위반 입증 가능 시 이의신청 + 계약 이행 청구 병행

IF (선행 상표가 불사용 상태로 추정되는 경우)
    → 이의신청과 별도로 불사용 취소 청구 병행 검토 (14장 참조)
      불사용 취소가 인용되면 이의신청 없이도 출원 등록 가능

IF (이의신청 비용이 예산을 초과하거나 해당 시장 가치가 낮은 경우)
    → 시장 가치 × 분쟁 비용 매트릭스로 이의신청 ROI 계산 후 결정
      포기 결정 시 해당 국가 출원 철회 및 대안 표장 검토
```

### 4-2. 자사 출원 거절 대응 전략

```
[Decision Box B: 거절 이유별 대응 방향]

IF (거절 이유: 선행 상표와 외관·호칭·관념 유사)
    → 구별 가능성 입증 서류 준비 (사용 실적, 소비자 혼동 없음 증거)
      이의신청인 또는 선행 상표권자에게 공존합의서 협상 타진
      협상 불가 시 이의 절차로 방어 집중

IF (거절 이유: 지정 상품·서비스 범위 문제)
    → 보정서 제출로 범위 축소 또는 재설정
      핵심 상품류가 제외되지 않도록 보정 범위 사전 검토

IF (거절 이유: 식별력 부족 / 기술적 표장)
    → 사용에 의한 식별력 취득 증거 제출 (매출, 광고, 소비자 인지도 조사)
      증거 불충분 시 표장 변경 후 재출원 검토

IF (거절 이유: 저명 상표와 충돌 주장)
    → 저명성 입증 수준을 현지 기준으로 재평가
      저명성이 낮은 외국 상표의 경우 저명성 불인정 주장 적극 전개
      자사가 저명 상표인 경우 저명성 증거 자료 집중 준비

IF (이의신청인과 협상 가능성이 있는 경우)
    → 공존합의서 또는 상표 양도 협상 개시
      협상 조건: 지역 분리, 류 분리, 공존 사용 조건 등 가능
      협상 타결 시 이의신청 취하 조건으로 진행

IF (대응 가능성이 낮고 시장 중요도도 낮은 경우)
    → 해당 국가 출원 포기 + 대안 표장으로 재출원 검토
      재출원 시 02장(포트폴리오 설계)부터 재개
```

### 4-3. 대응 방식 선택 (방어 vs 협상 vs 포기)

```
[Decision Box C: 이의·거절 상황의 전략적 판단]

IF (해당 국가가 브라질 또는 멕시코 → 핵심 시장)
    → 가능한 모든 대응 수단 동원 (방어 최우선)
      비용보다 권리화 여부가 우선 판단 기준

IF (해당 국가가 안데스 권역인 경우)
    → 1개국 이의가 다른 안데스 국가 사건에도 시사점을 주는지 확인
      단, 이의 절차 자체는 국가별로 분리 대응 전략 설계

IF (상대방이 현지 딜러 또는 과거 파트너)
    → 법적 대응 전에 계약서 조항 검토로 협상 우위 확보
      상표 이전 요구 + 손해배상 청구 가능 여부 확인

IF (예산 제약으로 모든 국가 동시 대응 불가)
    → 브라질·멕시코 우선 방어, 나머지는 대리인과 비용 협의 후 순차 대응

IF (이의 또는 거절이 전술적 목적(협상 도구)으로 사용된 것으로 판단)
    → 초기 협상 제안 후 수락 없으면 전면 방어로 전환
      협상 제안 내용과 거절 이유를 모두 증거로 보관
```

## 5. 실행 절차

### Step 1. 출원 접수 후 모니터링 체계 즉시 가동 (출원 당일)

출원이 접수되면 당일 모니터링 체계를 설정한다.

1. 현지 대리인에게 해당 국가 공보 모니터링 의뢰를 서면으로 발송한다.
2. 모니터링 범위를 명시한다: 보호 표장명, 유사 범위(외관·호칭·관념), 지정 상품류 + 인접 류.
3. 보고 주기를 계약서 또는 지시서에 명시한다: 브라질·멕시코는 주 1회, 기타 국가는 격주.
4. 내부 캘린더에 각국 공보 발행일·검토일을 반복 등록한다.

**담당**: 국내 IP팀(R), 현지 대리인(R), IP 총괄(A)

---

### Step 2. 공보 확인 및 유사 표장 검토 (공보 발행 후 3영업일 이내)

공보 발행 후 현지 대리인은 유사 표장 현황을 보고한다.

1. 현지 대리인으로부터 공보 스캔 또는 요약 보고서를 수령한다.
2. 자사 표장과의 충돌 가능성을 현지 대리인이 3영업일 이내 평가한다.
3. 충돌 가능성 있음 → Decision Box A 실행.
4. 충돌 없음 → 기록 후 다음 공보 사이클 대기.

**담당**: 현지 대리인(R), 국내 IP팀(R), IP 총괄(A)

---

### Step 3. 이의신청 결정 및 준비 (이의 기한 -21일 이전)

이의신청 결정이 내려지면 즉시 준비를 시작한다.

1. 현지 대리인이 이의신청서 초안을 작성한다.
2. 국내 IP팀이 이의 근거 증거(선등록 상표, 사용 실적, 저명성 자료)를 준비한다.
3. 스페인어 또는 포르투갈어로 공증이 필요한 서류는 공증·아포스티유 절차를 즉시 개시한다.
4. 이의신청서 최종본을 이의 기한 -7일 이전에 확정하고 제출 준비를 완료한다.

**담당**: 국내 IP팀(R), 현지 대리인(R), IP 총괄(A)

| 준비 서류 | 공증·번역 여부 | 준비 주체 |
|-----------|----------------|-----------|
| 이의신청서 | 현지 대리인 작성 | 현지 대리인 |
| 선등록 증명서 | 원본 또는 공식 출력본 | 국내 IP팀 + 현지 대리인 |
| 사용 실적 증거 | 스페인어·포르투갈어 번역 필요 | 국내 IP팀 |
| 저명성 입증 자료 | 공증 또는 아포스티유 필요 (국가별 상이) | 국내 IP팀 + 법무팀 |
| 위임장 | 공증·아포스티유 (브라질·콜롬비아 필수) | 법무팀 |

---

### Step 4. 거절 이유 대응 (거절통지 수령 후 48시간 이내 보고 체계 가동)

거절통지가 발생하면 다음 절차를 즉시 실행한다.

1. 현지 대리인이 거절통지를 수령하면 48시간 이내에 국내 IP팀에 번역·요약 보고서를 송부한다.
2. 국내 IP팀은 Decision Box B를 실행하여 대응 방향을 결정한다.
3. 대응 방향 결정 후 현지 대리인과 전략 협의를 진행한다 (수령 후 5영업일 이내).
4. 의견서·보정서 작성에 필요한 증거를 즉시 수집한다.
5. 최종 의견서를 거절 대응 기한 -7일 이전에 현지 대리인에게 전달하고 제출을 확인한다.

**담당**: 현지 대리인(R), 국내 IP팀(R), IP 총괄(A), 법무팀(C)

---

### Step 5. 협상 진행 (이의신청인 또는 거절 관련 선행권자와의 협상)

협상이 필요한 경우 절차를 별도로 운영한다.

1. 상대방(이의신청인 또는 선행 상표권자) 정보를 현지 대리인에게 조회한다.
2. 접촉 방식을 결정한다: 현지 대리인 통한 서면 접촉 우선.
3. 협상 제안 내용을 내부에서 먼저 확정한다: 공존 조건, 지역·류 분리, 합의금 상한.
4. 협상이 타결되면 공존합의서를 작성하고 해당 국가 상표청에 제출한다.
5. 협상 내용 전부를 증거 저장소에 보관한다 (향후 유사 분쟁 대비).

**담당**: 국내 IP팀(R), 현지 대리인(R), IP 총괄(A), 법무팀(C), 경영진(I)

---

### Step 6. 결과 등록 및 후속 조치 (결정 후 5영업일 이내)

이의·거절 결과가 나오면 즉시 포트폴리오 관리 대장을 업데이트한다.

1. 이의 기각 / 거절 극복 → 등록 결정으로 진행, 07장으로 이동한다.
2. 이의 인용 / 최종 거절 → 대안 표장 검토 또는 불복 심판 여부를 결정한다.
3. 결과와 대응 이력 전체를 증거 저장소에 저장한다.
4. 동일 상황이 다른 국가에서 재현될 가능성이 있으면 전 국가 대리인에게 사전 안내한다.

**담당**: 국내 IP팀(R), IP 총괄(A), 현지 대리인(I)

- [ ] 이의·거절 결과를 포트폴리오 관리 대장에 업데이트했다
- [ ] 결과에 따른 후속 조치(등록 진행 또는 재출원 검토)를 결정했다

## 9. 실패 사례 및 주의사항

### 실패 사례 1: 브라질 공보 모니터링 미실시로 유사 표장 등록 허용

**상황**: 한국 소비재 기업 A사가 브라질 INPI에 출원 후 현지 대리인에게 별도 모니터링 지시를 하지 않았다. 출원 후 14개월 뒤, 자사 표장과 유사한 브라질 현지 기업의 표장이 RPI를 통해 공고되었으나 발견하지 못해 이의신청 기간(60일)을 놓쳤다. 해당 표장이 등록된 후 무효 절차를 진행했으나 3년이 소요되었다.

**원인**: 공보 모니터링을 별도 업무로 설정하지 않고 대리인에게 암묵적으로 의존했다.

**재발 방지**: 모니터링 범위와 주기를 위임장 또는 별도 서면에 명시한다. 현지 대리인의 모니터링 의무를 계약서에 포함한다.

---

### 실패 사례 2: 멕시코 거절 대응 기한 도과

**상황**: 한국 화장품 기업 B사의 멕시코 출원에 선행 상표와의 유사성을 이유로 거절통지가 발행되었다. 현지 대리인이 거절통지를 수령했으나 B사 국내 담당자에게 2주 후에야 보고했다. 보고 후 대응 방향 결정에 또 2주가 소요되어 2개월 기한을 4일 초과했다. 출원이 거절 확정되어 재출원 비용과 출원일 우선권을 모두 잃었다.

**원인**: 거절통지 수령 후 내부 보고 체계가 없었다. 대리인 보고 지연을 감지할 수단이 없었다.

**재발 방지**: 거절통지 수령 후 48시간 이내 국내 IP팀 보고를 대리인과 서면으로 합의한다. 내부 결정 기한을 수령 후 7영업일 이내로 설정한다.

---

### 실패 사례 3: 콜롬비아 전 파트너 이의신청 대응 실패

**상황**: 한국 식품 기업 C사가 콜롬비아 유통 파트너와 계약 종료 후 자사 표장을 SIC에 출원했다. 전 파트너가 이의신청을 제기했고, 이의 이유는 자신이 사용해온 동일 표장에 대한 선사용권 주장이었다. 계약서에 상표 소유 조항이 없어 전 파트너의 선사용 증거가 이의 절차에서 유효하게 인정되었다. 출원이 거절되었다.

**원인**: 계약 체결 시 상표 소유 조항을 포함하지 않았다. 진출 전 C사 명의의 선출원도 없었다.

**재발 방지**: 유통 파트너 계약 체결 전 반드시 자사 명의로 해당 국가에 선출원한다(09장 참조). 계약서에 "상표 소유권은 본사에 귀속되며, 파트너는 상표 출원 권리가 없다"는 조항을 명시한다.

### 실패 사례 4: 안데스 3국 병행 출원에서 페루 이의를 놓쳤다

**상황**: 한국 IT 기업 D사가 콜롬비아·페루·에콰도르 3개국에 병행 출원했다. 페루에서 현지 기업이 이의를 신청했지만, 페루 공보를 별도로 모니터링하지 않아 답변을 제출하지 못했다.

**원인**: 안데스 권역을 하나의 묶음으로만 보고, 실제 공고와 대응은 국별로 움직인다는 점을 놓쳤다.

**재발 방지**: 안데스 3국은 권역 단위 전략으로 보되, 공고 모니터링과 이의 대응은 반드시 국가별로 분리 운영한다.
