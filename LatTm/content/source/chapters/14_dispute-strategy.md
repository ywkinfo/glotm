# 제14장. 분쟁 전략: 무효·취소·소송

## 핵심 원칙: 행정과 사법을 병행하라

중남미 브랜드 분쟁은 단순히 "소송을 할 것인가"의 문제가 아니다. 실제로는 무효, 취소, 불사용 취소, 이의, 침해금지, 세관 연계, 부정경쟁 주장, 형사 대응 가능성까지 복수 수단이 병행된다.  
한국기업이 실패하는 대표 패턴은 상대방 등록을 방치하거나, 반대로 소송부터 검토해 시간과 비용을 과도하게 사용하는 것이다. 실무적으로는 먼저 상대 권리의 약점을 분석하고, 행정절차와 협상카드를 설계한 뒤, 필요한 경우에만 소송으로 확장하는 구조가 효율적이다.

## 주요 리스크

- 상대방이 먼저 출원 또는 등록한 상표를 장기간 방치해 시장 진입 자체가 지연될 수 있다.
- 불사용 취소 요건을 확인하지 않고 침해소송만 추진하면 승산이 낮아질 수 있다.
- 국가별 절차 차이로 인해 동일 전략을 반복 적용하면 실패 확률이 높다.
- 현지 사용증거, 번역문, 공증, 위임장 요건을 늦게 준비해 대응 시점을 놓칠 수 있다.
- 행정절차와 협상 타이밍이 맞지 않으면 불필요하게 분쟁이 장기화된다.
- 본사와 현지 유통사가 분쟁 목표를 다르게 설정해 메시지가 엇갈릴 수 있다.

## Decision Box: 취소 vs 무효 vs 침해소송

| 상황 | 우선 수단 | 이유 |
|---|---|---|
| 제3자가 악의적으로 선출원하여 등록을 확보한 경우 | 무효 또는 악의 출원 공격 | 권리의 기초 자체를 붕괴시키는 것이 우선 |
| 상대 등록은 있으나 실제 사용 흔적이 약한 경우 | 불사용 취소 | 비교적 비용 효율적으로 장애물을 제거 가능 |
| 상대가 현재 온라인/오프라인에서 적극 판매 중인 경우 | 행정단속 + 침해금지 + 협상 병행 | 피해 확대를 즉시 억제해야 함 |
| 계정/도메인/광고와 결합된 복합 침해인 경우 | 플랫폼 조치 + 행정절차 + 민사 검토 | 단일 수단으로는 실익이 작음 |
| 장기 유통 파트너와의 충돌인 경우 | 계약 위반 주장 + 상표분쟁 병행 | 상표만으로 풀리지 않는 경우가 많음 |

### 판단 순서

1. 상대 권리가 출원인지 등록인지 확인한다.
2. 상대 사용 실적과 시장 노출을 조사한다.
3. 당사 사용증거와 우선권 근거를 정리한다.
4. 사업상 목표가 "등록 제거"인지 "판매 중단"인지 "협상 레버리지 확보"인지 정한다.
5. 국가별 절차와 기간을 비교해 가장 빠른 압박 루트를 선택한다.

## 분쟁 수단 매트릭스

| 수단 | 목적 | 장점 | 한계 | 적합한 상황 |
|---|---|---|---|---|
| 무효심판/무효소송 | 부적법 등록 제거 | 선출원 문제를 정면 해결 | 시간 소요 가능 | 악의 출원, 식별력 문제, 절대적/상대적 사유 존재 |
| 불사용 취소 | 미사용 등록 제거 | 비용 대비 효율적 | 사용증거 제출 시 방어 가능 | 장기 미사용 의심 등록 |
| 침해 행정조치 | 판매 중단, 압수, 경고 | 비교적 신속한 경우가 많음 | 손해배상 범위는 제한적일 수 있음 | 오프라인 유통, 시장 단속 필요 |
| 민사소송 | 금지명령, 손해배상 | 강한 압박 수단 | 비용과 기간 부담 | 실질 피해가 크고 증거가 충분 |
| 협상/공존합의 | 빠른 시장 진입 또는 비용 절감 | 불확실성 축소 | 통제 실패 시 후속 분쟁 가능 | 상호 공존 여지가 있는 경우 |

## 실행 절차

### 1. 사건 정의

분쟁은 감정이 아니라 목표로 정의해야 한다.

- 우리 목표가 등록 제거인지 판매 중단인지 확인
- 핵심 국가와 부차 국가를 구분
- 온라인 침해인지 유통망 분쟁인지 유형 정의
- 상대방이 실제 영업 주체인지 명의 대여자인지 확인

### 2. 권리 현황 맵 작성

필수 확인 항목:

- 당사 출원/등록 현황
- 상대 출원/등록 현황
- 각국 클래스 범위 차이
- 사용 개시 시점
- 계약, 대리점, NDA, 라이선스 존재 여부
- 도메인, 계정, 회사명, 상호 등록 여부

이 단계에서 많은 사건은 "상표 분쟁처럼 보이지만 계약 분쟁이 핵심"이라는 점이 드러난다.

### 3. 증거 전략 수립

분쟁 절차에 맞는 증거를 미리 구조화해야 한다.

| 증거 유형 | 활용 절차 | 예시 |
|---|---|---|
| 상표권 서류 | 무효, 침해, 플랫폼 신고 | 등록증, 출원정보, 갱신기록 |
| 사용증거 | 불사용 취소 방어, 침해 주장 | 인보이스, 광고물, 판매자료, 카탈로그 |
| 악의 입증 자료 | 무효, 협상 압박 | 대리점 관계, 이메일, 사전 미팅 기록 |
| 시장 혼동 자료 | 침해소송, 행정단속 | 소비자 문의, 오배송, 리뷰 혼선 |
| 침해 증거 | 행정, 민사, 플랫폼 | 제품 사진, 테스트 구매, 송장, 가격표 |

### 4. 절차 조합 설계

실무적으로는 하나의 절차만 쓰지 않는다.

예시 조합:

- 상대 등록 공격: 무효 또는 불사용 취소 + 협상 제안
- 판매 억제: 경고장 + 행정단속 + 플랫폼 제거
- 고의 침해 압박: 플랫폼 대응 + 세관 등록 + 민사 검토
- 파트너 충돌: 계약 종료 통지 + 계정 회수 + 상표권 집행

### 5. 협상 창구 단일화

분쟁이 길어질수록 본사, 현지 대리점, 외부 로펌, 유통사가 서로 다른 메시지를 보내기 쉽다.

- 협상 승인권자를 1명으로 정한다.
- 합의 허용 범위를 문서화한다.
- 공존합의 가능 시 금지 조항, 영역 제한, 온라인 판매 제한을 명시한다.
- 부분 합의 후 재침해 시 자동 집행 구조를 검토한다.

### 6. 소송 전 최종 점검

소송 또는 본격 행정절차에 들어가기 전 반드시 확인할 항목:

- 비용 대비 실익이 충분한가
- 상대방이 집행 가능한 자산이나 영업 기반을 가지고 있는가
- 사용증거와 침해증거가 국가별 요건을 충족하는가
- 임시조치 또는 가처분 필요성이 있는가
- 사업팀이 공급, 유통, 가격 정책까지 맞춰둘 준비가 되었는가

## 체크리스트

### 사건 진단 체크

- [ ] 상대가 출원인인지 등록권자인지 확인했는가
- [ ] 상대방 상표의 실제 사용 여부를 조사했는가
- [ ] 우리 브랜드의 현지 사용증거가 정리되어 있는가
- [ ] 침해 유형이 위조, 병행수입, 대리점 분쟁 중 무엇인지 구분했는가

### 절차 선택 체크

- [ ] 무효와 취소 중 어떤 수단이 더 빠른지 비교했는가
- [ ] 행정조치와 민사소송을 병행할 필요가 있는가
- [ ] 플랫폼 제거로 충분한지, 오프라인 집행까지 가야 하는지 판단했는가
- [ ] 경고장 발송이 오히려 선제 대응을 유발할 위험은 없는가

### 증거 체크

- [ ] 인보이스, 수입자료, 광고자료, 카탈로그를 확보했는가
- [ ] 상대방 판매증거와 유통경로를 확인했는가
- [ ] 악의 선출원 정황을 보여줄 이메일이나 계약 자료가 있는가
- [ ] 번역, 공증, 위임장 요건을 사전에 점검했는가

### 운영 체크

- [ ] 본사와 현지법인, 외부대리인의 RACI가 정의되어 있는가
- [ ] 합의 가능 범위와 레드라인이 정리되어 있는가
- [ ] 분쟁 종료 후 재침해 모니터링 계획이 있는가
- [ ] 사건 결과를 포트폴리오 전략에 반영하는가

## 국가별 분쟁 구조 포인트

| 국가 | 실무 포인트 | 전략 시사점 |
|---|---|---|
| 브라질 | 절차와 문서 준비의 정합성이 중요하고 포르투갈어 대응 품질이 결과에 영향 | 초기 서류 완성도를 높이고 현지 대리인 조율이 핵심 |
| 멕시코 | 온라인 유통과 행정 집행을 함께 봐야 하는 경우가 많음 | 플랫폼 대응과 행정 전략을 분리하지 말 것 |
| 콜롬비아 | 안데스권 맥락에서 권리 정합성과 증거 준비가 중요 | 안데스권 타국 전략과 비교 설계가 유효 |
| 아르헨티나 | 경제 상황에 따라 비공식 유통 이슈가 분쟁과 결합 | 침해 분쟁과 유통 통제를 함께 다뤄야 함 |
| 칠레 | 비교적 구조화된 대응이 가능하나 선제 준비가 중요 | 빠른 대응이 비용 효율을 높임 |
| 페루 | 사용증거와 실제 시장 자료의 중요성이 큼 | 취소/방어 전략에서 증거 관리가 관건 |
| 에콰도르 | 파트너 분쟁이 상표 문제로 비화하는 경우가 있음 | 계약 검토를 병행해야 함 |
| 파나마 | 물류와 재수출 이슈가 결합될 수 있음 | 상표 분쟁을 통관 통제와 연결해야 함 |

## 실무 시나리오별 권장 전략

### 시나리오 1. 현지 대리점이 브랜드를 먼저 등록한 경우

권장 조합:

- 계약관계와 사전 교섭 기록 확보
- 악의 선출원 주장을 검토
- 무효 절차와 협상 병행
- 계정, 도메인, 유통계약 통제권 회수 동시 추진

주의점:

- 감정적으로 경고장부터 보내면 상대가 증거를 정리할 시간을 줄 수 있다.

### 시나리오 2. 등록은 없지만 시장에서 유사표장을 적극 사용하는 경우

권장 조합:

- 즉시 출원
- 플랫폼 및 행정 경고 조치
- 시장 혼동 증거 확보
- 필요 시 민사 가처분 가능성 검토

주의점:

- 출원 전 공개 대응은 역으로 선출원을 유발할 수 있다.

### 시나리오 3. 상대 등록은 있으나 실제 사용하지 않는 경우

권장 조합:

- 불사용 취소 가능 시점 확인
- 사용 여부 조사
- 취소 절차 개시 전 협상 타진
- 취소와 병행해 우리 출원 포지션 확보

주의점:

- 취소 가능 시점과 증거 제출 기준을 국가별로 혼동하면 안 된다.

### 시나리오 4. 온라인 위조와 오프라인 도매 유통이 연결된 경우

권장 조합:

- 플랫폼 삭제
- 테스트 구매 및 물류 추적
- 세관 연계
- 행정단속 또는 민사 집행 검토

주의점:

- 온라인 증거만으로는 공급망 차단이 어렵다.

## 실패 사례

### 사례 1. 무효 사유보다 침해소송을 먼저 검토한 경우

상대가 먼저 등록한 표장이 있었음에도 판매 중단만 목표로 소송을 검토하다가, 정작 등록장애가 해소되지 않아 시장 진입이 지연되었다.

재발 방지:

- 등록 장애가 있으면 먼저 그 장애를 제거하는 절차를 검토한다.

### 사례 2. 불사용 취소 기회를 놓친 경우

상대 등록이 장기간 미사용 상태였지만 증거조사를 미루는 사이 상대가 사용 흔적을 급히 만들고 방어에 성공했다.

재발 방지:

- 경쟁표장 감시 시 미사용 의심 건은 별도 리스트로 관리한다.

### 사례 3. 현지 대리점과 상표 분쟁을 순수 IP 문제로만 본 경우

실제로는 계약 종료, 미수금, 재고 처분, SNS 계정 소유권이 모두 얽혀 있었지만 상표무효만 진행해 해결이 지연되었다.

재발 방지:

- 분쟁 시작 시 계약, 계정, 재고, 유통권을 한 화면에서 본다.

## 운영 KPI

- 분쟁 유형별 평균 종결 기간
- 무효/취소 성공률
- 협상 종결 비율
- 소송 전 단계에서 해결된 사건 비율
- 반복 침해 재발률
- 분쟁 대응 비용 대비 보호 시장 규모

## 실무 결론

중남미에서 분쟁 전략의 핵심은 "상대 권리를 약화시키고, 시장 피해를 즉시 줄이며, 협상력을 확보하는 것"이다.  
무효, 취소, 행정조치, 민사소송은 선택지가 아니라 조합 도구다. 한국기업은 특히 대리점 선출원, 미사용 등록, 온라인-오프라인 결합 침해에 취약하므로, 14장은 9장 계약 구조, 10장 모니터링, 11장 집행, 13장 온라인 대응과 하나의 운영 체계로 설계되어야 한다.
