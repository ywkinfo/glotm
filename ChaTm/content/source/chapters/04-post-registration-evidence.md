# 제4장. 등록 후 유지와 사용 증거

등록이 완료되면 상표 업무가 끝나는 것이 아니라, 이제부터 실제 사용과 권리 유지의 기록 체계를 갖춰야 한다.

### 사용 구조를 먼저 명확히 할 것

중국에서는 본사, 현지 법인, 제조 파트너, 유통사 가운데 누가 어떤 표장을 어떤 방식으로 사용하는지 흐려지기 쉽다. 이 구조가 흐리면 나중에 침해 대응과 불사용 이슈에서 방어가 어려워진다.

#### 증거 파일링의 기본 단위

- 상품 또는 서비스 화면
- 포장, 라벨, 인보이스
- 플랫폼 판매 페이지와 노출 시점
- 라이선스 또는 유통 계약상 상표 사용 주체

### 내부 운영 체크포인트

#### 분기별 확인

- 핵심 표장이 실제 판매 채널에 계속 노출되는지
- 중국어 표기가 등록 내용과 어긋나지 않는지
- 새로운 SKU가 기존 지정상품 범위를 벗어나지 않는지

#### 반기별 확인

- 경고장, 플랫폼 신고, 세관 협조 같은 집행 이력이 어디에 저장되는지
- 대리인 업데이트와 권리자 정보가 현재 조직 구조와 맞는지

### 최소 증거 보존 패키지

중국에서 실제 사용과 권리 유지 흐름을 방어하려면, 적어도 채널별 화면 캡처, 거래 문서, 상품 실물 자료, 사용 주체를 설명하는 내부 메모를 한 묶음으로 관리해야 한다. 핵심은 “증거가 있는가”보다 “누가 언제 어떤 표장을 어떤 상품에 썼는지 하나의 이야기로 연결되는가”다.

가능하면 분기 단위 폴더를 기준으로 정리하고, 각 폴더에 핵심 표장명, 사용 주체, 채널, 캡처 시점, 관련 계약 문서를 같이 넣어 두는 편이 이후 불사용 주장이나 분쟁 대응 때 가장 실용적이다.
