# 제2장. 출원 설계와 선행 검토

중국 출원은 서류 제출 자체보다 출원 전 설계의 정확도가 훨씬 중요하다. 지정상품 범위와 명칭 구조가 어긋나면 등록 이후 사용과 집행에서도 같은 문제가 반복된다. 그래서 이 장은 단순한 출원 절차 설명보다, 무엇을 먼저 판단하고 어떤 자료를 잠가야 하는지를 중심으로 읽는 편이 맞다.

## 출원 전 설계의 산출물은 신청서가 아니라 결정 메모다

좋은 선행 검토의 결과물은 검색 화면 모음이 아니다. 실제로 필요한 것은 다음 네 가지를 정리한 결정 메모다.

- 어떤 표장을 중국에서 먼저 선점할 것인가
- 영문, 한글, 중국어 표기를 각각 어떻게 가져갈 것인가
- 어떤 상품 범위를 코어로 잡고 어디까지 방어 범위로 둘 것인가
- 직접 출원과 국제출원 중 어느 경로가 현재 조직 구조에 맞는가

이 네 가지가 정리되지 않으면 출원은 들어가도 이후 심사, 공고, 사용, 집행 단계에서 같은 질문을 반복하게 된다. 중국은 선출원 구조, 상품 지정, 현지 표기 전략이 초기에 어긋나면 나중에 수정 비용이 더 커지기 쉽다.

### 검색의 목적

검색은 단순 중복 확인이 아니라 다음 세 가지를 같이 보기 위한 절차다.

- 동일 또는 유사한 선행표장의 존재
- 중국어 음역 또는 의역 충돌 가능성
- 실제 사업 계획과 지정상품 설계의 불일치

실무적으로는 결과를 go / revise / hold / abandon 중 하나의 방향으로 정리하는 편이 좋다. 같은 브랜드를 그대로 밀고 갈 수 있는지, 중국어 표기를 따로 조정해야 하는지, 상품 범위를 줄이면 살릴 수 있는지, 아니면 지금 접는 편이 전체 비용을 줄이는지를 한 장으로 정리해야 한다.

## 표장 구조는 영문 하나로 끝나지 않는다

중국에서는 영문 표장만 확보하면 충분하다고 보기 어렵다. 실제 소비자 인식, 검색 경로, 유통 현장에서는 중국어 표기와 영문 표기가 서로 다른 역할을 할 수 있기 때문이다. 그래서 출원 전부터 표장 구조를 적어도 세 층으로 나눠 보는 편이 좋다.

- 핵심 표장: 반드시 선점해야 하는 영문명, 로고, 결합표장
- 중국어 표기: 음역, 의역, 별칭 중 실제 시장에서 쓸 가능성이 높은 이름
- 방어 표장: 짧은 기간 안에 충돌 가능성이 높은 변형

### 중국어 표기를 설계할 때 먼저 볼 것

중국어 표기는 단순 번역 문제가 아니다. 소비자가 실제로 어떤 이름으로 부를지, 플랫폼 검색에서 어떤 표기가 먼저 걸릴지, 파트너와 대리점이 어떤 이름을 현장에서 쓸지를 같이 봐야 한다.

| 점검 항목 | 질문 |
| --- | --- |
| 음역 | 영문명을 소리대로 옮겼을 때 충돌 위험이 큰가 |
| 의역 | 뜻을 살린 현지명이 브랜드 정체성을 훼손하지 않는가 |
| 병행 사용 | 영문과 중국어를 동시에 쓸 때 포장과 판매 화면이 일관적인가 |
| 검색 경로 | 소비자가 실제로 어떤 표기로 검색할 가능성이 큰가 |

중요한 것은 완벽한 이름을 찾는 것이 아니라, 실제로 먼저 선점해야 하는 표기가 무엇인지 정하는 일이다. 늦게 정리할수록 브랜드와 유통 현장이 서로 다른 이름을 쓰기 시작한다.

## 지정상품과 서브클래스 리스크는 따로 본다

중국에서는 국제분류만 맞춘다고 끝나지 않는다. 실제 거래 구조와 충돌 가능성을 보려면 서브클래스 관점의 설계가 필요하다. 즉, 클래스 번호를 맞춘 뒤에도 실제 판매 SKU, 유통 채널, 핵심 제품군과 연결되는지를 한 번 더 확인해야 한다.

### 서브클래스 리스크를 읽는 방식

- 현재 실제로 판매하거나 곧 판매할 SKU를 먼저 적는다
- 그 SKU가 어느 지정상품 문구와 연결되는지 본다
- 같은 클래스 안에서도 충돌이 집중될 가능성이 큰 구간을 표시한다
- 방어 목적으로 넣고 싶은 항목이 실제 사용 계획과 얼마나 가까운지 따로 적는다

너무 넓게 잡으면 이후 사용과 집행에서 설명이 약해지고, 너무 좁게 잡으면 중국 시장 확장 단계에서 다시 비슷한 출원을 반복해야 한다. 그래서 코어 범위와 인접 확장 범위를 나눠 보는 편이 좋다.

| 범위 유형 | 의미 | 실무 판단 |
| --- | --- | --- |
| Core | 현재 판매 또는 즉시 출시 예정인 핵심 상품군 | 우선 확보 대상 |
| Adjacent | 12~24개월 안에 현실적인 확장 범위 | 사업 로드맵과 연결될 때만 포함 |
| Defensive | 당장 쓰지 않지만 충돌 방지 목적이 큰 범위 | 이유가 명확할 때만 제한적으로 검토 |

## 출원 주체와 사용 주체를 먼저 맞춘다

중국에서는 본사, 중국 법인, 제조 파트너, 유통사 가운데 누가 실제 사용자로 남는지가 흐려지기 쉽다. 이 구조가 모호하면 이후 심사 대응, 사용 증거, 집행 단계에서 같은 설명을 반복하게 된다. 따라서 출원 전에 적어도 다음은 잠가 두는 편이 좋다.

- 누가 owner인가
- 누가 실제 user인가
- 누가 패키지와 판매 화면을 통제하는가
- 누가 대리인과 커뮤니케이션하는가

owner와 user가 다를 수 있다는 점 자체가 문제는 아니다. 문제는 그 차이를 내부 문서와 실제 운영에서 설명하지 못하는 상태다.

## 직접 출원과 국제출원은 운영 속도로 고른다

직접 출원과 마드리드 출원은 비용만으로 고르면 안 된다. 중국 단일 시장의 속도, 지정상품 커스터마이징 필요성, 내부 의사결정 속도를 함께 봐야 한다.

### 직접 출원이 맞는 경우

- 중국이 최우선 시장이다
- 지정상품과 표기 전략을 중국 시장에 맞게 세밀하게 설계해야 한다
- 현지 대리인과 빠르게 왕복하며 수정해야 할 가능성이 높다

### 국제출원이 맞는 경우

- 다국가 확장이 동시에 진행 중이다
- 본사 차원에서 포트폴리오를 중앙 관리해야 한다
- 중국도 중요하지만 전체 관리 효율을 같이 봐야 한다

| 판단 항목 | 직접 출원 쪽이 유리한 신호 | 국제출원 쪽이 유리한 신호 |
| --- | --- | --- |
| 시장 우선순위 | 중국 단일 시장이 핵심 | 여러 국가가 동시에 중요 |
| 상품 설계 | 현지 커스터마이징이 중요 | 본사 기준의 통합 관리가 중요 |
| 속도 | 빠른 왕복과 세밀한 조정 필요 | 중앙 관리 효율이 우선 |
| 내부 구조 | 중국 전담 판단 체계가 있음 | 국가별 운영을 묶어야 함 |

실무적으로는 “어느 쪽이 더 고급 경로인가”가 아니라 “어느 쪽이 지금 조직이 끝까지 감당할 수 있는가”로 보는 편이 맞다.

## 출원 전 체크리스트

### 최소 판단 항목

| 점검 항목 | 확인 질문 |
| --- | --- |
| 표장 구조 | 영문, 한글, 중국어 표기를 각각 따로 출원할지 |
| 상품 범위 | 실제 판매 SKU와 지정상품이 연결되는지 |
| 사용 주체 | 본사, 중국 법인, 대리점 중 누가 실제 사용자로 남는지 |
| 우선순위 | 시장 진입 전에 반드시 확보할 클래스가 무엇인지 |
| 경로 선택 | 직접 출원과 국제출원 중 현재 운영 구조에 맞는지 |

### 제출 전 마지막 질문

- 이 표장을 중국에서 어떤 이름으로 먼저 쓰게 될까
- 중국어 표기를 지금 확정하지 않으면 누가 현장에서 대신 정하게 될까
- 코어 상품군과 방어 범위를 구분해서 설명할 수 있는가
- owner와 user를 문서로 설명할 수 있는가
- 현재 조직이 이 출원 경로를 끝까지 운영할 수 있는가

## 한 줄 결론

중국 출원은 신청서 작성보다 설계가 먼저다. 표장 구조, 중국어 표기, 지정상품, 서브클래스, 출원 경로, owner-user 구조를 초기에 잠가 두면 이후 심사와 사용, 집행 단계의 혼선을 크게 줄일 수 있다.
